　　薛仁贵与老赵两人怀古感今，各自讲着自己的经历，很快便成了无话不谈的好友。
<br/>　　而经过了解两人发现，其实对方都不是什么傻子，也不是什么坏人，只是以往缺乏互相之间的了解，所以才会有之前的误会。
<br/>　　而另一边，还在对拜占庭王都发起进攻的候君集也收到了同样的一批装备，十几个来自大唐科学院的研究员在一个十分适宜的时候找到了老候。
<br/>　　“扯蛋，都特么该干咐干啥去，什么实时与长安保持通讯，你们都吃饱了撑的吧？”候君集可没有薛仁贵那么好的脾气，无线电项目负责人找到他把来此的目的一说，顿时被骂了个狗血淋头。
<br/>　　老候打了一辈子的仗，隔了几万里能在半个月把消息传到长安在他看来已经是最快的速度了，现在跑来一群书呆子，竟然告诉自己，可以随时随地保持与长安的联系，这特么不是扯蛋又是什么。
<br/>　　无线电项目的负责被骂了之后也有些不知所措，想要争论几句，被老候拿眼珠子一眼顿时就不敢说话了，臊眉耷眼的退到了一边。
<br/>　　裴行俭在一边看那十几个家伙有些可怜，主动站出来的说道：“大将军，左右费不了什么事，不如派些人给他们，让他们折腾去吧。”
<br/>　　“老子不管那些，老子现在只要这拜占庭的都城，这都十来天了，竟然连城墙都没给老子拿下来，你们还有脸来提要求！”候君集鼻子不是鼻子脸不是脸的训斥着裴行俭，再也没有了半年前的那种和蔼可亲。
<br/>　　小裴被骂了之后倒也没有多说什么，只是尴尬的笑了笑，往前凑了凑，来到候君集身边压低声音说道：“大将军，这些人来咱们这里干什么并不重要，重要的是他们有陛下的圣旨，咱可以不搭理他们，可不能不搭理陛下啊。”
<br/>　　提到李承乾，候君集犹豫了一下，冷眼扫过等在一边的几个‘穷酸’书生，哼了一声：“罢了，既然你们是奉了陛下的命令，本将军便给你们一个机会，三天时间，把你们说的那个东西弄好，然后告诉本将军，一个月前，最后一份战报的第八十二个字是什么，若是对了还则罢了，若是错了，老子摘了你们几个的脑袋。”
<br/>　　三天时间，如果是在大食那边的无线电试验场弄好之前，这十几个来自科学院的家伙估计早就灰溜溜的走了。
<br/>　　可是现在不一样了，所有数据基本上都已经收集完毕，他们需要做的就是把设备组装起来，然后接上电源便可以直接使用，三天时间对他们来说已经足够。
<br/>　　想到这里为首之人便点了点头，算是接受了候君集的安排。
<br/>　　不得不说，这也就是那几个书生胆子大，不知道候君集的脾气秉性，还以为他是因为心情不好故意吓唬自己，根本没有想过他是个说到做一的人，如果三天之后他没有满足老候的要求，到时候很有可能真的会人头落地。
<br/>　　裴行俭原本还想再替他们说上几句好话，结果看他们一点头，心里顿是就凉了半截，李承乾的那句话说的好啊，不作死就不会死，都是被无知害的，也不知道三天之后这帮家伙在人头落地之前会不会后悔今天的草率。
<br/>　　……
<br/>　　几个书呆子很快就被打发走了，候君集信守承诺，派了一直五百人的队伍替他们警戒，顺便也是看着这十几个人，省得被他们见机不好跑掉。
<br/>　　等人都出去了，候君集才又拉下脸来，冷声训斥道：“老子不管你们用什么方法，总之从现在开始给你们三天，三天之后若是还不能攻破拜占庭的王都，所有人削耳示众！”
<br/>　　好好一个大活人，谁也不想没了耳朵，故而候君集才刚一说完，所有人都变了脸色，一个个紧张的要命。
<br/>　　其中一人平时有些机灵劲，被老候这一吓还真被他给想出一个办法：“大将军，依我看，不如我们挖一条地道过去，通到城墙底下，直接把那破墙给他炸了，您觉得怎么样。”
<br/>　　“我看不怎么样，你可别忘了，那城外边还有护城河呢，挖地道，只怕你不没有挖到地方，自己就先被淹死了。”龙义性子急，再加上又是老候的嫡系，不等候君集表态便主动接口说道。
<br/>　　“有护城河怕什么，那水是活水，把上游一堵，下流开一条沟，引走也就是了。”出主意的家伙满不在乎的说道，似乎并没有把那一护城河的水当成问题。
<br/>　　候君集凝神坐在主位之上，沉思了一会儿点头说道：“此计倒也可行，那拜占庭不过就是仗着坚城与我军为敌，野战并不是我们的对手，封锁河道，开挖引水渠倒也不愁他们来捣乱。”
<br/>　　“大将军，那您是同意了？”
<br/>　　“嗯，便按这个计划去执行吧，抓紧时间，若是三日之后那城墙还在，当心你们的耳朵。”候君集点了点头，算是认可以这个作战计划。
<br/>　　王都已经是拜占庭人最后的一块‘净土’，如果沦陷，拜占庭将不复存在。
<br/>　　所以这一次他们也是拼了，城中只要是能动弹的，不管男女老幼，全都上阵，能作战的便上城头作战，不能作战的便负责运送守城的物资，总之是人尽其力，物尽其用。
<br/>　　但就算如此，城破之依旧是迫在眉睫，每天都会有大量的伤兵被人从城头抬下来，整座王都之中到处都是哀嚎之声，甚至就连王宫里都不消停。
<br/>　　可是没有办法，亡国在即，好人都顾不过来，谁还有心思管那些伤兵呢。
<br/>　　君士斯坦早就已经乱了方寸，再也没有了当初引三十万大军去攻击候君集的威风，终日躲在王城之中惶惶不可终日，根本顾不上守城之战。
<br/>　　老国师这个时候已经回到了都城，不过此时他已经断了一条腿，眼睛也少了一只，如果不是有亡国之危近在眼前，这老头儿再怎么也不应该重回战场。
<br/>　　但形势比人强，如果他不顶上来，估计这个时候王都早就已经被攻破了，所有王族都会沦为阶下囚，更别提什么反抗了。
