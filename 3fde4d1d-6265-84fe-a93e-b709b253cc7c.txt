　　云羽天宫的传承，人人都想要得到。
<br/>　　在场的武者，谁没有一刻强者之心，若不想要得到这云羽天宫的传承，他们也都不会在这天羽神域中，待上这么久的时间。
<br/>　　“别傻了，这羽化晶，必须要得到五万枚，才可以真正得到云羽天宫的传承，你以为我们能得到五万枚？”
<br/>　　令一尊荒古门弟子，不屑的开口道。
<br/>　　“倒不如还是交给师兄，去换取一些好处。”
<br/>　　“也是，我们多多收取羽化晶，交给师兄，也足以换取好处，倒时候等出去，我们就可以想办法，突破生死，一旦是达到碎命之境就是真正的老古董，才宗门中也地位超然。”
<br/>　　令一名弟子开口道。
<br/>　　他们二人一边说着，一边不断的收取这宫殿之中的无数宝物，其中的羽化晶，也都被他们收走。
<br/>　　就在这时，突然一道身影，猛然之间冲入到了宫殿之中。
<br/>　　“谁？”
<br/>　　那两名荒古门弟子，脸色猛然一变，一脸杀气。
<br/>　　在这天羽神域中，为了无数宝藏，杀人易货也是随时可见。
<br/>　　等到他们二人，看清楚这进入其中的身影一刻，脸色猛然大变。
<br/>　　这一道身影，正是刚刚以一敌四，还将古白打成重伤的楚风眠。
<br/>　　“快逃！”
<br/>　　这两名荒古门弟子，身形一动，直接要逃。
<br/>　　楚风眠的实力，从刚刚便是展现的淋漓尽致，他们二人绝不敢跟楚风眠抗衡，几乎毫不犹豫，便是要逃。
<br/>　　“在我的面前，也想逃？”
<br/>　　楚风眠嘴角露出几分不屑，大手一挥，这两名荒古门弟子，便是被一股灵力，直接轰死，一身血雾，都被楚风眠全部收走。
<br/>　　楚风眠走过去，捡起来了这两名荒古门弟子的空戒，他们二人的空戒之中，一共有着一千多枚羽化晶，还有这不少的宝物，被楚风眠直接放入到他的空戒之中。
<br/>　　“荒圣君，看来对着云羽天宫的传承，也是势在必得，已经开启花大价钱去购买羽化晶了。”
<br/>　　这两人之前的对话，楚风眠也是听到了一些，自然知道这件事。
<br/>　　荒圣君如今是铁了心要得到这云羽天宫的传承，不然不会花费这么大的代价，来购买羽化晶。
<br/>　　因为这云羽天宫传承一旦是被别人抢先得到，手中的羽化晶便是都会没有任何意义，倒时候付在大的代价，都是打了水漂。
<br/>　　这让楚风眠的心中都有些了紧迫感。
<br/>　　要是荒圣君花费大力气，去收购羽化晶的话，不止是七大宗门的人，甚至是其他势力的武者，都有可能自知得到传承无望，而将羽化晶卖给荒圣君。
<br/>　　不管是得到一些宝物，还是得到荒圣君一个人情，都是稳赚不赔的交易。
<br/>　　这云羽天宫之中的武者，七大宗门的弟子，其实最多只有一成，而剩下的九成，都是属于其他势力的。
<br/>　　这些武者，不少人都有着自知之明，知道这云羽天宫的传承，绝不是他可以争夺的。
<br/>　　这些人都有可能将羽化晶卖给荒圣君，等于是这些人都在帮荒圣君来收集羽化晶。
<br/>　　必须要加快步伐才行。
<br/>　　楚风眠身躯一动，再度动用起魅影身法来，便是向着下一座宫殿之中跑去。
<br/>　　半个时辰的时间，楚风眠便是连续去了四五座宫殿，将其中武者手中的羽化晶，全部得到。
<br/>　　碰到七大宗门的弟子，楚风眠便是格杀勿论，没有任何可说的。
<br/>　　至于那些并非七大宗门弟子的武者，楚风眠都是主动让他们交出羽化晶来，若是交的，楚风眠便是不杀他们。
<br/>　　毕竟楚风眠跟其他宗门，也没有任何仇怨，他并非是一个滥杀无辜之人，只要是交出羽化晶来，他们所得到的其他宝物，楚风眠一个不取。
<br/>　　但是若是那些武者不识相的话，楚风眠也不会手下留情，只能斩杀。
<br/>　　楚风眠可不会让他们将这些羽化将卖给荒圣君的机会，不杀了他们，便是在帮助楚风眠的敌人。
<br/>　　楚风眠可没有这种兴趣。
<br/>　　一个时辰的时间，楚风眠便是已经将周围的宫殿，都给搜寻了一片。
<br/>　　这一座座宫殿巨大无比，在这云羽天宫之中，楚风眠也无法动用遁光，也无法利用战龙宝车，只能动用身法来，赶路的速度便是慢了许多。
<br/>　　一个时辰的时间，才将这周围的宫殿全部嗖完。
<br/>　　得到的羽化晶，也已经是有着一万多枚，除此之外，无数宝藏，更是楚风眠的空戒都装不下，撞了满满好几个空戒。
<br/>　　可以说现在就算是离开，从这云羽天宫中得到的宝藏，也不亚于一个颜古龙帝洞府的宝藏。
<br/>　　不过楚风眠又怎么可能会这样就离开，他的目标，是云羽天宫的传承。
<br/>　　不将这云羽天宫传承得到，楚风眠绝不会离开。
<br/>　　“继续！”
<br/>　　楚风眠停留了一阵，便是再次动身，前往更远的一处区域。
<br/>　　令一边，那七大宗门弟子，似乎已经是以荒圣君为首，将所有得到的羽化晶，全部都交给了荒圣君。
<br/>　　在一处宫殿前，荒圣君在宫殿之外，坐在地上，似乎是在修炼这一种功法。
<br/>　　“师兄。”
<br/>　　一名荒古门弟子，急忙的赶了过来，将一枚空戒，递了过去。
<br/>　　这空戒中，密密麻麻，全部都是羽化晶。
<br/>　　“这是玄天宗的人，前来献给师兄的。”
<br/>　　“好，告诉玄天宗的人，事成之后，不会少了他们的好处。”
<br/>　　荒圣君从地上缓缓的站了起来，开口道。
<br/>　　“还有，许诺更多好处，让所有人都将羽化晶交给我，这云羽天宫的传承，我必须要得到！”
<br/>　　“是，师兄，我还有一件事要禀告。”
<br/>　　那荒古门弟子，脸色有些阴沉的说道。
<br/>　　“说！”
<br/>　　荒圣君冷喝道。
<br/>　　那荒古门弟子，吓了一跳，急忙跪在了地上开口道。
<br/>　　“在这云羽天宫中，那楚风眠跟寒月漓二人在追杀我们七大宗门的弟子，还有一些不是七大宗门的人，都被他们斩杀，所有的羽化晶，都被抢走了。”
<br/>　　“哦？果然，这小子的目的，也是为了这云羽天宫的传承而来。”
<br/>　　荒圣君听到这一番话，脸上没有任何的惊讶，平淡无比，他的心中早就猜到了这一切。
<br/>　　“传令下去，加快速度，先不用对付这小子，等我得到这云羽天宫的传承，以后有这是办法杀了这小子！”
