“走，上车”<br/>　　<br/>　　    就在这个时候，王惊蛰和陈臣也开着卡宴过来了，但他俩看见前面干起来之后，就连忙靠边停下没敢过去。<br/>　　<br/>　　    “这是开枪了啊，闹得这么大？”陈臣小脸煞白的说道。<br/>　　<br/>　　    卡宴停车后，车灯就透了过去，王惊蛰顺着车灯光线眯着眼睛看着前方，那辆捷达旁边正站着道身影拉开车门准备坐进去。<br/>　　<br/>　　    “怎么是他？”王惊蛰顿时惊讶的说道，他认出来了那身影是一天之前还和他在一起喝酒的林汶骐。<br/>　　<br/>　　    但就在这个时候，卡宴后面忽然速度极快的蹿过来一辆奔驰，在路过两台别克的时候，吴洋放下车窗就喊道：“哪辆车能开，马上走，继续给我堵住他们，枪给我一把”<br/>　　<br/>　　    奔驰车身一顿，停了一下，别克里的人甩手把五连发顺着车窗就扔了进去，停顿的奔驰随即再次几步，车速在三四秒钟左右差不多就要破百了，直接照着已经开出去的捷达追了过去，同时一台GL8也开动了起来，剩余几个有枪又没受伤的人上了车。<br/>　　<br/>　　    王惊蛰皱眉寻思了下，说道：“跟上去，离着稍微远点……”<br/>　　<br/>　　    陈臣有点小怕怕的说道：“啊？”<br/>　　<br/>　　    “前面那辆车里有我一个朋友，我估计他可能要够呛，但我不能看着他不管啊”王惊蛰舔了舔嘴唇，他在看清是林汶骐之后，也看见了刚刚从奔驰车窗里伸出脑袋喊话要枪的吴洋。<br/>　　<br/>　　    这叫狭路相逢，正巧全碰在以前了，王惊蛰不知道林汶骐为啥跟他们碰撞了起来，但既然他赶上了，那说不得就得掺和一下了。
