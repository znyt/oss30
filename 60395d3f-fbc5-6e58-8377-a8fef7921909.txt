　　混沌道君自然是匆匆而来，又匆匆而去，一刻也没有在永恒岛上多待。
<br/>　　用他的话来说，就是他只是来送东西的，送完东西，自然也就没有他的事了。之后，他就再次前往混沌之中。而且，如今他已经突破了混元大罗金仙，他想到混沌之中多走走。
<br/>　　混沌无边无际，盘古所开的洪荒宇宙也只占了三千魔神的活动范围的三分之二大小，而且混沌不可能只有这点地盘。
<br/>　　混沌无垠，洪荒宇宙只是混沌海中的一个岛屿而已。
<br/>　　以前是没有实力，混沌之中危机重重，害怕冒险出去这尊分身会有身陨的危机。如今他已经突破混元境界，虽说仍然算不得无惧一切，但至少也有一点自保能力了，故而，这才有了游走混沌的想法。
<br/>　　混沌道君修炼的是混沌法则，不是仅仅闭关就能突破的。
<br/>　　用他的话来说，就是混沌这么大，我想出去走走。
<br/>　　既然想出去走走，那么就走走吧。更何况元始也想要看看这混沌之中的其他宇宙又是如何样？
<br/>　　就这样，混沌道君潇洒的走了，只带着一柄先天至宝弑神枪。
<br/>　　弑神枪元始没有留了下来，虽说它威力巨大，但元始手中又不是没有威力巨大的法宝，先不说因果之主的混沌灵宝因果莲台，就说黑暗魔神的黑暗天幕，昆吾老祖的生死石印，以及开天之初元始抢到的鸿蒙量天尺，就足够元始用的了。
<br/>　　混沌道君带走一件弑神枪也没什么，况且他带着弑神枪，在混沌之中关键之时也能解决一些危机。
<br/>　　混沌道君走后，元始打算先将十二品黑莲与乾坤鼎这两件先天灵宝炼化，之后在做其他事情。
<br/>　　元始最先炼化的是极品先天灵宝乾坤鼎，乾坤鼎乃混沌青莲莲叶所化，在开天之初为乾坤老祖所得。它的攻击力与防御力都很一般，不值一提。
<br/>　　但，它有一个逆天的作用，能够返后天为先天。
<br/>　　就此一个作用，元始认为这次出手就值了。因为元始根据盘古传承与自身的一些炼器知识结合自创的炼器法诀大道炼宝诀如今已经到了一个瓶颈。想要突破，就必须另想他法。
<br/>　　元始一直认为，真正的炼器大道就应该是能够以寻常后天之物炼出先天神器这样的大道。要不然，以顶级神材炼出顶级神器，那只能算是一般器道，算不得炼器大道。
<br/>　　之前的大道炼宝诀元始已经能够炼出顶级的后天至宝，但想要练成先天灵宝，他感觉总是差那么一点。
<br/>　　如今，有了乾坤鼎，元始就有机会参悟其中的奥妙原理。
<br/>　　若是能够参悟透，那就意味着元始以后能够源源不断地炼出先天灵宝，那是多么令人震撼的一件事。
<br/>　　当然，靠着乾坤鼎，元始依旧能够做到如此，但，元始不太喜欢依赖外物，外物终究是外物，不是自身的实力。要是有一天它不见了，那到时岂不是没地方哭了。
<br/>　　况且，外物终究只是大道路上的护道之法，没有人能够紧靠外物就证得大道的。自身之道，永远是最关键的。
<br/>　　越到后期，这种差距就越明显，前期一人只要拿着一件神器，就能够以低境界修为逆斩高于自身数个境界之人。
<br/>　　但，到了后期，这种差距就会越来越小，直至反过来。有了自己大道的低境界者逆斩拥有神器却没有自身大道的高修为者。
<br/>　　就如鸿钧老祖，他成圣之后，就只留一件法宝造化玉牒，其余的先天至宝都被他赐给了自己的弟子。但，就没见过他哪个弟子能够拿着这些灵宝去战胜鸿钧老祖的。
<br/>　　元始从一开始就不是很依赖法宝的力量，他大多情况都是参悟法宝的奥妙法则，使之成为自己实力的一部分，如昆吾老祖的生死石印，他就参悟出阴阳生死印。
<br/>　　乾坤鼎炼化之后，元始并没有直接参悟，而是取出十二品黑莲来，他想接着炼化黑莲。如今距离鸿钧成圣已经不远了，自己并没有太多时间浪费在参悟法宝奥妙之上。
<br/>　　这些虽然也很重要，但对于自身大道来说，他们还是要略次一等。参悟法宝规则这样的事，还是等他成圣后再说吧。
<br/>　　刚一开始炼化黑莲，元始就察觉到了不对，于是，他停止手中动作，而是对着它淡淡道：
<br/>　　“出来吧，罗睺。”
<br/>　　只是在元始说完话后，黑莲依旧一动不动，静静悬浮在空中。
<br/>　　“你还不出来吗？那可别怪我不客气了。”
<br/>　　元始皱了皱眉头，再次对着黑莲说道。同时，将十二品净世白莲取了出来。
<br/>　　就在元始想要催动净世白莲的净世之光对付黑莲之时，一道黑光从黑莲之中射了出来，之后以一种不可思议的极速钻进元始眉心的识海之中。
<br/>　　“桀桀桀！小子，任你百般小心，不管你用什么方法，就算被你发现了，但，那又如何？终究还是要为本魔所夺舍。哈哈哈！”
<br/>　　黑光钻进元始识海后，化作一道人影，一身黑袍，杀气腾腾，正是魔祖罗睺的模样，准确的说，这是魔祖的一道残魂。此刻，他正在哈哈大笑。
<br/>　　“哦？是吗，那你回头好好看看。”元始的元神一点也不惊慌，饶有兴趣地看着魔祖罗睺的残魂，淡声道。
<br/>　　“什么，小子，看什么……”罗睺嚣张道，只是，等他转过身来后，看到的是他站在一个漆黑空旷的空间之中，在他面前不知多远处，有一个金黄色圆球在缓缓旋转，散发出耀眼的光彩。
<br/>　　那是元始的元神。《神元九转》第五转之时，元始就已经将他的识海化为界域了，他所有的元神之力如今都在一个元神界域之中，故而元神空间变得十分空旷。
<br/>　　“那是什么？”不知为何，罗睺心中总有一种不好的预感。
<br/>　　“不是什么，就是元神啊。”元始轻声笑道。
<br/>　　罗睺大声叫道：“小子，你元神怎么会……”
<br/>　　只是，他的话还没有说完，便被一道冷漠的声音打断：
<br/>　　“五行印，定！”
<br/>　　顿时，一道五彩大印从光球之中飞出，迅速向罗睺残魂飞去，罗睺想要躲避，但却没办法躲开，只能任由五行大印将他定住。
<br/>　　“阴阳印，封！”
<br/>　　“四象印，镇！”
<br/>　　紧接着，黑白阴阳印与四角四象印迅速飞出，将罗睺残魂封住，镇住，不让他乱动。
<br/>　　“搜魂！”
