翌日清晨，夏夕绾醒来的时候发现自己已经睡到床上了，她想了想，怎么也想不起来自己是怎么睡上床的。
<br/>
<br/>        她以为昨晚会很难入眠，会做噩梦，但是昨晚竟然出乎意料的好眠，她红肿的眼眶也消了下去。
<br/>
<br/>        “爷爷，我现在就去夏家了，放心吧，我会让她们为自己的一言一行付出代价的！”
<br/>
<br/>        夏夕绾打车去了夏家，今天是夏振国的生日，但是整个夏家愁云惨淡的，也没有人来庆生。
<br/>
<br/>        李玉兰从医院里回来了，还在房间里坐月子养身体，她看见夏夕绾情绪很激动，“夏夕绾，你来干什么，快出去，这里不欢迎你！”
<br/>
<br/>        李玉兰不想让夏夕绾看自己的笑话。
<br/>
<br/>        夏夕绾倚靠在门边看了李玉兰一眼，“难道夏家已经换主人了，今天是我爸的生日，我来给我爸庆生，李玉兰你一个外人凭什么赶我走？”
<br/>
<br/>        “你！”
<br/>
<br/>        李玉兰生气的时候却被夏妍妍给拦下了，“今天是爸的生日，我们都是一家人，就不要再吵架了，绾绾，你留下来吃顿晚饭再走吧。”
<br/>
<br/>        夏夕绾看着夏妍妍，夏妍妍又恢复了平日里娇柔温婉的绿茶样，发生了这么多事，她竟然还能这样忍气吞声，夏夕绾嗅到了不正常的气味，这个夏妍妍肯定又想策划什么，她应该是要用她最后的筹码放手一搏了。
<br/>
<br/>        “好啊，那我就留下吃顿晚饭吧。”夏夕绾转身离开了。
<br/>
<br/>        房间里，李玉兰拉住了夏妍妍，“妍妍，你留夏夕绾干什么？”
<br/>
<br/>        夏妍妍的双眼里露出了算计的光芒，不过口风很紧的答道，“妈，我心里自有计划，你就不用管了。”
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾站在落地窗边喝了一口茶，这时她感觉身后有人在悄然的接近，她抬头看了一下眼前的玻璃窗，正好看到一个黑色保镖走了过来，带着危险的气息。
<br/>
<br/>        夏夕绾没有动，只是将红唇勾出了一道浅浅而冷漠的弧线。
<br/>
<br/>        这时身后的黑衣保镖抬手，直接劈向了她的后颈。
<br/>
<br/>        骤痛传来时，夏夕绾两眼一闭，晕了。
<br/>
<br/>        黑衣保镖伸手将夏夕绾接住，这时夏妍妍走了过来，命令道，“将她送到楼上房间去。”
<br/>
<br/>        “是大小姐。”
<br/>
<br/>        黑衣保镖将夏夕绾送到了楼上房间的大床上，夏妍妍拿出手机，拍了一张照片分别发送给了两个人。
<br/>
<br/>        做好这一切，夏妍妍坐在了客厅的沙发上，等人。
<br/>
<br/>        半个小时后，别墅大门被打开，一道俊美颀长的身影跃入视线，陆子羡来了！
<br/>
<br/>        夏妍妍勾唇，“陆院士，欢迎你，都说你是如今医学界最风华的年轻院士，没有之一，多少名媛爱慕你，但是你连看她们一眼都嫌多，没想到你却喜欢上了夏夕绾，成了她的裙下之臣，只要涉及她的事，你都会第一时间赶来。”
<br/>
<br/>        陆子羡清冷的黑眸淡淡的看着夏妍妍，“夕绾在哪里？”
<br/>
<br/>        “楼上房间。”
<br/>
<br/>        陆子羡抬脚就要上楼。
<br/>
<br/>        “陆院士，”夏妍妍在后面叫住他，“今天是绝好的一个机会，夏夕绾已经送上了你的床。”
