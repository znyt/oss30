　　混沌神殿中，还是一如既往地神秘。
<br/>　　“这是哪里……”忽然，一道隐含戒备的清脆女子之声响起。
<br/>　　随着声音传来的方向看去，顿时，一个近乎完美的女子出现在众人面前。
<br/>　　众人发现自己出现在一个陌生的地方，都有些戒备的互相打量着对方，此处神秘空间中有二三十人之多，有的相互认识，而有的却不认识。同时，他们也在疑惑地打量着周围的环境，希望找出点线索。
<br/>　　之前还在各自熟悉的环境之中，只是眨眼间，便步入一个完全陌生的地方，以他们的修为，竟然毫无察觉，这就有点惊悚了。
<br/>　　在这片混沌色空间之中，“混沌神殿”四个潦草大字高高悬挂，中间是一片宽阔地方广场，看似狭小，实则宽广，有点类似芥子纳须弥的样子，四周是各种神秘雕刻图案。
<br/>　　与此同时，一些陌生的信息出现在他们脑海之中。
<br/>　　片刻之后，消化了这段信息之后，他们也都明白了这是什么地方了。
<br/>　　简单来说，这里是一个叫“混沌道君”的神秘大神培养人才的地方，神殿发布任务，完成任务后，能够得到相应的价值点，价值点可以在神殿中兑换各种宝物。
<br/>　　在知道这是一个什么地方后，众人虽然心中仍然还有疑惑与不解，但都暗自松了一口气。虽然知道的不多，但至少大致了解了一些。同时，他们也知道，对于这个神秘的存在，他们完全没有反抗之力。生活就是这样，既然不能反抗，那就尝试着去接受。
<br/>　　而且，这里虽然是一个神秘的地方，但也代表着莫大的机缘，是能够让他们改变命运的机遇，这是他们值得高兴的事。
<br/>　　“这是真的吗？”一道疑惑的声音响起，问出了众人心中最真实的想法。
<br/>　　“是不是真的，一试便知。在下太玄门华云飞，见过各位道友。”一位身穿蓝衣的年轻修士朗声道。
<br/>　　“原来是太玄门的华师兄啊，小妹妙欲庵安妙依。”一身雪白衣裙的秀丽女子开口说道。
<br/>　　“原来是安仙子。想必这位就是瑶池的圣女殿下吧。”说着，他看向另一位同样绝美的女子。
<br/>　　“不错，我便是。”那女子开口应答道。
<br/>　　“不知这位妖族的道友是……”接着他看向另一位完美的女子，一眼便知道她是妖族之人。
<br/>　　“颜如玉。”那女子淡淡开口道。
<br/>　　“听说妖族出现了一位妖帝后人，不知仙子是否知晓？”
<br/>　　颜如玉目光微凝，紧接着淡淡道：“知晓。”之后便不再多语。另一边，一位桀骜的金发男子听了此话后看向这边，他是妖族的金翅小鹏王。
<br/>　　“不知这位小妹妹是……”众人目光看向一位身穿紫衣的女孩。
<br/>　　女子眼珠子忽悠忽悠的乱转，不知道在想些什么，见众人看向她，她才露出个小酒窝，笑嘻嘻道：“我叫姬紫月。”
<br/>　　“姬家小月亮！”有人眼睛一缩，说道：“听说姬家出现了一位神体，不知是不是真的？”
<br/>　　紫衣女孩微笑不语。
<br/>　　“道长叫什么名字？”众人看向一位身穿青色道袍的胖道士。
<br/>　　道长看了看众人，心中暗骂不已，这些人家族中的大墓，他都光临过，不知道说实话后他会不会被打死，于是，他决定说个假名，他面上微微一笑道：“咳，无量天尊，贫道段道！”
<br/>　　“各位美女，你们好，在下人欲道燕一夕，希望能和你们做朋友。”
<br/>　　一个风流倜傥的白衣男子对着几位仙子开口道，说着，他的目光还在暗自打量着众人，心中直叹为何他从没有见过如此完美的仙子，按理来说，如此绝色应该闻名天下才对。
<br/>　　“人欲道，没听说过。”
<br/>　　“嘿嘿，人欲道可是紫薇星的大教，在下人欲道历天，愿与各位美女同游。”一个五色彩衣男子邪邪一笑，嘿嘿道。
<br/>　　“紫微星，你们不是北斗星之人？”有人诧异道。
<br/>　　“北斗星？那是葬帝星吗？你们难道来自那里？”燕一夕与历天诧异道。
<br/>　　“不错，我们来自北斗星，你们呢？”
<br/>　　“哦？原来如此，怪不得我们不认识，原来是北斗的道友啊。不错，我们来自紫薇星。”
<br/>　　众人觉得不可思议，竟然不是北斗之人，混沌神殿好真是不可思议。
<br/>　　“你们呢？”有人将目光看向其他人。
<br/>　　“在下永恒星墨羽。”
<br/>　　“在下飞仙星李元。”
<br/>　　“在下……”
<br/>　　“在下地球天师道张清扬。”
<br/>　　众人面面相觑，发现各自都来自不同的星球，有的听说过，而有的却从来没有听过。想得越多，众人心中便越是惊骇，混沌神殿的道君到底有多强，谁也不知道，可如今能够将宇宙间各处的人全部聚拢在一个地方，那便是不可思议了。
<br/>　　这种修为，至少也是一位准帝。
<br/>　　就在众人都暗自猜疑之际，一道淡漠的声音在大殿中响起：
<br/>　　“叮！现在发布任务！
<br/>　　任务背景：紫微星太阴神教覆灭人族古皇太阴古皇后人，占据他们的教派，抢夺人皇传承，谋夺正统，自称太阴神教，视古皇后裔为奴仆。太阳古教，太阳圣皇的传承教派，自古经遗失后便一落千丈，金乌族对此虎视眈眈，欲覆灭古教。太阴神教神子将与金乌族十大太子聚会。”
<br/>　　“主线任务：三天内击杀太阴神教神子与金乌族十大太子，共同完成任务，完成者每人五百价值点，若是失败，集体抹杀！”
<br/>　　“支线任务：尽可能多的击杀两派弟子，每杀一苦海境修士，奖励十价值点，命泉修士三十价值点，彼岸修士一百价值点，道宫修士，五百价值点——一千价值点；四极境修士，一千价值点——五千价值点。化龙修士：五千——一万价值点。仙一修士……”
<br/>　　“任务提醒：任务将于一个时辰后开始。”
<br/>　　“什么？！竟然会有这样的任务？”众人将目光看向燕一夕与历天，因为他们是紫微星之人。
<br/>　　“情况大致如此。”两人苦笑一声，说道。
<br/>　　“该死，竟然有人谋害人族古皇后人！”两教的做法令人族寒心。“难道没有人管吗？”
<br/>　　“不知两教实力如何？”华云飞开口问道。
<br/>　　“两教的实力都深不可测，教中有圣人坐镇，这个任务实在是……”燕一夕苦笑道，不知该说些什么。
<br/>　　“圣人？”众人心中一凝，圣人可不是小事啊，北斗许多年都不曾出现过一位圣人，由此可见圣人的稀少，许多门派都不见有圣人出现。当然，不是没有圣人，而是圣人一般不会出现。
<br/>　　“我想主神不会发布我们不能完成的任务。我们的目标是金乌十太子与太阴教的神子们，他们的实力如何？”
<br/>　　“他们的实力在四极到化龙，修为不一。”人欲道两人有些凝重道。他们万花丛中过，片叶不沾身，经常会与一些门派的仙子历练，倒是多少知晓一些情况。
<br/>　　“化龙么？”众人心头有些沉重，四极境他们可以对付，因为他们中就有四极境的修士，化龙境就需要拼命了，而且还不一定成功。
<br/>　　“看来我们需要兑换一些宝物了。”说到宝物，众人将心神沉浸道脑海之中，同时，心中默念一声“兑换”。
<br/>　　顿时，在他们的脑海深处，便出现一道巨大的光幕，上面写着六个金光大字：法、术、阵、器、药、杂。
<br/>　　华云飞随意点了一个“法”字，只见上面闪烁出各种经文：
<br/>　　《盘古开天诀》，盘古开天之无上证道天经，共分九层，九层之上，可开天辟地。价值999万亿价值点。
<br/>　　《紫霄天道经》，道祖鸿钧讲道之天经，包含大道三千，为亿万道引，大成之后，可身合天道。价值99万亿价值点。
<br/>　　《玉清元始经》，即天罚圣主道印天尊上清圣人元始所创之功法，乃阐教根本法，价值9万亿价值点。
<br/>　　《太清道德经》，即太上道主道德天尊太清圣人所创之功法，乃人教根本法，价值9万亿价值点。
<br/>　　《上清灵宝经》，即通天剑主灵宝天尊上清圣人所创之功法，乃截教之根本法，价值9万亿价值点。
<br/>　　……
<br/>　　之后，他依次点开其他光幕，看到其他东西。其中大多数都是他不认识的，但看简介与价值点便知道那不是一般的东西。看到这些宝物，没人能够静下心来。他看到好多自己心仪的宝物，但是没有足够的价值点来兑换。于是，心中一狠，他将自己的狠人传承兑换给主神了。
<br/>　　“《吞天魔功》，狠人大帝所创无上魔功，价值一千万价值点。”
<br/>　　“《万化圣诀》，狠人开创无上攻伐之术，价值十万价值点。”
<br/>　　“《一念花开，君临天下》，狠人秘术，价值十万价值点。”
<br/>　　“《太玄经》，太玄门传承，价值五十万价值点。”
<br/>　　“《道经》，（残缺），价值一万价值点。”
<br/>　　……
<br/>　　华云飞将自己不需要的东西都一一兑换了，换取大量价值点，他再次将目光汇聚到兑换目录上。
<br/>　　“《吞天决》，绝世神功，汇聚五行之灵，阴阳之魂，时空之源，凝聚吞天造化烘炉，无物不吞，无物不炼。价值五千万价值点。”
<br/>　　“《不死经》，凝聚不死血脉，幻化不死之身，一千万价值点。”
<br/>　　“《不灭经》，修不灭之意，成不灭之体，肉身成帝，一千万价值点。”
<br/>　　“《长生经》，修长生真气，寿命绵长，一千万价值点。”
<br/>　　“《大道天音》，天音成道，三千万价值点。”
<br/>　　这些都是他想要兑换的，《吞天决》不说，他自己的《吞天魔功》是别人的炉鼎，想要摆脱命运，《吞天决》比《吞天魔功》更霸道，更厉害，修炼它自然可以废除《吞天魔功》的影响，《不灭经》与狠人的《不灭天功》有些类似，他没有《不灭天功》的传承，修炼它便可以与真正的狠人传承者对抗。《大道天音》是他最喜欢的。
<br/>　　可惜，没有足够的价值点，不然他都想兑换，最终，一番决定后，他决定暂时不兑换功法，先解决此次任务后，待以后价值点足够，再来兑换。他将目光看向器门与阵门。
<br/>　　“射日箭，一箭射日，专克金乌血脉，五百万价值点。”
<br/>　　“追日靴，驱山逐日，五百万价值点。”
<br/>　　看到这里，他顿时眼前一亮，这次任务便是对付金乌是十太子，追日靴与射日弓，是绝配。
<br/>　　他心神一动，便兑换了两件法宝，两件法宝虽然说是专克金乌血脉，但对付其他人，一样厉害，只不过是不想克制金乌那般厉害罢了。
<br/>　　兑换完后，他睁开眼睛看向其他人，见其他人满脸兴奋，看样子也被这其中的宝物给吓住了。
<br/>　　一阵兑换后，差不多一个时辰到了。
<br/>　　“叮！任务开始|！”
<br/>　　一个瞬间，众人只觉得眼前一黑，下一瞬，众人便发觉自己出现在一个陌生的山脚下。
<br/>　　“混沌神殿真是……神通广大啊！”
<br/>(本章完)
