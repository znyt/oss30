　　苏重走出大坑。一道黑影突兀出现在头顶，把他笼罩在其下。一股凶悍气势猛然压来，一颗狰狞龙头出现在苏重眼前。
<br/>　　“魔龙兽？”苏重饶有兴趣的打量眼前的生物。
<br/>　　鹿角龙须猩红眼睛，五爪闪烁寒光。猛一看，非常像中国图腾神龙。
<br/>　　“可惜，加上一前一后两双翅膀，显得有些不伦不类。”苏重摇头叹息。
<br/>　　而且魔龙搜全身覆盖的是羽毛而非鳞甲。虽然这样更符合实际情况，毕竟有翅膀有羽毛才能合理飞。
<br/>　　但这却和传说中腾云驾雾的神龙相去甚远。倒是桀诺揍敌客，念力幻化的龙形念力，更符合传说。
<br/>　　这是揍敌客家族豢养的魔兽。不仅战斗力强悍，而且飞行快速。是不可多得的坐骑。
<br/>　　桀诺揍敌客的念能力叫做龙头戏画，能够把念能力幻化成一头龙。如果没见过实体，只靠想象，可不好修炼。这魔龙兽就是参照物。
<br/>　　看了几眼，苏重不再理会，继续前行。这家伙虽然也叫魔龙兽。但和他的魔龙法相可不是一回事。根本就不是一个物种。
<br/>　　吼！
<br/>　　一声大吼，魔龙兽张开血盆大口，化作一条黑影，猛然窜向苏重。
<br/>　　竟然想吃了我？苏重又好气又好笑，这家伙是在枯枯戮山当贯了山大王？竟然敢攻击自己？
<br/>　　哼！
<br/>　　强悍气魄猛然释放，凶猛威压如水银泻地，轰然炸开。
<br/>　　吱！
<br/>　　刚才还凶残无比的魔龙兽，好似被掐住脖子的鸭子。骤然停住身形，浑身颤抖的僵在原地。
<br/>　　刚才还满是凶残的眼里，此刻全是恐惧。
<br/>　　苏重冷冷的扫了一眼这个胆大包天的家伙，身影一闪消失在原地。
<br/>　　等苏重离开好半晌，魔龙兽才敢活动。心有余悸的看了一眼苏重离开的方向，忍不住的打了个寒颤。
<br/>　　好可怕的人，刚才差点儿就死了！不行，必须多吃点压压惊。这货没心没肺的想着，转头一溜烟就跑没了影。
<br/>　　……
<br/>　　一间阴暗的房间内，四周墙壁肮脏，很多处都掉了皮。
<br/>　　啊！
<br/>　　陡然间，一声惨厉嘶吼传来。好似有人在经历酷刑一般，配合四周幽暗环境，显得格外凄厉。
<br/>　　帕里斯通光着上身，汗水遍布全身，好似刚从水里捞出来。往常梳理的整齐无比的金发，此刻也被汗水打湿成一缕缕。
<br/>　　他脸色惨白，牙关紧咬，剧烈的疼痛一波波袭来。嘴角甚至不受控制的流出口涎。
<br/>　　被苏重打断的左手臂，此刻正在诡异的生长起来。伤口周围，肌肉好似蠕动肉虫子，不断向前攀爬生长。一条手臂竟然以肉眼可见的速度长了出来。
<br/>　　剧烈的麻痒疼痛，不断刺激着帕里斯通的神经，越是疼痛，他就越发痛恨给他造成这种伤害的苏重。
<br/>　　“这一切都要算在你身上，等伤势恢复，一定要你死无葬身之地！”帕里斯通心中发狠，脸色变得越发狰狞。
<br/>　　“对对，就该这样。你的怨恨越强烈，治疗效果就越好。”一个身材矮小的瘦老头站在帕里斯通身后，脸上满是诡异笑容。
<br/>　　老头瘦的近乎皮包骨头。眼眶深陷，两个灯泡似的眼睛布满血丝。看着帕里斯通狰狞表情，开心的手舞足蹈。
<br/>　　正是因为这个怪老头的念能力，帕里斯通的手臂才能够长出来。
<br/>　　瘦老头本来是个法医，不知怎么的觉醒念能力。而且能力奇特，能够重塑体型。付出一定代价的情况下，可以让人断肢重生。
<br/>　　有这能力，他本来应该功成名就。不过这家伙心里扭曲，最喜欢折磨人。即便他能够让人毫无痛觉的恢复肢体，却更喜欢把痛苦加大，看对方的被折磨的死去活来。
<br/>　　正因为这样，才混的如此凄惨。成了地下世界的黑医。但他的能力确实出色，恢复的肢体和原版的一模一样。尽管知道这老头心里扭曲，但仍然有人来找他就医。
<br/>　　帕里斯通是老牌猎人，体魄强健。常年被念能力淬炼的身体，即便不用念能力，也能够轻易捏碎石头。
<br/>　　怪老头却能让他的手臂完全恢复，拥有以前的力量，根本不用再次训练。
<br/>　　好半晌！帕里斯通的整个手臂完全长了出来。他大口喘气，看着恢复完整的手臂，帕里斯通想笑一笑，可脸上的表情却比哭还难看。
<br/>　　恢复手臂，并不是没有代价。在眼前晃动的一缕灰白头发，就是代价。他损耗的是生命力，也可以说是寿命！
<br/>　　“嵌合蚁试验差不多了吧。”帕里斯通低着头，任凭汗水沿着下颌滴下。
<br/>　　“嗯。已经准备就绪了。”阴影里走出一个高大身影，正是一脸威严的比杨德。
<br/>　　他静静的看着一身怨恨的帕里斯通，知道对方已经完全恨上了苏重。
<br/>　　“把糜稽揍敌客引入嵌合蚁大本营怎么样。”帕里斯通低沉道。
<br/>　　比杨德眉头微微皱起，想借着嵌合蚁来说杀死糜稽揍敌客？
<br/>　　这种情况下的帕里斯通，已经失去了往日的冷静，可不是一个好的合作者。他没说话，却表达了他的意见。
<br/>　　嵌合蚁是对黑暗大陆物种的实验，同时也是为了引尼特罗出手。甚至隐隐还有弑杀尼特罗的目的。
<br/>　　尼特罗是阻挡他、阻挡五大国进军黑暗大陆的拦路石。高高在上的五大国，也已经忍不住黑暗大陆的诱惑。准备阴死尼特罗，扫清一切障碍。
<br/>　　帕里斯通的提议，很可能会破坏这一行动。他不允许，他背后的五大国也不会允许！
<br/>　　“糜稽揍敌客依仗强大武力，肆无忌惮。甚至直接毁掉卡金帝国的实验室。即便是揍敌客家族，也在扮演者他自己的角色。他却毫无顾忌。那些喜欢掌控一切的大人们，难道就没有什么心思？”帕里斯通阴沉道。
<br/>　　“你要相信，说服他们同意我的计划并不难。”帕里斯通长袖善舞，最擅长用言语谈判。
<br/>　　“他们会同意，但前提是要让他们相信，你的计划能够杀死糜稽揍敌客，而且不能影响原定目标。”比杨德不为所动。
<br/>　　“会让他们看到的，没有什么是一颗吝啬蔷薇的解决不了的。”帕里斯通抬起头，满脸阴冷的道：“如果有，那就两颗。”
<br/>　　比杨德眯起眼睛，刚想开口反驳。
<br/>　　“我会亲自把炸弹送到对方身边。”帕里斯通斩钉截铁：“你知道的，我有办法。”
<br/>　　比杨德顿时沉默。
<br/>　　此刻的苏重，还不知道有人在算计他。他正在一头扎进自己的实验室里。
<br/>　　魔龙法相短时间内无法提升，苏重把重心转向其他地方。比如那颗重力水晶，比如那个能够传送的黑门。他对这些奇特念能力，有着强烈的兴趣。
