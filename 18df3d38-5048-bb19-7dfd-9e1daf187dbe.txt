
<br/>　　<br/>　　    徐少中的公司很大，是单独的一栋写字楼，三十几层的高度，最上面的八层是他们办公的地方，余下的地方都租了出去，这栋楼得算是上饶地标性的建筑了，甚至在它刚盖起来的时候，几乎能够俯瞰整个城市，因为徐少中的楼是上饶最高的。<br/>　　<br/>　　    “你来这里做什么，别告诉我你真打算故伎重施，我跟你讲徐少中的身份太复杂……”<br/>　　<br/>　　    王惊蛰仰着脑袋，眯眯着眼睛说道：“这句废话，你已经讲过第二遍了，你不厌其烦，我却听的闹心了”<br/>　　<br/>　　    郭茜茜磨了磨牙，说道：“你不这么想，那你过来做什么？好，我问你，你打算怎么对付他？徐少中这个人很白，身上几乎没有任何的污点，就连一般公司最常见的偷税漏税做假账他的公司都没有，他还是个慈善家，捐助了不少的学校还修建了一座不以盈利为目的的医院，所以他在整个赣省的风评都很好，企业是明星企业，他也是个明星老板，你很难从他的身上找到什么突破口，所以在我看来你除非用强硬的手段对付他，要不然你是根本无从下手的”<br/>　　<br/>　　    “那你真的认为，他是干净的么？”王惊蛰低下头，反问了一句。<br/>　　<br/>　　    郭茜茜语气顿时一滞，才说道：“至少表面是这样的”<br/>　　<br/>　　    “我既不想绑了他，也没想杀了他，我只是想找出是谁指使他的罢了，很简单的一个事”<br/>　　<br/>　　    “很简单？”郭茜茜啼笑皆非。<br/>　　<br/>　　    “简单！”王惊蛰着重的点了点头。<br/>　　<br/>　　    “好好笑……”郭茜茜笑的露出一嘴小白牙。<br/>　　<br/>　　    王惊蛰却很认真的解释道：“能指使得了徐少中，然后他买通了一个降头师，神不知鬼不觉的去对付范成梁，这指使他的人，必然是徐少中需要仰望或者依仗的关系，也就是说这人是他必须要巴结的，那么问题来了，他肯定要比徐少中厉害多了，所以，你猜徐少中要是碰到不能解决的麻烦时，他会选择怎么做？”<br/>　　<br/>　　    郭茜茜几乎都没有多想，就脱口而出道：“如果是他自己都解决不了的麻烦，他肯定会选择求援”<br/>　　<br/>　　    “向谁求援？”<br/>　　<br/>　　    郭茜茜顿时不可置信的张着嘴，憋了半天后，惊异的说道：“你是说，徐少中会碰到麻烦，然后向这个指使他的人，求援？”<br/>　　<br/>　　    王惊蛰点头说道：“恭喜你，都会抢答了”<br/>　　<br/>　　    “不，不，不，这不可能，徐少中的企业做的很好，属于少见的优质企业，他怎么可能会碰到麻烦的，而且还是他解决不了的麻烦”<br/>　　<br/>　　    “我告诉你，官位做的越大，生意做的越大的人，麻烦就会越多，只是没有被纰漏出来而已，这个潜藏的麻烦就像是星星之火，一旦被点燃瞬间就会燎原了，从古至今都是如此，不管是奇货可居的吕不韦，还是江南首富沈万三，他们的生意做的牛逼闪电不？几乎都要冠绝天下了，可最后呢？”<br/>　　<br/>　　    王惊蛰意味深长的说道：“天底下的钱，哪里有那么好赚的啊，不用些特殊的手段能赚得来？开什么玩笑……”<br/>　　<br/>　　    郭茜茜结结巴巴的问道：“你，你，有办法了？”<br/>　　<br/>　　    “徐少中可能做梦都想不到，自己会碰上这种比天上掉馅饼还要小概率的事吧”王惊蛰拉开车门，半靠在座椅上，拿出手机后抬头问道：“你说，做生意的人最怕发生什么事？”<br/>　　<br/>　　    “生产事故，官方封查，还有资金链断裂”郭茜茜狐疑的盯着他，说道：“你该不会是说，这些事都会落到徐少中的头上吧”<br/>　　<br/>　　    郭茜茜这个警花很聪明，远不是胸大无脑的那一种女人，她几乎就是个女版福尔摩斯，什么事一点就透，王惊蛰跟她交流起来属实非常的舒服，至少能省了自己不少的口水。<br/>　　<br/>　　    随后，王惊蛰跟范成梁联系了下，告诉他自己要个懂得商业操作的团队，人员不用太多，但必须得要顶尖，最好是今天下午就能到齐。<br/>　　<br/>　　    这种事范成梁不难办到，他们范家下面就操盘着几家公司，并且在很多领域都是站在塔尖上的，招募的人员也都是精英类型的，他告诉王惊蛰自己这边马上联系，争取今天晚上以前，就让人马备齐了。<br/>　　<br/>　　    王惊蛰这属于是在替自己解决麻烦，范成梁自然比他要用心多了。<br/>　　<br/>　　    和范成梁通完电话之后，王惊蛰让郭茜茜等着自己，他独自一人进入了徐少中的那栋楼里，这一进去就是将近两个小时的时间，他几乎是将楼里上上下下都给转了一遍才出来的，下午，两人随便找了个地方吃了口饭，没过多久范成梁就告诉王惊蛰，他要的人已经乘坐飞机飞过来了，来的都是范家商业团队里面的扛鼎人物，人数不多就寥寥几个，一家商业银行的财务官，一个律师事务所的首席大律师，还有两人是地产公司里的运营总监和投资部的主管，一共四个人正在从京城和沪上飞抵上饶，范成梁也交代过他们，看见王惊蛰之后一切就以他马首是瞻，他说什么这帮人就照着做什么就行了。<br/>　　<br/>　　    下午四点左右，王惊蛰和郭茜茜在一家酒店里开了几个房间，他俩就等在一栋套房里，五点钟不到的时候，范成梁委派的人就先后过来了。<br/>　　<br/>　　    王惊蛰开始时对这个事的兴趣其实不是很大，但忽然之间却也来了感觉。<br/>　　<br/>　　    他很想看看，凭借他的手段和一些因素，要把徐少中给拖垮到要求援的时候，到底需要多少时间和手段能够办得到。<br/>　　<br/>　　    这个过程，想想也挺新奇的，这种团队作战的方式，他还第一次碰到。<br/>　　<br/>　　    以往他比较善于单打独斗，但并不排斥团队做战。<br/>　　<br/>　　    郭茜茜看看志得意满的王惊蛰，忽然觉得他这负自大的样，比较欠揍。<br/>　　<br/>　　    于此同时，阿难多的大弟子，也离泰进入了内地，他知道自己的游历在外的师弟大概在哪，那是从佛牌中传递的信息中得到的消息。
