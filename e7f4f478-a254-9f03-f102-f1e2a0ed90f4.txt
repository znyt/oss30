<br/>　　楚风眠大笑一声，他身上的气势猛然暴涨起来，一直压抑不动的境界，如今再度破退。

<br/>　　六尊生死境巅峰的魔尊，在加上之前所斩杀的剑苍穹等人，所有人的精血，终于是让楚风眠的境界，再度突破，达到了生死境四重的地步。

<br/>　　宝车所过之处，都是被上面恐怖的力量，直接粉碎。
<br/>　　小魔主催动宝车，直接冲着楚风眠撞了过来。
<br/>　　突破境界的一瞬间，其实便是灵力亏空的一刻，因为所有的灵力，都用来突破境界的时刻。

<br/>　　“宝车压天！”
<br/>　　小魔主怒喝一声，天魔宝车都是一声咆哮，无穷无尽的魔意爆发出来，碾压而来。
<br/>　　这天魔宝车，身为九魔城的镇宗之宝，不知道曾经有过多少强者，都是被这天魔宝车碾死在车轮之下。
<br/>　　现在这小魔主，便是也打算要将楚风眠，碾死在这车轮之下，为这天魔宝车，再添一份鲜血。

<br/>　　楚风眠手中灵剑一挥，猛然一剑，直接迎了上去。
<br/>　　在这剑刃之下，无数剑光，疯狂闪烁这，其中一道最为庞大的剑光，足有万丈之长，猛然冲着天魔宝车，斩了过去。

<br/>　　这万丈剑光，跟天魔宝车的力量，顿时撞在一起，形成惊天爆炸声。

<br/>　　楚风眠后退三步，看向这天魔宝车，脸色露出了几分认真。
