　　萧逸不知道毒袍的手段，或者准确来说，是不知道毒袍的武魂，到底是什么。
<br/>　　因为他明显感觉到，那些蛛丝之上，存在着武魂力量。
<br/>　　这代表着，那是武魂手段。
<br/>　　可即便不知道，即便如此。
<br/>　　毒袍仅仅一根蛛丝，便让得他费尽了九牛二虎之力。
<br/>　　那跟蛛丝，无声无息便进入武者气泉之内，缠绕元力。
<br/>　　若非他修了九阳轮回诀，而且六种火焰融合，有了元力也自成轮回的手段，还真不知道该如何奈何那根蛛丝。
<br/>　　还有缠绕在体外手臂的蛛丝，如此坚固，如此难以烧毁。
<br/>　　明明只是一根细微得肉眼难视的蛛丝，却连世间强悍火焰紫晶灵炎，都无法在短时间内奈何。
<br/>　　毒袍，现在可是仅仅只动了动手指，仅仅只用了一根蛛丝。
<br/>　　可他单凭这一手，怕是就能轻易掌控他人性命，哪怕这个人是绝世强者。
<br/>　　难怪毒袍的凶名如此之甚。
<br/>　　难怪毒袍二字，连一些霸主势力闻之都胆寒万分。
<br/>　　邪君府护法，果真非寻常人物。
<br/>　　“云雾里。”毒袍，淡漠地吐出三个字。
<br/>　　呼…
<br/>　　霎那间，森林之内的层层毒雾，顷刻暴走。
<br/>　　随便就能将毒雾覆盖整个辽阔妖兽森林，毒袍的修为和手段，可想而知。
<br/>　　嘭…
<br/>　　萧逸左手一甩，火焰奔腾而出。
<br/>　　左手之上，兽手覆盖其中。
<br/>　　那明明是一双兽手，可覆盖着，却丝毫不显突兀，反倒是完全烙印在皮肤之上。
<br/>　　萧逸不知道该如何形容，那不像是圣器、手套之类的戴在手中，有着覆盖感；而更像是仿佛就是从自己身上生长出来的手臂一般。
<br/>　　兽手，覆盖皮肤之上，完全没有丝毫不适，反倒如同自己的手一般，运用自如。
<br/>　　左手火焰轰出，袭来的毒雾，顷刻被冲散。
<br/>　　萧逸的控火能力，在这一瞬间大幅度飙升。
<br/>　　不，甚至于，控火的感觉，也不一样。
<br/>　　萧逸，虽然是个控火武者，也对自己的控火本事极其自信。
<br/>　　但，火，终究是外物；自己使用时，终究是操控着。
<br/>　　可如今兽手覆盖时，那种操控的感觉，却消失了。
<br/>　　取而代之的，是一种仿佛本能般的驾驭，更加微不可言，却更加强大。
<br/>　　那种控火的精准度、完美度、爆发度，还有强大度，远远超过了天地间被誉为火焰宠儿的火族血脉。
<br/>　　或者，对于兽手，再多的形容，都不如一个词来得准确，这个词，便是…强大！
<br/>　　呼…呼…呼…
<br/>　　黑气毒雾，从四面八方疾速袭来。
<br/>　　萧逸眼眸眯了眯。
<br/>　　袭来的毒雾，太多、太快了。
<br/>　　单凭一只兽手，根本无法完全挡下。
<br/>　　萧逸的右手，拳头握了握，兽手再次凝聚，覆盖其中。
<br/>　　轰…轰…轰…
<br/>　　萧逸双手齐出，两只兽手打出一道道火柱。
<br/>　　火柱，顷刻挡下了袭来的黑气毒雾。
<br/>　　“驭鬼丝。”毒袍，再次淡漠地吐出了三个字。
<br/>　　黑气毒雾，早便弥漫了整个森林。
<br/>　　但它们的袭来，更像是无边际、漫无目的地成片袭来。
<br/>　　可此时，毒袍的双手，完全伸出了黑袍中。
<br/>　　双手，快速抖动。
<br/>　　空气中，一根根蛛丝，密集而动。
<br/>　　毒雾，乃飘渺空乏之物。
<br/>　　可这些诡异的蛛丝，却轻松将其穿刺、缠绕。
<br/>　　密集的蛛丝，数以百计。
<br/>　　每一根蛛丝的尽头，必是缠绕着一团团浓郁毒雾。
<br/>　　毒袍手指轻动，数百蛛丝，齐齐破空而来。
<br/>　　数百团毒雾，宛若瞬间有了‘灵性’一般，漫天而至。
<br/>　　萧逸双手齐出，一道道火柱打出，火柱，渐成火海。
<br/>　　大片火海，准备挡下这些毒雾。
<br/>　　只是，原本的毒雾，是飘荡冲击而来。
<br/>　　可现在蛛丝缠绕下，却是破空而来。
<br/>　　后者的冲击力，显然更为惊人。
<br/>　　一团团毒雾，瞬间冲破了火海，直噬萧逸而来。
<br/>　　“云雾里，驭鬼丝。”萧逸双眼一眯，“好诡异而惊人的手段。”
<br/>　　吼…
<br/>　　忽然，一道震天龙鸣，奔吼而出。
<br/>　　萧逸双手成爪，道道紫炎瞬间打出。
<br/>　　紫炎，顷刻化作游龙。
<br/>　　巨大的火焰游龙，发出震天之吼。
<br/>　　如果说，以前的萧逸，是操控紫炎游龙杀敌。
<br/>　　那么，现今两只兽手之下，则是完全驾驭紫炎游龙。
<br/>　　这是两种完全不一样的层次。
<br/>　　蛛丝缠绕着袭来的毒雾，异常快，又异常精准。
<br/>　　可紫炎游龙，同样异常快，异常精准。
<br/>　　两条巨大的紫炎游龙，身躯游摆间，轻松挡下了所有毒雾。
<br/>　　身躯之上，火焰沸腾。
<br/>　　“果然有些本事。”毒袍头一次眯了眯眼。
<br/>　　手指，再次动了，不过动得更加快速。
<br/>　　嘶…嘶…嘶…
<br/>　　空气中，越来越多的蛛丝出现。
<br/>　　越来越多的毒雾被缠绕。
<br/>　　不过片刻，已是成千毒雾化团而至。
<br/>　　萧逸眉头紧皱，又是数头紫炎游龙凝聚而出。
<br/>　　两只兽手下，轻松驾驭。
<br/>　　对比以前仅仅是凝聚出紫炎，操控游龙，动辄数十条、百条。
<br/>　　现今，仅仅是兽手之下，驾驭数条紫炎游龙，反倒显得压力更大。
<br/>　　那种驾驭，对萧逸元力还有心神，皆是不小的消耗。
<br/>　　但这区区数条紫炎游龙，却比以前简单打出，随后操控的数十、数百条紫炎游龙，更加强横。
<br/>　　如果真要说区别的话，那便是现今驾驭下的数条紫炎游龙，宛若真实存在，多了灵性。
<br/>　　灵动、狂猛，而睥睨天下。
<br/>　　嘭…嘭…嘭…
<br/>　　萧逸双手驾驭之下，再度加入了数种火焰。
<br/>　　地脉金火、星辰之火、十界灭生火。
<br/>　　三色火焰，顷刻融入数条紫炎游龙之内。
<br/>　　现在的火焰游龙，已是四色光芒。
<br/>　　任凭袭来的蛛丝毒雾强横，却丝毫奈何不得。
<br/>　　看真切些，毒袍的手指，动得愈发剧烈，愈发快。
<br/>　　数以千计的蛛丝，缠绕着毒雾，在四周宛若一团团毒雾鬼魅。
<br/>　　同时操控如此多跟蛛丝，毒袍的手指，丝毫不显混乱。
<br/>　　数千毒雾鬼魅，宛若张牙舞爪，构成了一副阴寒而杀意凛然的鬼魅之舞。
<br/>　　萧逸双手驾驭下的数条四色游龙，同样防御得密不透风。
<br/>　　四色游龙，蜿蜒游摆，如同数条在毒雾之内嬉戏游走的庞然大物；又如云间翻腾，风云相随的龙腾之兽。
<br/>　　“雕虫小技罢了，给我破。”毒袍，头一次露出了略微难看的脸色。
<br/>　　“哼。”萧逸冷哼一声。
<br/>　　蛛丝之下，鬼魅之舞，变得更为汹涌密集。
<br/>　　可是，四色游龙，却如潜龙而起，无物可当。
<br/>　　二者的交锋，忽然在这一瞬间，爆发至极致。
<br/>　　“爆。”
<br/>　　“爆。”
<br/>　　两声暴喝，同时从毒袍与萧逸口中喝出。
<br/>　　蛛丝操控下的毒雾鬼魅，萧逸驾驭下的紫炎游龙。
<br/>　　二者，同时爆发。
<br/>　　轰…轰…轰…
<br/>　　也是这一瞬间，整个天地，蓦地一白。
<br/>　　剧毒与火焰的交锋，席卷肉眼可见的整个天空。
<br/>　　待得剧毒与火焰同时消散，整个妖兽森林，已被夷为平地。
