　　司徒青扬问过佟千雪和慕容飞在东海市没有住处后，便主动提出让他们住在万草堂附近一个独立院子里，那里也同样是司徒家的产业。
<br/>　　慕容飞拿完药回来，司徒青扬已经把该叮嘱的事情都对佟千雪进行了叮嘱，并且把独立院子的钥匙交给了佟千雪。
<br/>　　独立院子距离万草堂的距离并不算远，走路也就十来分钟的路程，用钥匙打开院子门，走进这个同样古色古香，江南园亭风格还带有荷花塘的独立宅院时，慕容飞和佟千雪皆是吃了一惊。
<br/>　　就以目前东海市的房价来说，这么一个独立宅院的售价至少在2亿，即便这里离市中心有点远，但如果司徒家真的要把这里出售的话，相信会有大量土豪排着队要来买。
<br/>　　这年头，想要拥有这么别致的独立宅院，远比拥有一栋豪华别墅要难得多。后者只要有钱就能买的到，而前者，是有钱都未必能买得到。
<br/>　　因为佟千雪在这里要治疗一段时间，而两个人来这里也没有携带什么换洗衣服和生活用品，所以把配好的药放下后，两个人又是打了车去市中心购买生活用品。
<br/>　　刚下出租车，就有一个穿着夹克，双手插在口袋里，不时向过路行人兜售二手手机的男子朝着两个人走来。
<br/>　　“先生要手机吗？全新的iPhone，三星，质量可靠，国行正品，可开发票。”男人从夹克衫口袋里拿出一个成色很新的iPhone给慕容飞看了眼。
<br/>　　“不需要。”
<br/>　　慕容飞摆了摆手，往旁边让开，就要往前走。
<br/>　　这种在大街上偷偷摸摸向来往行人兜售手机的人在全国各地都有，尤其是在一些学校门口更是猖獗，他们以远低于行情的价格向经济并不宽裕的学生兜售，很受青睐。
<br/>　　这些手机虽然便宜，但是质量却堪忧。有些是偷来的黑手机，有些是维修后的翻新机，花钱能买来可以使用的手机都是好的，有时候买来的手机只是一个模型。
<br/>　　慕容飞读大学的时候，还没有现在苹果，三星也没有现在这个火爆，那还是诺基亚称王称霸的时代，一部当时售价3000多的N81绝对是大学生群中身份的象征，但是高昂的价格却是让绝大多数学生望而却步。
<br/>　　慕容飞同寝室一个哥们在周末回学校的时候就遇到一个卖黑手机的家伙，向他出售一部九成新的N81“黑货”，只要900块钱。那哥们立刻就心动了，跑回宿舍借钱，好不容易凑满了900块钱，找到那个卖“黑货”的家伙，把他手中那台九成新的“N81”给买了回来。
<br/>　　买的时候，慕容飞寝室的哥们还特意留了个心眼，让对方把手里的手机演示了一下功能，确认无误后，才付钱交货。可是等到那哥们把新买的“N81”拿回寝室炫耀的时候，傻眼了。说好的高清摄像不能使用了，说好的各种牛逼功能没有了，这特么根本就不是一部手机，而是一个制作精良的模型！
<br/>　　这可把那哥们郁闷了好久，那个时候900块钱是一个月的生活费，自己东借西凑买来想要装逼的“N81”却只是一个完全不能使用的塑料壳子。那哥们儿仔细地回忆了交易过程，却始终没想明白自己到底是怎么被坑的。
<br/>　　也正是因为这件事情，让慕容飞对这种路边兜售手机的“个体商”没有什么好感。
<br/>　　慕容飞跟自己眼前卖手机的“个体商”错开一个身位，可是对方却没有要放弃的意思，跟着一步又是挡在了慕容飞面前，把自己手中崭新的iPhone递到慕容飞面前，说道：“兄弟，你可以看看嘛，质量绝对没话说。”
<br/>　　慕容飞皱眉，正欲再次拒绝的时候，对方一句话却是让慕容飞把话给咽了下去，“通过这部手机联系。”
<br/>　　闻言，慕容飞脸色变了变，接过对方手中的手机，顺手塞进了自己的夹克衫口袋里。整个过程非常迅速，再加上有两个人身体的掩护，以及周围走动行人的掩护，以致于就连距离慕容飞最近的佟千雪都没有注意到慕容飞跟他眼前的男子之间发生了什么。
<br/>　　把手机交给慕容飞之后，男子压下头上的鸭舌帽，双手揣进口袋中，迅速离开，走了一段路又开始向周围的行人兜售手机，既然是演戏自然是要演全套，表演要到位。
<br/>　　“怎么了？”佟千雪发现自己身后的慕容飞不见了，转头去寻找的时候，看到慕容飞匆匆跟了上来，便是多嘴问了一句。
<br/>　　“哦，刚才那个男人。”慕容飞指了指那个还在向路人兜售手机的男人，说道：“想要把他不知道从哪里捣鼓来的手机卖给我，被我拒绝了，耽搁了一点时间。”
<br/>　　佟千雪不疑有他，点了点头，挽着慕容飞的手走进了商场。
<br/>　　看到慕容飞和佟千雪走进了商场，兜售手机的男人和正在讨价还价的情侣打了个眼色，三个人立刻分散进人群中。
<br/>　　这附近除了这个兜售手机的男人，云岳还安排了不少人打掩护。而这些人全都是特勤中队的人，而且还是今年刚进的新人，这也算是对这些“菜鸟”的一次考核。
<br/>　　慕容飞和佟千雪买了一些生活物品和换洗衣服便打车回到了万草堂附近的独立宅院，回到独立宅院的时候，慕容飞发现之前那个带着他们去见司徒青扬的年轻女子也在。
<br/>　　“慕容先生，佟小姐。”年轻女子向慕容飞和佟千雪打招呼，解释道：“煎药有些讲究，大掌柜担心你们会把药煎坏破坏了药性，就让我来帮忙，以后我负责帮忙煎药，同时负责佟小姐的饮食搭配，我就住在你们旁边的厢房，你们有事可以直接招呼我，你们叫我小秋就行。”
<br/>　　“替我谢谢司徒先生，不但安排我们住处，还要你来照料千雪。”慕容飞微笑着对小秋说道。
<br/>　　“应该的。”小秋回答道：“司徒先生说了，你们是小少爷的朋友，就是司徒家的朋友，这么做都是理所应当的。”
<br/>(本章完)
