　　诛仙再出，凶阵再战，凌厉剑气破空，圣人大战重启。
<br/>　　与此同时，九州南方同样爆发了惨烈的人魔大战，南方魔君蚩尤与血尊者，毒王等一众劫魔殿高手，大战人族雨皇雨伯、风皇风后、剑皇赤松子等数位皇道高手。
<br/>　　双方大战，自是惊天动地，血流成河，无数人族被劫魔灭杀，但劫魔同样也有许多被人族高手所灭。劫魔没有固定的形体，只要灵魂不灭，便能随时聚散无形，他们身体是由劫气魔雾等组成。但正因如此，他们的实力普遍不高，只需找到正确的方法，抹除灵魂，便可灭杀劫魔。
<br/>　　无尽鲜血流入劫魔转生池之中，又有无尽灵魂被吸入其中，不断转生出新的劫魔出来战斗，高空之上，一柄黑幡不断摇动着，如同一只饕鬄凶兽，爆发出强烈的吞噬之意，将无尽生灵的灵魂吞入其中，黑幡一晃，数道神魂攻击打出，无数人神魂遭受重创。
<br/>　　这是劫魔炼魂旗！
<br/>　　魔威滔天，熊熊魔焰燃烧苍穹，奏响苍凉魔歌！
<br/>　　人魔大战注定是一场惨烈无比的大战，为人，当斩妖除魔，为魔，当魔屠苍生！
<br/>　　“蚩尤，上古之时，你已败于人皇之手。若是回去好好躲着，还没人能够伤你，此刻却出来祸乱人间，真以为没人能够收拾你，看剑！”
<br/>　　一声大喝，在空中爆响，随后，一道如烟花般璀璨的剑芒劈开虚空，一道巨大裂缝横空，向着大战中央，一个数万丈高的巨汉劈来。
<br/>　　“哈哈，赤松子，上古三族大战，你便屠我巫族，今日我蚩尤定要为死去的巫族兄弟们报仇，你也吃我一刀。”
<br/>　　蚩尤怡然不惧，面上有股令人心悸的凶悍之气，全身魔气纵横，魔威盖世，手中虎魄魔刀携嗜血凶威，一刀劈出，血**焰便遮蔽半个天空，迎着那道剑光斜劈回去。
<br/>　　轰隆隆！
<br/>　　魔刀将剑气湮灭，带着一股不可阻挡地气势继续向着赤松子斩去。赤松子一道剑光再次斩出，带着煌煌如大日般的威压，将那魔刀的刀气斩破。
<br/>　　“杀！”
<br/>　　两人大战在一起，浴血奋杀，刀剑相交，锵锵作响，魔道与武道都是极为霸道的大道，战斗起来，也颇为凶悍。
<br/>　　魔，何为魔？损人利己即为魔！
<br/>　　武，何为武？止戈即为武，虽然每个人的武道都不相同，但学武的最初目的并不是为了与人争斗，若真是想与人争斗，那不学也罢。
<br/>　　“夏雨化剑！秋雨悲情！冬雨死亡！”
<br/>　　雨皇不愧为雨皇，挥手间便使出无尽神通，夏雨最为狂暴，倾盆大雨顿时化为无数利剑，自九霄降落，一落便带着一片生命，淅淅沥沥，无尽剑气自雨中散开，斩落群魔。
<br/>　　秋雨绵绵，却带着一种莫名的悲秋之意，看似落在人身上，实际上却是落在众人内心深处，将人心中的悲伤引出。群魔都是人死后灵魂所化，只要是人，生前都有令人悲哀之事，此刻那心中的悲伤被扩散开来，一股莫大的悲意，悲恸上天，弥漫于这天地之间。
<br/>　　冬雨最为稀少，淅淅沥沥的冬雨之中，带着一种死亡的意境，冬雨落下，便是死神收走生命的时刻。万物枯灭，生命寂灭，生死注定，一场冬雨带走一片灵魂。
<br/>　　“控血之术，血爆！”
<br/>　　另一边，一身血红色衣袍，满脸苍白色的血尊者，使出一手血爆之术，直接掌控他们血也流动频率，控制他人自爆。
<br/>　　一片血雾炸开，如同一朵朵烂漫的红色花朵在虚空绽放，而且还是一片片地绽放，怎么看都有一股凄美的气氛弥漫其中。
<br/>　　毒王更毒，一大片的毒雾弥漫开来，遇谁谁死，毒雾散在空气之中，专门针对人族高手，一旦毒雾入体，不到一时三刻，便会化为浓水。
<br/>　　雨皇对上毒王，用雨水冲刷毒王所施之毒，但毒王还有各种毒虫，漫天飞舞，噬毒虫，不仅能吞噬毒物，更能释放出剧烈无比的毒，一样咬谁谁死。
<br/>　　风后对上血尊者，无尽风刃切割空间，形成数道风暴追着血尊者袭来，幸好血尊者的血盾之术够快，不然便倒霉了，但就算如此，他也被追得狼狈不堪。
<br/>　　最为诡异的是咒言师，一手诅咒之术，没有几人能够抵挡住他的三次诅咒，他的对手是人族盘王。盘王精通蛊术，两者有些相似，但却完全不同，都是与十分诡异的秘术，是故，两人的战斗虽然不似其他人那便激烈，但却是最为凶险的，一个不慎，便会命丧黄泉。
<br/>　　人族之中，还有一个青年模样的道人，他手持一柄长剑，一剑一人，从不多杀，也从不少斩，每剑只杀一人，他所过之处，身后没有任何劫魔存在。
<br/>　　“桀桀……小子，你去死吧！”
<br/>　　突然，一道令人难受的声音传入他的耳中，青年抬眼一看，看到一位身穿黑袍，全身瘦得只剩皮包骨头的老者，他双眼之中冒出诡异的绿色鬼火，手持一柄黑色骨杖，向着青年一杖点来。
<br/>　　青年瞬间感觉到自己的魂魄不受自己控制，好似要脱离自己而去一般，这十分诡异。眼前这老人便是劫魔殿中的鬼道人，能够御使鬼怪，更能操控他人之魂魄，让人无法抵挡。
<br/>　　之前便御使无尽鬼魂，将无数人族高手给吞噬了，造成无穷杀孽。
<br/>　　“仗剑诛妖邪，除魔天地间！斩——”
<br/>　　青年一声大喝，身体漂浮起来，一个缓慢旋转，手中长剑好似分成无数，上千剑影绕着青年一周旋转，最后汇成一剑，一剑向着那鬼道人劈去。
<br/>　　并没有璀璨的剑光，也无惊人的剑意，只有平平凡凡的一剑，一剑枭首。
<br/>　　“你——”
<br/>　　仅仅一剑，看似极为缓慢的一剑，鬼道人甚是都来不及闪躲便被斩去头颅。
<br/>　　“快慢真意，你是谁？”
<br/>　　“武当山太玄真人！”
<br/>　　青年收剑，缓缓答道，闻言，鬼道人头颅一歪，滚落在地。
<br/>　　与此同时，西方灵山，亦是一片大乱，爆炸声不断响起，魔气沸腾，劫魔殿正在与西方教开战。
<br/>　　“杀杀杀——”
<br/>　　那是阿修罗族的喊杀声，阿修罗是一种嗜杀好战的种族，男的丑陋无比，女的却又极为漂亮。西方教本就没有多少弟子，此刻被屠杀一空，更有数人的真灵入了封神台，如弥勒、水火童子、阿逸等人。
<br/>　　万劫魔尊一身黑衣在风中猎猎作响，他脸色雪白，并不是受伤，而是他的脸色本身就苍白无比，给人一种病态的感觉。他一头黑发随风飘散，更添几分霸气。
<br/>　　“接引，你的末日到了！”
<br/>　　万劫魔尊冷笑一声，他手中赤色嗜血魔刀一挑，对着前方的两人斩去，此二人便是接引的执尸与恶尸。
<br/>　　“阿弥陀佛，我不入地狱谁入地狱！”
<br/>　　接引双手合十，错分开来，一道遮天巨掌向着万劫魔尊拍来，掌中带着浓浓禅意，以及无尽佛光。
<br/>　　佛光与魔光相触，消融于空中。魔佛本就相对，双方相互克制，只是看谁强谁弱罢了。
<br/>　　另一边，冥河老祖双手持元屠阿鼻两柄杀剑，头顶业火红莲，向着准提的恶尸攻去，剑光之中，是黑色的杀伐意境，冥河本就修炼杀道与血道，杀伐果断，剑出即死人，不沾血的剑更是不能回鞘。
<br/>　　数个回合，两人战得旗鼓相当，两人修为相仿，都是准圣圆满，但准提本尊身为圣人，连着恶尸对道的领悟也加深了，是以冥河能够与之抗衡，除了杀道的凶狠，还与一身厉害的先天法宝有关。
<br/>　　红莲当空，业火无尽！
<br/>　　冥河能否证道，就看他此次能否以手中杀剑，屠灭圣尸？
