　　天地奥妙，值得自己穷极一生去探寻！
<br/>　　想到此处，元始心里突然放开了，现在担心又有何用，不如做点实在的事情。当务之急，是提升自己的实力，至于最后如何，就看最后了。
<br/>　　不管结果如何，至少我曾经努力过，奋斗过，追求过，不曾放弃过。我就不悔！
<br/>　　将心态调好，元始感觉自己的境界更加圆滑了，只要再潜修几年，就能真正突破到大罗金仙圆满之境。
<br/>　　感受到自身的变化，元始思索道：“这就是心态问题吗？修行即修心，这话说得一点没错。”
<br/>　　不再多想，元始再次将注意力放到眼前这座神山之上，这一次，虽然依旧震撼，但已经不再如先前一般敬畏了。
<br/>　　心中起念，步步登天，修炼到如今，元始意志何等坚定，初始只因想到盘古，如今回过神来，管他天高地厚，也欲要一攀天柱绝险，他能否成功登临九天之顶，一探不周究竟？
<br/>　　煌煌不周山，巍巍天地关！
<br/>　　继承着盘古神尊的意志，支撑着整个洪荒宇宙的天地，周山天柱，直耸九天，宛若一尊万古不朽的神灵化身，俯视着整个洪荒天地，万物生灵！
<br/>　　元始顶着莫大的天地威压，艰难的向山上攀爬而去，一步一步，每一步，都要付出巨大的努力，心神意志亦同受磨练。
<br/>　　走在这大山之间，元始内心有一种违和的亲切感，像是失散许久的游子找到了自己亲人一般，那般的情切。
<br/>　　元始知道，这是因为他是盘古元神所化，不周山又是盘古脊骨所化，两者本质上都是由同一人所化，故而，元始会倍感亲切。
<br/>　　元始并没有多想，而是默默地体会着这种感觉，一步又一步，继续顶着莫大威压向上走去，仔细地感受着盘古独有的意志。
<br/>　　承受着这股威压的同时，元始心中在默默思索，相传不周神山后来被水之祖巫共工一怒之下，撞断了，从此天柱倒塌，天地崩裂。
<br/>　　这怎么可能？这是元始的第一感觉。不周神山，以元始现如今的修为，想攀爬到顶都十分困难，想要用头撞断不周山，那简直是天方夜谭啊！
<br/>　　但，事实的确如此，这其中是否有某种隐秘？还是共工就真的如此强大？
<br/>　　元始有些不信，共工按照传说来看，当时就是一个祖巫，了不起，也就堪比准圣后期修为，怎么可能撞断有如此威压的不周神山，估计他还没有靠近，就会被不周山的威压给压趴下了，还怎么撞断？
<br/>　　故而，元始更相信这其中有什么隐秘。
<br/>　　比如，有什么外力的帮助？
<br/>　　那么，这人又为何要帮助共工撞断天柱呢？天柱一断，天就塌了。这对他又有何好处？
<br/>　　这一切的一切，好似一个谜团，又好似一个大局？元始默默思索着。
<br/>　　不周山并不像传说一样，这里没有遍地仙草灵根，这里也没有强大禁制结界，除了残留盘古意志形成的天地威压外，这座山就是光秃秃的一片，就像是一块普通的坚硬大石！
<br/>　　真不知道后来这里为何会是仙草灵根满地，天才地宝随处可见，难道是威压在渐渐消散了。
<br/>　　呃？！威压渐渐消散？
<br/>　　元始突然想起，也需正是不周山的威压消尽，后来的共工才能撞倒这不周天柱。只是，就算这威压消失殆尽，不周神山也不是那么容易撞断的呀！
<br/>　　心中默默思索，元始再次踏步向前。此时的他，已经攀爬了数万米的距离，不过，对于整个不周山的高度而言，他其实还在山脚处，但即便如此，他却已经步履艰难，每踏一步，都要蓄足力气。
<br/>　　天地初开不久，盘古残存的意志还很强大，形成的天地威压更是可怕，别说大罗金仙级别的元始了，就算是混元大罗金仙一般的存在，想要登上这不周之顶，也绝不是一件容易的事！
<br/>　　这个时代，洪荒有些特殊！
<br/>　　杀劫刚过，转劫的先天生灵要重新来过，他们还远没有完成蜕变，他们虽然根基雄厚，潜力无限，但，在潜力没有完全转化为实力之前，终究只是潜力。
<br/>　　同时，这个时代也很强，那些秉承盘古遗留的造化之力而生的诸多先天神邸，强者辈出，大神通者比比皆是，堪称万族争辉！
<br/>　　元始同样如此，他既可以算作是转劫的先天生灵，又可以算作是盘古遗留的造化之力孕育而生的先天神邸。他的潜力依然没有转化为实力，一样没有达到巅峰境界，终究是不能完全抵御盘古意志所形成的天地威压！
<br/>　　一步一步，倾尽全力，元始不断往上，耗费数日时间，他又攀上了数万米的高度，强大的威压，是最好的磨砺，比登山之前，他一身气息更加凝练了！
<br/>　　他停了下来，没有再继续向上。
<br/>　　修行不是一味的逞强，现在的他，需要的是静心，静下心来参悟不周山中隐藏的奥妙，与残存的盘古意志交流，推演天地规则奥妙。
<br/>　　盘坐在一块山石之上，感应着不周山那磅礴巍峨、坚韧不屈的浩瀚气息，渐渐地，元始心中杂念尽皆消散，取而代之的是一股与天斗、与地斗、与神斗、与魔斗、与天地众生斗、永不屈服、永不放弃的信念，这便是盘古残存的一点意志！
<br/>　　渐渐地，元始身上散发出一种斗天战地的气息，渐渐地，他心中意志圆满，修为自然而然地突破了，修为真正踏入了大罗金仙圆满之境。
<br/>　　不过，元始并没有起身，而是继续盘腿坐下，继续参悟盘古意志，他有盘古传承，如今再与盘古残存意志交流印证，能够加快他吸收盘古的传承。
<br/>　　“呼……”
<br/>　　仿佛过了千万年之久，又好似一瞬之间，元始闭上双眼，深吸一口气，在长呼出来。顿时，只见两道犹如实质般的白雾从他鼻孔之间滚滚而出，直冲百丈开外，方才渐渐消散。
<br/>　　等他再次睁开双眼，双眼之中，尽是一片清澈，只有双瞳最深处，隐隐有一股战天斗地的强大意念。
<br/>　　却见元释也不起身，他双手结印，一个复杂玄妙的大印缓缓而出，这大印就像是一座缩小无数倍的神山，这，就是不周山。
<br/>　　在这大印出现的同时，缓缓放大，与此同时，一股强大的天地威压瞬息而至，在这股威压之中，有一股与天斗、与地斗、与众神斗、与苍生斗的斗天战地的不屈信念。
<br/>　　这大印一出现，就抵消了元始周边的天地威压，同时，也向着那片威压反抗而去。
<br/>　　不周印！
