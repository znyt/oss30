　　轰隆！
<br/>　　一道长达数千米的金色巨枪，陡然出现在九天之上，犹如一根巨大的擎天柱，朝叶尘狠狠砸落下来。
<br/>　　不得不说，这紫袍老者，到底是出自无极宗的高手，不但修为深厚，战斗经验和对法宝的运用，也全都达到了炉火纯青的地步，即便是面对真正的返虚天君，也丝毫不弱。

<br/>　　相比之下，叶尘似乎显得有些措手不及，面对对方的这惊天一刺，还完全没有反应过来，只能勉强举起灭世魔刀，迎面挡了上去。
<br/>　　铿！！
<br/>　　一道惊人的撞击声，响彻整个天地之间，叶尘的灭世魔刀，和紫袍老者的金色长枪，狠狠撞击在一处。
<br/>　　虽然两件法宝的等级相差悬殊，可是法宝，终究是要靠使用者发挥实力的。

<br/>　　地面之上，顿时出现了一个巨大的深坑，犹如陨石坠落一般，连同周围方圆数千里的山河，都跟着一阵巨颤。

<br/>　　“那灭世魔刀，也不愧是绝世神兵，相隔这么远，都能感受到那股令人心悸的毁灭气息！”
<br/>　　“可惜了，那个姓叶的神秘年轻人，到底是太嫩了，纵然有绝世神兵，恐怕也不是这位特使大人的对手！”
<br/>　　……
<br/>　　众人纷纷议论着。
<br/>　　冥沧荨听到众人的议论声，俏脸顿时露出浓浓的担忧之色，心中暗骂叶尘太过托大，根本就不该追上去，
<br/>　　“原本都已经把对方吓跑了，干嘛还非得追上去？真是笨死了！！”
<br/>　　也难怪冥沧荨会这么想，毕竟那紫袍老者的实力，实在是太强悍了。
<br/>　　叶尘和他之间的实力差距，即便是一把绝世神兵，恐怕也无法弥补。
<br/>　　“哈哈哈！小子，本尊早就说过，灭世魔刀，不是你有资格拥有的！！”
<br/>　　紫袍老者占据优势之后，顿时兴奋的哈哈大笑，当即再次挥舞着金色巨枪，朝下方深坑之中，狠狠刺落了下去！
<br/>　　只要能从对方手里得到灭世魔刀，那就是大功一件，等回到无极宗之后，他必然会被宗主云苍生大加赞赏，成为无极宗长老，几乎是班上钉钉的事情，从此一份冲天！
<br/>　　一想到这里，紫袍老者顿时更加兴奋，将全身的真元，全都注入了金色巨枪之中，苍老的脸庞之上，也露出了一抹狰狞之色，
<br/>　　“小子！给我，去死吧！”
<br/>　　轰隆隆！
<br/>　　紫袍老者的一击，没有一丝一毫的保留，威力比之前更加逆天！
<br/>　　那道巨大的金色巨枪，还没有落下，下方百里之内的地面，就已经被那道强大的气势压迫，压的凹陷了下去。
<br/>　　毫不夸张的说，紫袍老者的这一枪，若是放在地球之上，应该足以将整个地球捅出一个窟窿来！
<br/>　　如果不是眼下这个星球，比地球要大上百倍不止，对方这一枪下去，整个星球恐怕就要崩塌了！
<br/>　　而这，就是返虚真君的威力！
<br/>　　一击，足以毁灭一个小型星球！
<br/>　　“快！快撤退！”
<br/>　　“太强大了！！”
<br/>　　“那小子这次肯定死定了！”
<br/>　　……
<br/>　　远在千里之外的三大家族的众人，看到这仿佛要毁天灭地的一击，顿时吓得纷纷倒退，远远的退开，生怕会波及到自己。
<br/>　　唯独冥沧荨，依旧站在原地，俏脸煞白无比，
<br/>　　“完了！在如此强大的一击之下，他，还能活命吗？”
<br/>　　冥沧荨几乎已经不抱任何希望。
<br/>　　轰隆隆！！
<br/>　　伴随着那璀璨夺目的一枪狠狠落下，
<br/>　　方圆千里之内的地面，都重重陷落了下去，而位于最中心叶尘所在的位置，更是深不见底。
<br/>　　原本地面上的一座高山，硬是被那杆巨枪，捅成了一个深不见底深渊！
<br/>　　“哈哈哈哈！”
<br/>　　站立九天之上，宛如战神般的紫袍老者，顿时再次发出一道得意的狂笑声，

<br/>　　只见下方，那深不见底的深渊之中，一道金色人影冲天而起，瞬间直冲九天云霄！

<br/>　　就在紫袍老者开始怀疑人生的时候，叶尘没有任何犹豫，直接干净利落的一刀，狠狠斩落！
