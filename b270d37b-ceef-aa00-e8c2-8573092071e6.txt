　　寂静的夜，静悄悄的溜走，竖日清晨，天色刚蒙蒙亮的时候，沉睡中的慕清霄就习惯性的睁开双眼。
<br/>　　“那件东西就在青山镇附近的魔兽山脉中，必须拿到手才行，正好去碰碰运气，看能不能找到炼制三品丹药的辅助药材。”
<br/>　　慕清霄实在是有些无语，青山镇毕竟太小，连一些低等药材都找不齐全。
<br/>　　从床榻上跃起，慕清霄在客房中简单的洗漱了一番，来到楼下退过客房后，便转身离去。
<br/>　　片刻后，慕清霄就来到小镇门口，此刻已有不少的佣兵簇拥在此，一声声吆喝，不断的招呼着想要进山脉中的单独佣兵。
<br/>　　小镇上的佣兵，大致分为三种。
<br/>　　第一种便是已经组成严密组织的佣兵团，这种佣兵团，在青山镇中只有三支，人数都在百人左，团员实力大多都在斗者之上。
<br/>　　而且，说三支佣兵团的团长，都是斗师级别的强者，可以说是青山镇中最强大的势力。
<br/>　　第二种佣兵，便是一些临时的佣兵队伍，这种队伍一般都是做完一次任务便解散，彼此的信任度以及配合默契度，远远比不上那种正规的佣兵团。
<br/>　　第三种佣兵，便是那些单人独行地佣兵，这些人一般都是有着几分保命的底牌，此时小镇门口处这些大声吆喝的佣兵，便是属于第二种。
<br/>　　隐匿气息站在人群中，根本就没人能够发现他，慕清霄是在碰运气，若是记得没错，在原著中，最近小医仙也会进入魔兽山脉。
<br/>　　正当慕清霄的视线在人群中扫视的时候，小镇门口处忽然就骚动起来，陡然响起一阵兴奋的大喊声。
<br/>　　“万药斋要进魔兽山脉采药了，同行的还有小医仙哦，名额只有五十名，实力要在二星斗者之上，各位抓紧了！”
<br/>　　大喊声使得喧闹的小镇门口微微寂静片刻，众人面面相觑了一会，那些达到了标准的佣兵，顿时争先恐后的对着那名身着万药斋服饰的中年人涌去。
<br/>　　慕清霄嘴角牵起一丝笑意，看来自己运气不错，万药斋正是昨日他购买药材的地方，于是悄无声息的挤进人群中。
<br/>　　小医仙的名头，在这青山小镇里，简直比什么都管用，五十个名额，几乎使得众人抢破了头。
<br/>　　“还有最后一个名额！”中年人笑眯眯的摇了摇手中的羊皮卷，对着拥挤的人群笑道。
<br/>　　“我来。”
<br/>　　慕清霄平静温和的声音响起，缓缓来到中年男子面前。
<br/>　　望着面前这富家公子哥一样神骏的青年，中年男子明显一愣，旋即笑道：“这位公子，我们的条件是需要二星斗者，而且魔兽山脉中危险重重，若是遇到什么不测，我们可不担责任。”
<br/>　　中年男子特地强调了下‘不担责任’四个字，显然将慕清霄误认为了某个家族的大少爷，看你穿着打扮，也不像是缺钱的样子，何必冒这种风险？
<br/>　　“我在青山镇这么多年，怎么没见过这位公子哥？”
<br/>　　“啧啧，你看他那白嫩嫩的小脸，就跟女子似得，真特么闲着蛋疼，没事凑什么热闹啊？”
<br/>　　对于周围的议论声，慕清霄充耳不闻，攥着拳头，轻飘飘的轰在最近的一颗树上，没有任何斗气波动，也没有任何花哨的招式。
<br/>　　“咔嚓！”
<br/>　　伴随着一声清脆的声响，那颗颇为粗壮的树干，在周围人群惊愕的目光中，轰然倒地，顿时风沙漫天，现场陷入一片死寂。
<br/>　　“你看我可有自保的能力？”慕清霄朝着中年男子微笑道。
<br/>　　“咳咳，果然人不可貌相，公子年纪轻轻就修炼到了二星斗者，真是前途不可限量啊。”
<br/>　　中年男子惊叹的说道，其实他根本看不出慕清霄的修为，但是仔细想想，估计也就二星斗者左右，年纪轻轻的，能强大哪里去？
<br/>　　“好吧，最后一个名额，便给你了，佣兵酬金是五百枚金币，来回护送我们万药斋采药队的安全，待会先付给你一半的酬金，另外一半，则要回来以后才能付给。”
<br/>　　“可以。”
<br/>　　微笑着点了点头，慕清霄当然不会在意区区五百枚金币，炼一次丹药都不止这么多，他的目的是魔兽山脉中的某件东西，而佣兵团能够给他指路。
<br/>　　或者说，佣兵团队中的小医仙能够给他指路。
<br/>　　见到最后一个名额，被不知何处冒出来的青年公子哥抢走，周围围观的佣兵们，只得悻悻的离开。

<br/>　　寂静的森林中，大队人马安静的行走着。
<br/>　　森林中光线阴暗，一双双警惕的目光，不断在周围树林中的阴暗处扫过，紧握着武器，随时准备应付一切突发状况。
<br/>　　作为已经在魔兽山脉混了多年的老佣兵，互相之间虽是第一次合作，可却都能保持着基本的默契。
<br/>　　眼神交错间，也能从对方眼神中识别一些代表危险与安全的型号。
<br/>　　走在佣兵团的最后面，慕清霄自然是最轻松的，仿佛此处不是魔兽山脉，低声呢喃道：“速度真慢，也不知道什么时候才到目的地。”
<br/>　　“兄弟，你在说什么呢？”就在这时，慕清霄左侧的一名青年雇佣兵问道。
<br/>　　“也没什么，只是觉得后面那名女子很美罢了。”
<br/>　　回头看了一眼，处于重重护卫中的万药斋采药队，视线最终停留在那犹如众星捧月一般拥簇在中间的白裙女子身上。
<br/>　　此时，小医仙也正好微微直起身子，手背轻轻的擦拭着头额上的汗珠，轻轻气喘的模样，配合着那副柔美的脸颊，看上去使人心生怜爱。
<br/>　　见到小医仙露出这幅模样，周围一些佣兵顿时有种将之背到目的地的冲动，不过他们也知道，即使他们想背，小医仙也智慧含笑婉拒。
<br/>　　在佣兵们目光汇聚到小医仙身上时，一名模样颇为英俊的青年，却是满脸笑容的从一旁的护卫队中走出，低头对着气喘的小医仙说着什么。
<br/>　　两人交谈了一会，小医仙便是微笑着摇了摇头，眼底却是闪过一丝微不可查的厌恶，然后继续向前面赶路。
