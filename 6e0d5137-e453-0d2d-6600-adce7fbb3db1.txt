　　第314章
<br/>　　目前的山寨已步入正轨，一方面有商队每十天与戎族交易一次，用帝国的绢布、铁器、瓷器、草药、茶叶、稻米换来戎族的兽皮、骏马、美石美玉等等。
<br/>　　目前已经有成熟的渠道，运过来的货物很快就能换成银子和稻米。
<br/>　　一来一往，给蛤蟆寨带来了巨额收益，靠着商队带来的收益，支撑起了山寨的建设。
<br/>　　不仅蛤蟆寨，落玉寨、云中龙、乔鹤、游九楼四家也在其中吃得肚满肠肥。至少他们现在明白，跟在蛤蟆寨后面，还是有前途的。程大雷不像莫鸣米，他保持着利益均享的原则，收益人人有份。
<br/>　　除了经营商队，建设山寨，队伍的训练程大雷也没有放下，平时多流汗，战时少流血的道理，程大雷也明白。
<br/>　　秦蛮、关鱼、张肥与高飞豹走后，队伍的训练由赵子龙负责。如今，刘悲负责山寨建设，李行哉负责商队，赵子龙负责训练，程大雷万事脱身，做了甩手掌柜。
<br/>　　自从秦蛮几人去后，程大雷一颗心便揪了起来。西北距江南万里之遥，中间又有正义教起事，不知会不会发生什么意外。
<br/>　　程大雷让酒楼专门收集关于正义教的信息，每天递过来的情报如雪片般堆积在程大雷案前。程大雷想要通过这些情报，推测背后波澜起伏的大势。
<br/>　　这一日，程大雷又坐在书案后，手里拿着一页纸，这是昨日酒楼刚送来的。
<br/>　　在书案对面，摆着两张太师椅，上面坐着徐神机和李婉儿，这二人算是蛤蟆寨少有的认字又有脑子的。徐神机……也勉强算是吧。
<br/>　　书案上焚着香炉，袅袅檀香升起，银眸立在程大雷身后，像一座无言的冰雕。只有程大雷茶杯中的水尽时，才上前给里面添满水。
<br/>　　咕咚。
<br/>　　徐神机将程大雷的茶杯端起，一饮而尽，口中感慨道：“这还真是不查不知道，一查吓一跳，正义教原来已经这么大势力，都传到我们这边来了。”
<br/>　　程大雷无语的看他一眼，想说什么，最后又摇摇头，示意银眸将杯子换了。
<br/>　　“大当家，你看这个字念什么？”徐神机用吐沫星子舔着书页，将手中的纸页递到程大雷面前。
<br/>　　程大雷手扶着太阳穴，觉得脑袋又开始痛了。
<br/>　　“军师，再如何，你也不至于连字都认不全吧，一张纸有十个字你都不认识，平时你就不能多读读书？”
<br/>　　“嗨，我靠脑袋吃饭，字够用就行，不要在这无所谓的事情上浪费时间。”
<br/>　　程大雷无语，刚才觉得徐神机有脑子绝对是个错误，他摆摆手：“军师，您先去外面转转，透透气再回来。”
<br/>　　“怎么，不用我帮大当家出谋划策了……”
<br/>　　“不用，不用，您请，您请。”
<br/>　　徐神机也是如坐针毡，见程大雷松了口，麻溜快的撤了。
<br/>　　程大雷冲李婉儿尴尬的笑笑，李婉儿也笑笑回应。
<br/>　　“徐军师，真乃奇人也。”李婉儿由衷的赞叹。
<br/>　　程大雷再次无语，不知李婉儿为何会这么认为，难道和徐神机待久了，智商也会被拉下来。徐神机的隐藏属性还附带弱智光环。
<br/>　　徐神机的存在，暴露出蛤蟆寨一个问题，蛤蟆寨的弟兄普遍文化程度不高，当然真读过圣贤书的，很少有人愿意落草为寇。若有时间，给给他们补补课了，士兵的战斗素质要抓，指挥者的文化素养也不能放过么。
<br/>　　不得不说，徐神机走后，屋内的空气顿时清新起来。面前有曾经的帝国公主，身后有异域冰女郎，书案上清香袅袅，窗外时不时几声鸟鸣。
<br/>　　怪不得人说，男人是泥做的，女人是水做的，怪不得人说，红袖添香才好读书，这都是有生活的经验之谈呐。
<br/>　　隐隐地，程大雷觉得自己也像个大人物了。
<br/>　　没了徐神机捣乱，工作进度立刻变快了。程大雷这些日子以来，一直在摸正义教的情况，如今对正义教已经有了一定了解。
<br/>　　正义教的主要活跃区域在扬州，是最近半年才兴起的，短短时间内，聚集十几万信徒。他们的组织结构，大概是这样的：
<br/>　　正义教的最高首领是神主，关于他的情报少之又少，姓名不详，来历不详。但据说此人本事很大，能呼风唤雨，撒豆成兵。程大雷也明白，能在短时间拥有如此多的信徒，其人本事必然不小，所图必然也不小。
<br/>　　比较活跃的是神主下的神将，关于此人的情报很多，据说他是神主的徒弟，许多事都有他出面，关于其的姓名来历也查不到。
<br/>　　神将之下是各地信徒的领袖，称作信长。
<br/>　　现在正义教的势力已经发展到西北一地，就连浩甲城也有了正义教的信徒。
<br/>　　正义教打出的旗帜是：邪神降世，正义将大放光明。
<br/>　　程大雷啧啧称赞，口气还真的不小。
<br/>　　自然，正义教发展这么快，势头有这么大，朝廷不可能不管。如今朝廷已经开始通缉正义教信徒，攻打他们的老巢。
<br/>　　“这次攻打正义教的是李星率领的星字军，算是帝国的精锐军队。”李婉儿。
<br/>　　“李星？”程大雷问：“我在长安时怎么没听说过，很有名么？”
<br/>　　李婉儿无语，半晌才道：“算起来，是我的皇叔，封扬州王”
<br/>　　程大雷差点忘了，李本是国姓。这李星就是帝国明帝的兄弟。
<br/>　　江南一直没受戎族骚扰，算是富庶之地，只是不知道这扬州王的星字军能否打下正义教。
<br/>　　“大当家……”
<br/>　　徐神机急匆匆跑进来，跑得气喘吁吁，他一进来，程大雷就觉得屋内高端大气上档次的气氛瞬间被破坏掉。
<br/>　　“大当家，酒楼送来一份正义教的圣图，你要不要看看，我保管你吓一跳。”
<br/>　　所谓圣图，就是上面绘着正义教信奉的神明，教徒对它膜拜上香。
<br/>　　“别卖关子，我什么没见过，还能被吓着。”
<br/>　　“嘿嘿，那可说不好，你看看……”
<br/>　　徐神机将手中的图像展开，程大雷只看了一眼就愣住了。上面一只癞蛤蟆，头顶一个圆环状物事。
<br/>　　“这是……蛤蟆顶绣球么？”程大雷结结巴巴。
<br/>
