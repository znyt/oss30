　　“呵，好大的口气！”小道童听这话，顿时有点火大，指着他说道，“我跟你说啊，忍你很久了，你是哪里来的疯道士，敢来我们桃花山撒野，我们桃花山不是什么名山大川，不过在法术界也是有一定地位的，你敢让我们掌教来见你……”
<br/>　　李浩然听他啰嗦半天，没有任何反应，依旧平静如常。
<br/>　　两个道童中的师兄比较稳重，看李浩然这副样子，不像是一般人，于是拉了师弟一把，给他使了个眼色，对李浩然道：“阁下稍等。”
<br/>　　将师弟拉到后院去，去找他们的师父。
<br/>　　他们的师父是这里的监院，也是掌教几大弟子之一，位高权重，徒弟有很多，每天都带着一干弟子在后院修炼法术，这两个道童不过是他记名弟子中最普通的两个，平时都没什么资格见他。
<br/>　　桃花山虽然在景点中并不出名，不过在法术界内部，却是有一定的地位，弟子有几百人，在江南一带也算有影响，一些法术界的活动和会议，也是有资格参加的。
<br/>　　两个道童一路进了两重院子，通报过去，才把事情告知了自己的师父，秋亚子。
<br/>　　秋亚子正在指导自己内门弟子修炼体术，听说了汇报之后，根本没当回事，拍着大腿说道：“哪里来的疯子，也来找我！”
<br/>　　当下随便点了一个弟子，让他去前厅看看怎么回事，为了维护山门的名声，必要时候可以动手给他点教训。
<br/>　　等这个弟子离开之后，秋亚子继续教导弟子，几乎把这件事给忘了，过了好半天，那个弟子又回来了，慢吞吞地走过来。
<br/>　　秋亚子看了他一眼，说道：“怎么样，把那狂徒赶走了吗？”
<br/>　　“师父，那人狂妄至极，我忍不住跟他出手，结果他在我右臂上不知道下了什么禁制，我整条胳膊现在都动不了。”那弟子左手捉住右手，愁眉苦脸地说道。
<br/>　　“有这种事？”
<br/>　　秋亚子怔住，骂道：“让你们平时努力修炼，一个个偷懒！快来我看看！”
<br/>　　等那弟子走到面前，秋亚子抓住他的胳膊，很快找到了禁制所在，是在靠近肩膀上的一个很小的标志，看上去是一朵花，闪烁着晶莹的光泽。
<br/>　　这是什么封印？
<br/>　　秋亚子大拇指按在上面，释放罡气，结果……封印上一点反应也没有。
<br/>　　怎么会这样！
<br/>　　秋亚子怔住。但凡各种封印禁制，接受到外力的时候，就算是某些独门法术，别人解不开，至少也会有迹可循，而现在的情况是……自己的罡气释放进去，完全石沉大海。
<br/>　　只有一种可能……对方的实力太强，双方的差距太大，自己这点法力，连唤醒禁制的防御之力的程度都没有。
<br/>　　不可能会这样的，自己怎么说也是个准天师，就算是地仙牌位，法力也不可能比自己高出那么多吧？
<br/>　　“你们怎么交手的？”秋亚子问道。
<br/>　　“我……没有交手。”那弟子嗫嚅着。
<br/>　　秋亚子一怔，“什么，你刚不是说你出手了吗？”
<br/>　　“是，弟子是出手了……不过他并没有还手，只是抬起手，凌空一弹手指，我就感到肩上一麻，接着这条手臂就不能动了……”
<br/>　　“只是凌空一弹？”秋亚子倒吸一口气，一再问道，“你确定他没有用别的手段？”
<br/>　　那弟子一再保证，就是凌空弹了一下手指，没有多余的动作。
<br/>　　秋亚子怔了半天，招呼了自己几个内门弟子，一起往前殿赶去。他要去亲自会一会这个道士。
<br/>　　李浩然还在前殿，从香案上抄起了一本道经，在认真地读着，对于秋亚子等人的到来，似乎全无察觉。
<br/>　　秋亚子进去之后，先远远地打量了这个叫李浩然的人一会，得出的结论却跟那两个道童一样：普通，看上去人畜无害，不过也给人一种深不可测的感觉。
<br/>　　秋亚子舔了舔嘴唇，干咳两声，对着李浩然拱了拱手，说道：“阁下久等了。”
<br/>　　李浩然的目光慢慢从道经上移开，看了他一眼，道：“你是掌教？”
<br/>　　“贫道不是，贫道是这桃花山的监院，不知道阁下一定要见我们掌教，究竟为了何事？”
<br/>　　李浩然看也没看他，道：“让你们掌教来。”
<br/>　　秋亚子一听，顿时火冒三丈，身后几个内门弟子也是跃跃欲试，这家伙，简直太欺人太甚了！
<br/>　　秋亚子阴沉着脸，朝李浩然走过去，冷冷说道：“阁下难道是专门来找麻烦的？”
<br/>　　李浩然微微一笑，突然抬起一只手，朝他点了过来。
<br/>　　秋亚子见他出手，急忙栖身而上，左手不断变幻法诀，右手提着一支拂尘，对着李浩然扫了过去。
<br/>　　这一扫，是桃花山法术中不定符中最强的一手，秋亚子几乎用上了全力，意在一击制敌，他也知道，对方是个厉害的角色，自己手下弟子们都在场，万一自己吃个小亏，实在是太没面子了。
<br/>　　然而……尽管是有备而来，但秋亚子还是错了，他把李浩然想的太简单了——
<br/>　　面对他的拂尘，李浩然摊开一只手，五指并拢，紫色的光华从指缝间飞出，愈发凝聚，变得刺眼起来。
<br/>　　秋亚子眼前只剩下了一片紫光，别的什么都看不见了，拂尘也扫了个空。
<br/>　　紫光弥漫，将他包围在了中间，秋亚子立刻感受到了一股强烈的威压，浑身不能移动，然而，他内心刚感受到惊悚，紫光突然消失了，李浩然又出现在了面前，抬着右手，食指轻轻顶在了他两条眉毛中间的印堂穴上。
<br/>　　李浩然面无表情，双眼古井无波，没有任何情绪。
<br/>　　秋亚子一脸茫然，半晌才回过神来，明白发生了什么，或者说，根本就没有明白——他不知道李浩然是怎么消失的，又是怎么样出现的，自己全力一击，连他的边都没有沾到，就……被人用手顶住了死穴。
<br/>　　只要李浩然手指再往前一点，立刻就会戳进他的印堂穴中，让他元神俱灭。
<br/>　　两个人的实力差距……如此之大。
<br/>　　秋亚子浑身颤抖，但还是努力压制着情绪，牙关紧要，颤声问道：“阁下究竟要做什么！”
