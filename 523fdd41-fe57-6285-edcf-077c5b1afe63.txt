　　百道拳风，四面八方之间，向着楚风眠轰击而来。
<br/>　　就在五灵宗众人，感叹这拳风厉害玄妙之际，楚风眠突然动了。
<br/>　　在这五灵宗众人的众目睽睽之下，突然看到了不可思议的一幕。
<br/>　　只见楚风眠只是伸出手指来，轻轻一点，突然龙吟咆哮，响彻而起。
<br/>　　一尊龙影，从楚风眠的背后升腾而起，龙躯震动，百道拳风，尽数破碎。
<br/>　　风卷残云一般，这小耀王寒夜的一拳，都轻而易举的所化解，烟消云散。
<br/>　　而楚风眠就站在那里，甚至都没有大动干戈，仅仅一指，便是破解了这寒夜的拳风。
<br/>　　“这是什么手段？小耀王的拳风，居然被此人如此轻易的挡下？”
<br/>　　“一拳打出，千变万化，哪怕是九劫古帝应对起来，都困难无比，这是何等的实力？”
<br/>　　“此人才八劫古帝的境界，莫非是比起九劫古帝，还要恐怖？”
<br/>　　“怪不得流传此人的资质，比起云尊还要高上几分，如此看来，此人的实力，的确逆天。”
<br/>　　“怎么办？看来五灵宗跟圣龙宗已经是联合在一起了，此人势大，不好对付啊。”
<br/>　　圣院宗的诸位圣子，看着楚风眠，也是纷纷小声议论起来。
<br/>　　他们也都不傻，在知晓楚风眠的身份的一刻，这小耀王寒夜突然出手，便是想要一招来轰杀了楚风眠。
<br/>　　毕竟圣院宗一直是想要取代圣龙宗，成为圣龙秘境的四宗之首。
<br/>　　而楚风眠身为圣龙宗如今最为耀眼的弟子，能够当即轰杀，对圣院宗来说，绝对的好事。
<br/>　　所以这小耀王寒夜出手，看似是在阻止楚风眠杀王鸿光，实际则是直接出手想要轰杀楚风眠。
<br/>　　可现在楚风眠的实力，比起他们想象之中的，可是还要强横，比起小耀王寒夜，居然是丝毫不弱。
<br/>　　这种实力，一旦是让交手，只会两败俱伤。
<br/>　　“楚兄的实力，在下佩服，方才只是试探一二，想要跟楚兄切磋一下实力，还请楚兄不要见怪。”
<br/>　　小耀王寒夜被逼退出去，看着楚风眠，突然脸色一变，露出几分笑容道，一步退后，显然是不打算在出手了。
<br/>　　如此一来，没有人阻止楚风眠，那王鸿光的精血，便是被楚风眠捏在了手心之中。
<br/>　　在这一团精血之中，浮现出了王鸿光的身形，焦急的大喊道。
<br/>　　“小耀王师兄，快救我！”
<br/>　　“住口！王鸿光，你挑拨同门，差点让我们同门相残，你是何居心？”
<br/>　　寒夜听到王鸿光的话，当即咆哮道，猛然一拳再度轰击而出。
<br/>　　但是这一拳的目标，却不是楚风眠，而是那一团王鸿光的精血。
<br/>　　轰隆一声。
<br/>　　随着寒夜的一拳轰击而去，王鸿光的精血，全然湮灭在空中，再也没有活下去的可能。
<br/>　　这王鸿光的突然出手，也是让炎撼世等人，一时间没有反应过来。
<br/>　　刚刚这王鸿光，还对楚风眠等人嚣张无比，但是现在居然是被寒夜，亲手所杀。
<br/>　　只有楚风眠的表情，无比平静，从刚刚寒夜收手的时候，楚风眠就料到了这一幕。
<br/>　　所以楚风眠也没有阻止的打算。
<br/>　　“奸细以除，楚兄请上座。”
<br/>　　小耀王寒夜在轰杀了王鸿光之后，便是转过头去，对着楚风眠说道。
<br/>　　旋即几名圣院宗圣子，便是上前来，搬出了一座王座来，位于这寒夜的王座身旁。
<br/>　　楚风眠也不客气，上前一步，坐在了这王座之上，炎撼世等人，则是紧随其后，站在了楚风眠的身后，守护这楚风眠。
<br/>　　“刚刚的一切，都是误会，楚兄有着如此实力，统领五灵宗，我等心服口服。”
<br/>　　寒夜看到楚风眠坐在王座上，脸上露出几分笑意道。
<br/>　　丝毫不提刚刚冲突的事。
<br/>　　这寒夜不提，楚风眠没有主动提起，二人便是坐下聊了起来。
<br/>　　“对了楚兄可知道，这一次我请诸位前来的为何？”
<br/>　　寒夜随意聊了几句，便是步入正题道。
<br/>　　“要联合我们五灵宗一起，开启一座宝藏。”
<br/>　　楚风眠不动声色间的说道。
<br/>　　“不知是一座什么宝藏？”
<br/>　　“这一座宝藏，可是名声极大，在这藏地之中的宝藏中，都排的上号，楚兄可听说过九域净土宗？”
<br/>　　小耀王寒夜淡然的说道，语气之中却是流露出几分兴奋的语气。
<br/>　　“听闻过一二，据说这佛寺，就是九域净土宗的宗门所在。”
<br/>　　楚风眠随意的回答道。
<br/>　　楚风眠出身自九域的事，现在还没有暴露，楚风眠也不多说，说完便是静静的等着寒夜的话。
<br/>　　“不错，根据探察，这一尊佛寺，应该就是曾经净土宗的核心圣地，也是净土宗无数佛门强者的埋葬之地。”
<br/>　　小耀王寒夜开口道。
<br/>　　“九域，是曾经的四大圣域之一，虽然在如今的时代之中，已经没落，但是在古老的时代之中，九域并不缺乏强者。”
<br/>　　“这净土宗的祖师，也就是净土佛祖，便是一尊无上强者，而且天人境的佛尊，净土宗都不知道诞生过多少。”
<br/>　　“所以在这藏地之中，处处皆是宝藏，哪怕是我们得到一两件，都是极大的机缘。”
<br/>　　“根据我们圣院宗的圣子探察，突然是发现了一处宝藏，即将开启的消息，但是这一处宝藏，想要打开的难度极高，仅凭我们圣院宗，还难以开启，这才特意邀请楚兄等人前来。”
<br/>　　“是哪一位佛尊的藏地陵墓？”
<br/>　　楚风眠淡然的开口道。
<br/>　　楚风眠知晓，这佛门传统厚葬，这所谓的宝藏，便是曾经佛门强者的陵墓。
<br/>　　这些陵墓之中，蕴含着大量的陪葬宝物，几乎不亚于是一座座的宝藏。
<br/>　　“是金身佛尊的陵墓，金身佛尊，可是了不起的一尊人物，此人肉身极致，以强横肉身之力，踏入天人境的绝代强者。”
<br/>　　小耀王寒夜不动声色的说道，他的目光始终盯着楚风眠。
<br/>　　“不知楚兄可有兴趣。”
<br/>　　金身佛尊？
<br/>　　听到这个名字，楚风眠神色不由的动容了几分。
<br/>　　外人对这净土宗了解不多，但是楚风眠可是九域中人，在九域的无数古籍之中，也曾经记载过不少的净土宗佛门强者。
