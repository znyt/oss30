　　“此人到底是谁？”
<br/>　　“五十亿财富，只怕是一些大势力，都无法拿出来，让天商会的会长来，都不会如此奢侈。”
<br/>　　“本来以为这极道仙兵已经尘埃落定了，没想到此人居然又开口了。”
<br/>　　天字包厢之中，一道道灵识都在交谈着，他们已经退出了争斗。
<br/>　　如今这极道仙兵残片的争斗，便是十一号包厢中的楚风眠，跟一号包厢的乾元。
<br/>　　“乾元师兄，居然又有人敢来跟你竞争了。”
<br/>　　一尊圣王宗弟子，冷冷的开口道。
<br/>　　“此人的财富，似乎也不少，只怕也是从古洞府中，得到了不少的财富，才会如此挥霍。”

<br/>　　轻描淡写之中，又是一个数字喊了出来，只是乾元的心中，却是有着几分不爽。
<br/>　　这极道仙兵的残片，刚刚差一点便是就落入到了他的手中了。
<br/>　　没想到突然杀出来了一个楚风眠，彻底打乱了他的计划。
<br/>　　不过乾元却是有着自信，以他的财富，得到这极道仙兵的残片，应该不难。

<br/>　　楚风眠再度喊出一个数字，他的语气，跟着乾元一样。
<br/>　　七十亿的天价喊出，似乎都毫不在意。
<br/>　　楚风眠拥有血海，这血海中的财富，足足是可以以万亿来计算，别说是七十亿，七百亿，楚风眠都不会眨了眨眼。
<br/>　　“八十亿！”
<br/>　　乾元的眼神中，露出几分异样，再次开口。
<br/>　　而在他的话音未落，楚风眠的声音便是接连响起。
<br/>　　“九十亿！”
<br/>　　“可恶！”
<br/>　　乾元的脸上，尽是怒色，这完全打破了他的计划，没想到楚风眠，居然还敢跟他抗衡。
<br/>　　“一百亿！我出一百亿！”
<br/>　　乾元咬牙切齿的说道。
<br/>　　“小子，我看你如何跟我争！”
<br/>　　“一百亿？区区一百亿灵石罢了，这点财富，你就想要买下这极道仙兵的残片？”
<br/>　　楚风眠冷笑的声音，从包厢中响起。
<br/>　　“两百亿灵石！”
<br/>　　轰！
<br/>　　这一句话，简直是令在场无数人都是脑子一阵。
<br/>　　两百亿灵石。
<br/>　　这价格，居然是又再度翻了一倍。
<br/>　　这种加价方式，简直是不将灵石当做财富来看，如同楚风眠身上的财富，都跟无穷无尽一般。
<br/>　　“此人，到底是谁？”
<br/>　　“两百亿灵石，能够拿出两百亿灵石的人，虚空海中，都是寥寥无几。”
<br/>　　“这是那尊老古董？或者是得到了古洞府中大机缘的人？不管是哪一种，都不好对付。”
<br/>　　“不能查了，此人的来历，只怕是很不可测。”
<br/>　　天字包厢中，一道道声音响起。
<br/>　　若是说之前楚风眠表现出来的财富，他们还有些想法，想要趁机掠夺一番的话。
<br/>　　现在这财富，已经不是可以掠夺的，不管是老古董，还是从古洞府中得到大机缘的人，都是极为可怕的存在，不能招惹。
<br/>　　“你……！”
<br/>　　乾元听到楚风眠的数字，也是一下子愣住了，脸色通红，似乎是气的说不出话来。
<br/>　　如此强势，以财富将他无情碾压，这简直是令他愤怒之际。
<br/>　　“三百亿，我出三百亿！”
<br/>　　乾元疯狂的开口道，他声音中的愤怒，简直是席卷整个会场之中
<br/>　　这乾元的一身实力，根本不在生死境高阶之下，这股怒意，令包厢上的禁制，都开始颤抖起来，显然是气愤到了极点。

<br/>　　楚风眠平静的开口道，这话语，虽然平静，但是又是将价格，足足提升了一百亿的地步。
<br/>　　一百亿的灵石，就在楚风眠的口中随意说了出来。
<br/>　　好像这不是一百亿灵石，是一百块灵石一样。
<br/>　　而且就算是一百块灵石，都不足以让武者如此奢侈。
<br/>　　“小家伙，不用跟我争了，你争不过我的，你的所有财富，在我看来，都只是一些小意思罢了。”
<br/>　　楚风眠幽幽的声音响起。
<br/>　　让乾元听到，简直是要活生生的气死。
<br/>　　乾元可是得到了一座古洞府的财富，这财富简直是比起许多宗门的底蕴都要庞大。
<br/>　　没想到，居然会被人如此嘲笑。
<br/>　　“可恶！小子，这是你逼我的！五百亿！”
<br/>　　乾元疯狂的吼道。
<br/>　　他本来不打算将所有财富都压上的，但是这个时候，乾元却是疯狂之际。
<br/>　　如今他的一切想法，都是要压倒楚风眠，得到这极道仙兵的残片。
<br/>　　“六百亿！”
<br/>　　楚风眠轻描淡写之间，再次开口，毫不在意，每次就加上百亿灵石。
<br/>　　这语气中，甚至都带有着几分戏虐，好像是在跟乾元玩游戏一般。
<br/>　　所有局面，都尽是被楚风眠掌控。
<br/>　　“七百亿！”
<br/>　　乾元咬牙切齿的冷哼开口道。

<br/>　　楚风眠继平静开口。
