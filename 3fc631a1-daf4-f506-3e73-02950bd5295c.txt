　　“什么！”
<br/>　　神王路西法和一众神界强者，顿时全都一脸惊愕的循着那道淡漠声音，纷纷朝上方望去。
<br/>　　只见那九天之上，弑神大阵之外，不知道何时，出现了一道瘦削的少年身影，不是叶尘又是哪个？
<br/>　　“身外化身！你——你已经迈入化神境了？”
<br/>　　一直自信满满的路西法，此刻满脸震撼之色，眼眸深处透着深深的忌惮。
<br/>　　要知道，一般情况下，修真者只有达到化神境界之后，元神才能分裂，从而利用傀儡之术，制造出身外化身，实力倍增！
<br/>　　而当身外化身修炼到极高境界，就算比本尊的实力，也相差不了太多。
<br/>　　与此同时，
<br/>　　百里外的魔祖和无涯老人等一众强者，同样个个满脸骇然之色。
<br/>　　“不！这不可能！本祖修炼了近千年，也才只是半步通天境，他区区一个世俗少年，还不足二十岁，怎么可能达到那个境界？”
<br/>　　就连魔祖也忍不住大吼了一声。
<br/>　　一旁的无涯老人，沉默了片刻，忽然双眸一凝，一脸笃定道：
<br/>　　“不对！这小子刚刚施展的法门，乍一看像是身外化身，可是太弱了！而且从此子身上，丝毫察觉不到化神天君的气息，应该是施展了某种特殊的秘术！”
<br/>　　原来，
<br/>　　这几个月以来，叶尘的修为境界，虽然并没有太大进步，可他却也并没有闲着。
<br/>　　前世八百年，叶尘在修真界，登临苍穹之巅，成为一代狂帝，除了强大的修为境界之后，收集的各种神功秘法自然也不在少数。
<br/>　　而这其中，就有一门类似于身外化身的神奇道法。
<br/>　　叶尘如今才刚刚踏入金丹境，自然施展不成身外化身的神通，之前不过是利用秘术，模拟出了一个气息和他极度相似的镜像分身罢了。
<br/>　　方才他分身踏入大阵之中，成功的骗到了神界众高手，真身却一直隐藏在外面。
<br/>　　“你敢耍我？”
<br/>　　路西法到底也是半步化神境的高手，经过片刻的惊慌之后，很快也醒悟了过来，顿时气得肺都要炸了！
<br/>　　九天之上，叶尘冷冷一笑，
<br/>　　“怎么？就许你算计别人，不许别人算计你吗？”
<br/>　　说完这话的同时，叶尘手臂猛然抬起，
<br/>　　轰！
<br/>　　一道璀璨的紫色剑芒，突然冲天而起，瞬间到达了一千余米的高度，然后狠狠斩落下来，
<br/>　　“什么狗屁弑神大阵，给我破！”
<br/>　　轰隆！
<br/>　　剑芒狠狠斩落下来，直接劈落在下方的大阵之上。
<br/>　　这弑神大阵乃是困阵，若人在阵法之中，想要破阵自然千难万难，而叶尘此时从外部破阵，却就容易的多了。
<br/>　　轰隆隆！
<br/>　　咔！
<br/>　　神界众强者都还完全没有反应过来怎么回事，他们精心为叶尘设计的弑神大阵，已经被叶尘一剑劈开。
<br/>　　“啊！！”
<br/>　　神王路西法顿时忍不住发出一道愤怒的咆哮声，
<br/>　　“纵然没有了弑神大阵，本王也要让你死！所有人一起上！给我杀了他！”
<br/>　　嗖嗖嗖！
<br/>　　随着路西法一声令下，众强者纷纷冲天而起，瞬间便将叶尘包围其中。
<br/>　　“就凭你们这群废物也想杀我？”
<br/>　　叶尘被数十名元婴境高手团团包围，反而冷冷一笑，脸上挂着一抹戏谑。
<br/>　　没有了弑神大阵的掣肘，这些人在他们眼中根本不堪一击。
<br/>　　轰！
<br/>　　叶尘直接一记虚空斩，诡异莫测的剑芒，瞬间便斩中一名元婴中期的高手，后者连惨叫都没有来得及发出，就被叶尘一剑劈成了两半！
<br/>　　“啊！！”
<br/>　　其余神界强者看到叶尘如此威猛，顿时纷纷惊退，就连神王路西法也不由得瞳孔微微一缩，
<br/>　　“空间法则！你居然参悟了空间法则！这、这怎么可能！”
<br/>　　如果说，之前叶尘所施展的‘身外化身’，不过就是投机取巧的小伎俩，可是眼前他对空间法则的运用，却实实在在是化神境界之上才能参悟的大神通啊！
<br/>　　“没有什么是不可能！我的境界，尤其是你能揣测的？”
<br/>　　叶尘冷冷一笑，身体一晃，直接穿梭空间，瞬间杀到路西法的跟前，再出一剑斩出！
<br/>　　刺啦！
<br/>　　这一剑斩落下来，仿佛苍穹都要裂开两半，空中竟然出现一道长达数千米的空间裂痕！
<br/>　　“哼！”
<br/>　　路西法冷哼一声，慌忙举起手中的权杖迎了上去。
<br/>　　“光之守护！”
<br/>　　轰！
<br/>　　一道璀璨的白光陡然自那权杖之上发出，瞬间形成了一个巨大的防护罩，将路西法笼罩其中。
<br/>　　轰隆隆！
<br/>　　璀璨的剑芒斩落在那白色防护罩上，周围空间猛然一阵剧烈晃动。
<br/>　　蹬蹬蹬！
<br/>　　叶尘和路西法各自倒退了七八步，竟然旗鼓相当。
<br/>　　然而，还没等路西法松一口气，叶尘的身影已经在他视野中消失。
<br/>　　“嗯？”
<br/>　　下一刻，路西法猛然察觉到身后，传来一道危险的气息！
<br/>　　却是叶尘，已经动用空间法则，穿梭到了他的身后，随之又是一剑！
<br/>　　叶尘经过近半年的参悟和修炼，对空间法则的利用，已经达到炉火纯青的地步。
<br/>　　亏得这路西法，好歹也是半步化神境的高手，而且又擅长防御，总算勉强可以捕捉到叶尘的行动轨迹。
<br/>　　轰轰轰！
<br/>　　叶尘一连几剑斩出之后，均被路西法以乌龟式的防守抵挡了下来。
<br/>　　纵然，参悟了空间法则之后，叶尘的速度，可以将路西法吊打，可是两人的修为差距毕竟在那摆着，路西法一心防守，他在不动用所有底牌的情况下，想要杀掉此人，却也极为困难。
<br/>　　而其他神界强者，看到神王落入下风，顿时纷纷赶来救援。
<br/>　　“找死！”
<br/>　　叶尘暴喝一声，随手斩出几剑，
<br/>　　“啊！”
<br/>　　“啊！”
<br/>　　“啊！”
<br/>　　每一剑斩出，便会有一位神界强者陨落！
<br/>　　一转眼的功夫，数十名元婴级别的强者，已经被叶尘斩杀了十余人！
<br/>　　“不！”
<br/>　　路西法大吼了一声，
<br/>　　眼前的这些人，可都是他们神界中，精锐中的精锐，如今却被叶尘如同砍西瓜一般随意斩杀，这让他如何能忍？
<br/>　　“小子！这是你逼我的！神界所有人听着，光明祭祀！”
<br/>　　“是！”
<br/>　　随着路西法一声令下，神界众强者当即围成了一个巨大圆圈，同时全都低声吟唱起来。
<br/>　　而随着他们开始慢慢吟唱，每个人身上，都开始发出璀璨的白色光芒，恍若神灵降世。
<br/>　　继而，
<br/>　　那些白色光芒又全都朝中央的神王路西法汇聚而去……
