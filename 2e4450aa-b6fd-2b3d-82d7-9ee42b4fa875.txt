　　第82章
<br/>　　告示贴到山下后，周围山寨一直没有动静。程大雷也并不在意，他还有别的事情要忙碌。
<br/>　　在辣椒出售前，程大雷要把山寨所有辣椒都干炒一遍，这样是避免种子流出去。留出山寨自需的种子后，整个山寨都在忙碌这件事。一时间，蛤蟆寨笼罩在辣气中。
<br/>　　此外，就是安排刘关张三人的工作。关张二人的等级目前都起不到太大作用，随着大家一起炒辣椒就是。
<br/>　　可刘悲却是个人才。这时代的医疗水平不怎样，山寨一多半人都有些小伤小病，药是不缺的，只是没有知道如何对症下药的医生。如此，这刘悲来得正好。
<br/>　　程大雷打算给他收拾出一间房子做医馆，专门让他给山寨的人看病。目前暂时没腾出屋子，也就只能临时给他腾出块地方。
<br/>　　可没想到，刘悲刚当大夫的第一天，就闹出状况来。
<br/>　　他看病的时候，程大雷去看了一眼。只见刘悲搬个小马扎在大树下面一座，虽才17岁，愣生生坐出71的样子。
<br/>　　给人号脉的时候时而长吁短叹，时而愁眉苦脸。
<br/>　　“你这个病喔……唉……”
<br/>　　“刘大夫，我的病怎么了，很严重嘛？”
<br/>　　“严重不严重的……唉，生死有命，你也别太在意，不值得的。”
<br/>　　病人的心狠狠揪起来，冷汗唰唰往下落：“大夫，我究竟是什么病呢，您可要想想办法。”
<br/>　　“唉……”刘悲长长的叹口气，口中才慢吞吞吐出两个字：“风寒。”
<br/>　　病人心里咯噔一下：“很严重的风寒？”
<br/>　　“一般风寒。”
<br/>　　“……”病人。
<br/>　　程大雷在旁边看了一会，刘悲都是这么给人看病的。凡是被他看过的人，皆是愁眉苦脸，一副天不假年的架势。
<br/>　　程大雷算是看明白了，这刘悲就是个极端的悲观主义者。隔了数百年，祖先的悲伤在他身上复活，看什么都是一副红尘不可恋，人间不值得的表情。
<br/>　　以刘悲为中心，悲伤的气氛向外蔓延，凡是被刘悲看过病的，都是一副愁云惨淡的表情。程大雷站在太阳底下，忽然觉得今天阳光落在身上都凉了许多。
<br/>　　程大雷急忙打断刘悲的行医，再这样下去，乱我军心呐。
<br/>　　独自把刘悲叫到议事厅，对于这位英雄后人，程大雷还是不失尊敬的。
<br/>　　“我这不让你行医，你不会有什么情绪吧？”
<br/>　　“唉……”刘悲未语先叹气：“黄土埋枯骨，是人就免不了一死，看不看病都是一样。”
<br/>　　程大雷觉得刘悲说得不错，人活百年不过荒冢枯骨，如此说来，争名夺利又有什么……
<br/>　　咦！程大雷忙摇头，驱逐脑海中的念头，争名夺利还是蛮有意思的。看来刘悲这传染能力还真是强啊。以后自己要和他少接触，免得折损自己少年英雄的志气。
<br/>　　想到这里，程大雷心中忽然一动，道：“现在我有件事情很为难，你能不能帮我办了？”
<br/>　　“有事大当家吩咐就行，既然我兄弟在山寨入伙，这一条命就是大当家的。”
<br/>　　话出口就有些豪杰气，这刘悲果不愧是英雄后人，抑郁是有的，慷慨也是有的。
<br/>　　“好！”程大雷道：“你随我来，有个人我想拉他入伙，你帮我说服他。”
<br/>　　程大雷口中这人自然就是杜茂，那刘大耳可是三国第一HR，只希望这刘悲除了继承祖先的抑郁，他祖先忽悠人的能力，刘悲也能稍微继承点。
<br/>　　哪怕仅仅是一丁点，也就够用了。
<br/>　　地牢内，杜茂翘着二郎腿躺在稻草堆上，听得外面脚步声，他才睁开眼睛。
<br/>　　“怎么……又给爷爷送酒来了？”
<br/>　　扭过头，看到一面白无须的少年站在牢门外，未曾开言，先悠悠叹上一口气。
<br/>　　“……不值得的。”
<br/>　　杜茂一激灵，就感觉从尾椎骨冒上来一股软绵绵的劲儿。心中豪横去了大半，只觉俗世太烦，活着真累，过往逞强斗狠像一场笑话。
<br/>　　“我是谁，我在哪儿，为什么他只是看我一眼，我就这样啦……”
<br/>　　“来，我们谈谈吧。”刘悲盘腿坐下，这完全是持久战的架势。
<br/>　　“干嘛，谁要和你谈，滚，快点滚……”杜茂方寸大乱。
<br/>　　……
<br/>　　这天下午，一片马来到蛤蟆岭下。
<br/>　　小白狼翻身从马上下来，见蛤蟆寨立着的那块牌匾下，聚着几个人。
<br/>　　在整个青牛山，程大雷是唯一重视信息的，至于其他山寨从前都活在闭塞的自我世界中。如今，程大雷在山下立了快牌子，有时候贴些告示，俨然就成了青牛山众山寨的新闻发布中心。
<br/>　　“瞧瞧，一斤辣椒换一斤粮食，兄弟，你们山寨种了多少辣椒？”
<br/>　　“一棵也没有。”
<br/>　　“嘿，那你完了，这一亩辣椒的产量要高出粮食十几倍呢。”
<br/>　　“那……你说这会是真的么？”
<br/>　　“真……嘿，要是真的就好了，可世上哪有这种好事。”
<br/>　　几人这么说着，其实心里也有些打鼓，万一是真的呢，当初程大雷让大家种辣椒的时候，大家可是一棵都没种，还把这当笑话讲。
<br/>　　他们这么说，却也是安慰自己，有祈祷的意思在。
<br/>　　小白狼自然听到了他们的对话，随之便是『呀，你们杏花岭要发财了』『是呐，这个冬天不用愁』之类的嘲讽的话。
<br/>　　“都特么给老娘闭嘴，再啰嗦小心老娘隔了你们。”
<br/>　　小白狼回身亮出了刀，然后牵着马上山。
<br/>　　其实，这样这些话，她这段时间已经听过无数次。当初程大雷让大家种辣椒时，也就她觉得，程大雷或许是可信的，试着种了一亩。结果没少遭到山寨里姐妹的鄙视，搞得她一颗心七上八下的。
<br/>　　说白瞎了一亩好地，否则能打好几百斤粮食之类的话。
<br/>　　好不容易等辣椒收获，程大雷又贴出告示，小白狼觉得自己终于可以吐气扬眉了。结果听到的，还是这样嘲讽的话。
<br/>　　不是这件事不可能是真的，而是这件事太好，好得让人不敢相信这是真的。
<br/>　　小白狼这次上山，就是要当面问问程大雷，求个真假。实在不行，能补我一亩地几百斤的粮食就行。
