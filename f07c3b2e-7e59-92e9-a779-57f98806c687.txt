被“大嫂”的夏夕绾惊呆了，手里的抹布被抢了，她还被人安置在了椅子上，她只能仰着巴掌大的绝色小脸望着陆子羡，小声道，“这是…怎么回事啊？”
<br/>
<br/>        陆子羡脱下了外面的西装，里面是一件白色衬衫，清冷风华的他穿白衬衫格外的好看，他看着夏夕绾，“你不会怪我先斩后奏吧？”
<br/>
<br/>        “什么？”
<br/>
<br/>        陆子羡凑近了她，清冷的唇角溢着柔软的笑意，“我已经公开了我们的关系，绾绾，我已经迫不及待的将你介绍给了我身边所有的人。”
<br/>
<br/>        夏夕绾一顿，一下子就撞进了陆子羡深情的眼眸里，“我…”
<br/>
<br/>        “什么都不要说。”陆子羡打断了她的话，“现在我来问，你来答，你是X，对吗？”
<br/>
<br/>        夏夕绾看着陆子羡，没说话。
<br/>
<br/>        陆子羡已经得到自己想要的答案了，她就是X，X就是她！
<br/>
<br/>        五年前他远赴国外进行巅峰学术论坛会，因为所有消息的封锁，他是等病毒得到控制了才得以回国的。
<br/>
<br/>        那时她已经分离了病毒，找到了病毒源，而且，消失在了风口浪尖里。
<br/>
<br/>        他回国就看了她的病毒肢解，深深的惊艳。
<br/>
<br/>        原来她就是X！
<br/>
<br/>        陆子羡勾唇，干净的黑眸里划出几分黯然，“绾绾，原本我可以先认识你的，但是我们一次又一次的错过了。”
<br/>
<br/>        夏夕绾不知道该说些什么，缘分强求不得，有时候有些东西就是命中注定的，她刚出生那会儿，第一个就见到了…陆寒霆。
<br/>
<br/>        “子羡，我有一件事想问你，你知道这个科学院里藏了什么宝箱吗，或者有什么神秘的地方可以藏宝箱之类的东西？”
<br/>
<br/>        “宝箱？”陆子羡看着她，“这就是你一定要进入科学院，闯入这个龙潭虎穴的原因？”
<br/>
<br/>        这里对于夏夕绾而言充满了危险，她太大胆了，孤身一人就闯入了夜家的地盘。
<br/>
<br/>        夏夕绾点头，“是的，我在找一个宝箱。”
<br/>
<br/>        陆子羡想了一下，“我从未听说这个科学院里有什么宝箱，如果说这里有什么神秘的地方，的确有一个。”
<br/>
<br/>        夏夕绾双眼一亮，“哪里？”
<br/>
<br/>        陆子羡牵住了夏夕绾的小手，将她带到了窗口，他指着电台的后院，“看到那个暗红色的大门了吗？”
<br/>
<br/>        夏夕绾看到了，暗红色的大门，很像是那种几十年代深宅大院那种大门，它静伫在那里，透出一股神秘又诡异的气息。
<br/>
<br/>        里面好像有什么东西在吸引着你，又让你畏惧。
<br/>
<br/>        “那个大门一直锁着，从未打开，没有人进去过，那是科学院的禁地，里面有狼守护着。”
<br/>
<br/>        夏夕绾的心突然砰砰砰的乱跳，她感觉到了，她感觉到了妈咪的气息。
<br/>
<br/>        宝箱一定在那扇门后面。
<br/>
<br/>        “子羡，你知道大门的钥匙在哪里吗？”
<br/>
<br/>        暗红色的大门上了锁，必须要钥匙打开，她要进去！
<br/>
<br/>        “绾绾，你不能进去，都说那片禁地被施了诅咒，进去的人都不会再出来了，虽然我不相信这种鬼神论，但是你进去只会喂了里面的狼。”
<br/>
<br/>        夏夕绾勾起了红唇，“子羡，我并不想死，但是，我也一定要找到宝箱。”
<br/>
<br/>        陆子羡看着她坚定的眼眸，半响道，“钥匙在夜滢那里。”
