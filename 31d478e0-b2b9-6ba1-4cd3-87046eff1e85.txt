　　东海龙宫。
<br/>　　龙族高高在上，坐于九龙宝座之上俯视着众生，缓缓道：“现在大军准备的如何？”
<br/>　　在下方左侧第一个座位上，龟族族长，也就是现在的龙宫丞相龟丞相，他站起身来，对着龙皇行了一礼，沉声道：
<br/>　　“启禀龙皇，龙族大军已经集结完毕，恭请龙皇试阅，族中战士也已经准备好了，就等着龙皇一声令下，杀向麒麟一族！”
<br/>　　“好！很好，非常好！那就去准备吧，我们龙族才是这个世界的霸主，谁能争锋？而且麒麟一族竟敢杀我龙族子孙，是时候拿他们开刀了。”
<br/>　　龙祖带着无边霸气，站了起来，气势冲天而起，东海也被搅得波浪翻滚，云海溃散。
<br/>　　“是，龙皇大人，此战必胜！”
<br/>　　众人齐齐一贺，心中也涌起无限斗志，实在是由不得众人不自信啊。现如今整个龙族的底蕴已经不是千万年前可比的，龙族实力大大提升，无论是个体还是群体，大罗金仙有数十个，太乙金仙上百，金仙强者更是数千，玄仙那就更多了，数万玄仙还是有的。
<br/>　　龙族的底蕴，要超过凤凰族或者是麒麟族中的任何一个种族，他们有底气这样说。有着这样的实力，要不是龙皇大人一直在默默发展壮大，不让打架，他们早就开始大战了。
<br/>　　而这一刻，龙皇终于下令开启大战，每一个人心中都涌起无尽欢喜，他们龙族争霸的日子终于要到了。
<br/>　　要说为什么龙祖要如此着急的去对付麒麟呢，这还与元始有关。
<br/>　　元始用度心诀度化了龙族四太子敖玉，叫他收集百滴龙族精血，及三滴龙祖精血，敖玉设计杀害了他的兄弟子孙们，提取他们的精血，并将这一切推到麒麟凤凰两大种族上。
<br/>　　之后再加上西方罗睺的推波助澜，使之三大种族中不断发生摩擦事件，就这样，战争愈演愈烈，三大种族从先前的子孙被害一事上升到了种族争霸大战，三族谁也不服谁，最后不得不卷入这样的战争之中。
<br/>　　洪荒南部不死火山。
<br/>　　“终于要开始了，时机就快要到了，我风祖必定能凤舞九天！”
<br/>　　洪荒不周山的麒麟崖。
<br/>　　“终于来了，哼！别以为只有你龙族强大，我麒麟一族同样想做这天地之主！”
<br/>　　……
<br/>　　这一刻，天地之中天机渐渐变得模糊不清，天地之间的煞气也越来越浓了，洪荒一时间变得乌烟瘴气。
<br/>　　西方灵山魔窟之中，罗睺感受到天地的变化，顿时大笑道：
<br/>　　“好！好！好！终于开始大战了，让天地煞气来得更猛烈一些吧，劫气、煞气、业力、瘴气，我全要，哈哈哈！用不了多久，我就要突破成圣，我要成为天地之间第一圣！哈哈哈……”
<br/>　　……
<br/>　　西昆仑玉京山中，鸿钧老祖依旧还是原来那副模样，仿佛数万年都未曾动过分毫，看着天地之间浓郁的煞气，不由叹道：
<br/>　　“可怜啊可怜，三族已身在劫中，却仍然不知，看来是气运要衰竭了，我也该开始准备了。”
<br/>　　这一次大劫，既是龙凤麒麟三族的龙汉大劫，同样是他与罗睺的道魔大劫，在此场大劫的最后，鸿钧与罗睺之间，必有一战，而这一战的结果将决定谁成就混元道果，证道成圣！
<br/>　　……
<br/>　　“嗯？三族大战的天地大劫就快要开始了吗？”在距离昆仑山不远处，元始感受到天地之间日愈增加的劫煞之气，就知道，天地大劫快要开始了。
<br/>　　要说这天地大劫却是不可避免的，从开天之劫到凶兽之劫，再到如今的龙汉大劫以及道魔劫，甚至后世的巫妖劫、封神劫等，这些大劫都无法避免，只能延迟。
<br/>　　天地有阴阳，灵气能量同样分阴阳，若说众生平时吸收的是天地之间的正能量，也就是阳性能量，那么这天地之间的劫气、煞气就是负能量，也称阴性能量。
<br/>　　负能量没人管，这样长时间一积累，没有人处理，就会形成天地大劫，用大劫的方式来宣泄天地之间的负能量。
<br/>　　减缓大劫来临的时间，可以用杀戮的方式解决，天地之间修士减少了，宇宙所承受的负担也就少了，大劫的时间自然就会减少了。只不过，随着时间的流逝，天地之间的修士只会越来越多，又哪会减少？
<br/>　　所以，杀戮是解决大劫的最好方法。
<br/>　　“嗯？想多了，现在最要紧的是赶紧回到昆仑。”元始哑然一笑，想这么多干什么，这又不干自己什么事儿，他们闹腾他们的，只要不惹到自己就是了，自己何必为别人担忧。
<br/>　　不再多想，元始快速向着昆仑山谷飞去。
<br/>　　……
<br/>　　东昆仑山谷之中，三才大阵的守护早已消失不见，山谷之内一如既往地灵气纵横，山水俊秀。
<br/>　　“二兄，你说大兄何时能够回来？都出去如此之久了，怎么还不回来，不会是发生什么意外了吧？”
<br/>　　洞府之中，一道身穿碧青色道袍，头束长发的青年走来走去，对着一身穿白色道袍，满头白发的老道说道。
<br/>　　老道眯着双眼，静静地坐在蒲团之上，无悲无喜，无欲无求，清静无为，不为所动。
<br/>　　青年见老者不为所动，依旧叽叽喳喳地说着话，不见丝毫停下来的迹象。
<br/>　　后来，也许是老者见青年叽叽喳喳，不耐烦了，就随口道：
<br/>　　“大兄修为高深，能出什么意外？不知为何，天地之间的煞气多了起来，你还是静心下来，好好修炼为好。”
<br/>　　“真的吗？也是，大兄修为高深莫测，应该不会出事。”青年男子也喃喃道。
<br/>　　就在此时，洞府之外，一道朗笑声传入洞中——
<br/>　　“哈哈！还是二弟三弟了解我啊，以本座的修为，又怎么可能出事？”
<br/>　　紧接着，一道身穿白玉道袍，脚踏七彩祥云的中年男子一步踏入洞府之中。
<br/>　　两人一见到中年男子，顿时就高兴起来。青年男子激动道：
<br/>　　“大兄，你可算回来了，都出去如此长时间，我还以为你出事了。”
<br/>　　老者同样有些激动，只是他不像青年一样不懂控制自己情绪，他尽管面无表情，但眼中那一闪而逝的激动没能瞒住元始。
