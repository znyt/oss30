　　五千里之外。
<br/>　　萧逸的身影，疾速飞行。
<br/>　　“噗。”
<br/>　　鲜血从他嘴中喷出。
<br/>　　他却无暇理会。
<br/>　　此刻的他，一手握剑，破空飞行。
<br/>　　一只手，却紧紧地扣住了自己的乾坤戒。
<br/>　　乾坤戒内，远远不断地精纯力量，不断涌入他的掌心。
<br/>　　那是灵脉的精纯力量。
<br/>　　他的气泉之内，元力虽未耗尽，但已几乎见底，根本无法再支撑他的元力燃烧，以及暴增的速度。
<br/>　　此刻还在维持着的，是远远不断涌入的灵脉力量。
<br/>　　灵脉，不愧是蕴含着天地间最为精纯的力量。
<br/>　　燃烧起来，其爆发程度竟还比他体内元力还强猛一些。
<br/>　　再者，这是中品灵脉，威力更甚。
<br/>　　这也是他从坑洞逃离后，速度暴增，比之前还甚几分的原因。
<br/>　　不过同时，乾坤戒内，灵脉的消耗速度，同样异常惊人。
<br/>　　整整一条灵脉，在萧逸的感知中，以肉眼可见的速度不断削减着。
<br/>　　又是十数分钟轰。
<br/>　　萧逸瞳孔一缩。
<br/>　　因为，前方，北隐宗大长老的武道虚影，再次凝聚。
<br/>　　“怎么可能。”萧逸煞白的面庞上，脸色难看到极点。
<br/>　　若无这武道虚影，他早便远远逃遁了。
<br/>　　若再这般一直下去，他能撑几回？
<br/>　　武道大能的追杀，难道他就真地逃不掉？
<br/>　　这般想着，萧逸还是一剑劈出。
<br/>　　无论如何，不能被这武道虚影拦太长的时间。
<br/>　　剑出，星河激荡，硬撼武道长河。
<br/>　　当然，接过便是星河溃散，武道长河束缚而来。
<br/>　　萧逸再次一剑破之。
<br/>　　这一次，北隐宗大长老追来的身影倒是堪堪慢了些许。
<br/>　　不过，那道凌厉掌风，还是如期而至。
<br/>　　嘭…
<br/>　　萧逸直接被轰得吐血，却也再次借力远遁。
<br/>　　嗖…
<br/>　　北隐宗大长老看着萧逸再次消失的身影，脸色霎时发黑。
<br/>　　“难怪我北隐宗屡次对上你都折戟沉沙。”
<br/>　　“果真是个难缠的小贼。”
<br/>　　北隐宗大长老边疾速追击，边咬牙自语。
<br/>　　他现在已经放弃了等待萧逸元力耗尽的想法。
<br/>　　原本在他的判断中，萧逸早该元力耗尽。
<br/>　　此时，萧逸不仅元力没有耗尽，反倒速度更甚之前。
<br/>　　“虽然你很能逃，但你躲不过我的武道力量追踪。”
<br/>　　“老夫倒要看你还能受我多少掌。”
<br/>　　每一次的凌空掌风，打在萧逸身上，他都看在眼里。
<br/>　　每一掌，都能重伤萧逸。
<br/>　　若他能完全追上萧逸的话，要杀萧逸，不过是片刻之间。
<br/>　　现在既然追不上，那便不断重伤之。
<br/>　　萧逸再能逃，但若重伤而死，又有什么意义呢？
<br/>　　……
<br/>　　数千里之外。
<br/>　　疾速飞行的萧逸，脸色已然煞白到极点。
<br/>　　体内伤势，早已极其严重。
<br/>　　虽未致命，却也让他极其难受。
<br/>　　北隐宗大长老的每一掌，几乎都能轰得他五脏六腑几乎移位，体内气血翻滚不停。
<br/>　　若非有云生铠甲护身，卸去了大半掌力，恐怕他现在已经重伤而死了。
<br/>　　武道大能的实力，着实恐怖。
<br/>　　这是真正站于大陆顶尖，能够横行中域的强者。
<br/>　　这等强者，哪个不是一方巨擘，一方大势力的超级强者。
<br/>　　现今，这等强者竟不厌其烦地追杀他。
<br/>　　也不知道该说是萧逸荣幸，还是说萧逸运气太背。
<br/>　　不过，无论如何，萧逸还是快速思索着办法。
<br/>　　他知道，距离下一次武道虚影凝聚，还有十数分钟。
<br/>　　这十数分钟，是他唯一能思索对策的时间。
<br/>　　萧逸感知了一番乾坤戒内的灵脉。
<br/>　　此刻，一条灵脉已消耗过半。
<br/>　　萧逸咬了咬牙。
<br/>　　原本，在他的打算中，乃是不断远遁逃离。
<br/>　　若能拉开与北隐宗大长老的距离，他还有一线生机。
<br/>　　再不济，便让北隐宗大长老一路追杀便是。
<br/>　　他只要能一直逃遁，便有机会。
<br/>　　大不了一直逃遁至风刹殿总殿。
<br/>　　他就不信这北隐宗大长老敢去风刹殿总殿杀他。
<br/>　　可现在，这些想法都被萧逸放弃了。
<br/>　　先不说北隐宗大长老每次的全力一掌都能让他重伤，他根本撑不了几次。
<br/>　　再者，以这灵脉的消耗速度，怕撑不了几天。
<br/>　　一旦所有灵脉耗尽，他元力耗尽。
<br/>　　那时，便是逃无可逃，再无生机。
<br/>　　“若逃不了，便只能一战。”萧逸心头暗暗想着，脸色一狠。
<br/>　　不过，这丝狠色，仅仅是出现了数秒，便已散去。
<br/>　　萧逸遇到过许多危机，可每一次，无论危险程度多高，他都有一定的把握。
<br/>　　唯独这一次，却几无办法。
<br/>　　他很清楚，即便是手段尽出，乃至将各种强悍火焰祭出，都绝对不是北隐宗大长老的对手。
<br/>　　武道大能，太强了。
<br/>　　“或许…”萧逸皱眉思索着。
<br/>　　“破晓钟的手段。”
<br/>　　萧逸忽然眼前一亮。
<br/>　　从破晓钟上领悟的手段，他大概只掌握到一剑出，15剑齐至的程度。
<br/>　　若是这个程度再高些…
<br/>　　他记得，他虽未能伤北隐宗大长老，却也能一剑轰退。
<br/>　　若是这个程度，达到20剑以上，未必就没有一战之力。
<br/>　　萧逸嘴角咧过一道笑意。
<br/>　　一边疾速飞跃，一边暗暗参悟着。
<br/>　　临阵参悟，以及逃遁中参悟，萧逸不是没有试过。
<br/>　　以他天赋，这一点并不太难。
<br/>　　……
<br/>　　十数分钟后。
<br/>　　前方，北隐宗大长老的武道虚影如期而至。
<br/>　　当然，最后结果，同样是萧逸负伤逃遁。
<br/>　　……
<br/>　　又是十数分钟后。
<br/>　　结果同样如此。
<br/>　　……
<br/>　　时间渐渐过去，一个时辰后。
<br/>　　萧逸再受北隐宗大长老一掌，远遁离去。
<br/>　　北隐宗大长老，却冷笑一声。
<br/>　　“气息愈发虚弱了。”
<br/>　　“顶多再受我几掌，你便再无战力，黯然等死。”
<br/>　　然而，两个时辰后。
<br/>　　北隐宗大长老的脸色，忽然变得不可置信起来。
<br/>　　“整整两个时辰了，硬受我不下十掌，还能远遁？”
<br/>　　看着萧逸再次消失的身影，北隐宗的目光，仿佛在看着一股怪物般。
<br/>　　他显然远远低估了萧逸承受伤势，以及逃遁的能力。
<br/>　　这个妖孽，绝不是那等从庇护中慢慢成长的武道天骄。
<br/>　　而是一个历尽艰辛，历尽无数九死一生，才成长至今的妖孽。
<br/>　　……
<br/>　　直至半天后。
<br/>　　疾速飞行的萧逸，前方，再次出现了北隐宗大长老的武道虚影。
<br/>　　可这一次，萧逸却笑了。
<br/>　　武道虚影上，一条武道长河封锁而来。
<br/>　　萧逸一剑劈出。
<br/>　　八万星光降临，幻化星河，碾压而过。
<br/>　　嘭…
<br/>　　一声爆响，星光长河狂猛如斯；武道长河，顷刻溃散。
<br/>　　一剑落下，武道虚影分奔离析。
<br/>　　......
