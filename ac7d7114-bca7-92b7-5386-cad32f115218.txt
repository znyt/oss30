　　“这小子，好变态的实力。”老者显然吃了一惊。
<br/>　　那阴冷的眼眸中，杀意愈发浓郁。
<br/>　　“老夫若没猜错的话，你的修为并不高吧。”
<br/>　　“不过是借着世间强悍火焰和手段，方有与老夫一战之力。”
<br/>　　“是又如何。”萧逸冷笑一声。
<br/>　　“炎龙大陆，乃是实力为尊，而非修为为尊。”
<br/>　　“多说无益，战吧。”
<br/>　　萧逸没有解释，以老者的眼力，要看出这些并不难。
<br/>　　轰…
<br/>　　萧逸身影一闪，却宛若一道迸发的陨石，速度快到了极点之下，产生了震耳的破空声。
<br/>　　老者眯了眯眼，不战，反退。
<br/>　　“嗯？”萧逸眉头皱了皱。
<br/>　　恰在此时，下方，一道惊人的炙热气息冲天而起。
<br/>　　萧逸循着气息看去，火鸦大阵边缘，一只巨大的火鸦凭空凝聚。
<br/>　　萧逸皱着眉，若有所思。
<br/>　　这些天来，萧逸越是和邪修接触得多，了解得深，便越是对邪修这一门职业感到惊讶。
<br/>　　对于邪君府的底蕴，萧逸愈加高看。
<br/>　　如果说以前仅仅是高看的话，那么，现在完全便是对这个庞然大物感到深不可测。
<br/>　　上次在三风地域，他已经见识过黑魔炼妖大阵。
<br/>　　这一次，直接便是连荒古苍炎、火鸦大阵都见识了。
<br/>　　火鸦大阵，可是上古之时炎殿有名的大阵之一。
<br/>　　除却炎殿总殿，外人根本不知晓布阵之法。
<br/>　　黑魔炼妖大阵，虽不如火鸦大阵，却也相距不远。
<br/>　　邪君府，到底是怎样将这些稀有而威力莫测的阵法学到手的？
<br/>　　若说炎殿与邪君府勾结，那是绝对不可能。
<br/>　　上古八殿，乃是大陆的八座基石，殿内先辈，曾以血肉之躯守护着人类大陆的最后一分希望，乃至最后平定妖兽之祸。
<br/>　　千万年来，八殿之内一直有极其严格的殿规。
<br/>　　而八殿武者，亦一直恪守殿规，履行本职。
<br/>　　八殿殿主，则个个皆是大陆传奇，受人景仰。
<br/>　　至于邪君府这等势力，则是人人喊打。
<br/>　　邪修，更直接是八殿的通缉对象。
<br/>　　若说炎殿与邪君府有干系，那是决计不可能，可邪君府，却还是能拥有不少的八殿手段。
<br/>　　这个势力，怕是传承年月，绝不亚于上古八殿的任何一殿。
<br/>　　这时，未等萧逸多想，那头巨大的火鸦，已飞跃而出。
<br/>　　下方，一众武者，早已脸色大变。
<br/>　　那头巨大火鸦的气息，远比之前山腹之内的那头，更加惊人，亦更加恐怖。
<br/>　　哪怕远远视之，亦一阵心悸。
<br/>　　单单是其上气息，竟以让得过万武者浑身炙热，宛若身体融化。
<br/>　　各大势力的长老，直接惊呼一声，“是炼狱火鸦。”
<br/>　　“该死，难怪刚才火牙宗主等人一直没有出手，竟是要借大阵之力凝聚出这等凶兽虚影。”
<br/>　　“传闻炼狱火鸦，乃是上古凶兽，所过之地，炽焰千里，一片炼狱焦土。”
<br/>　　“这头虚影，虽不如当年炎殿先辈所施火鸦大阵而凝聚出的火鸦那般强，但也强横万分。”
<br/>　　“紫炎，小心。”一个长老高喝一声。
<br/>　　萧逸眼眸淡漠，直视着老者。
<br/>　　这头巨大火鸦，乃是火牙宗主与十数邪修长老借火鸦大阵凝聚而出。
<br/>　　火鸦飞出，并未攻向萧逸，而是飞跃至老者脚下。
<br/>　　观这头火鸦的气息，竟与老者一致。
<br/>　　老者踏于火鸦之上，身上气息，陡然暴涨。
<br/>　　“易霄，你不过是借着各种手段，才有与老夫一战之力。”
<br/>　　“如今，这火鸦的实力不亚于老夫。”
<br/>　　“二打一，你还有胜算？”
<br/>　　话音落下，老者眼眸一冷。
<br/>　　脚下火鸦，霎时飞跃。
<br/>　　那恐怖速度下所产生的火焰气爆声，比萧逸刚才还要剧烈数分。
<br/>　　几乎是一个瞬间，老者便脚踏火鸦来到萧逸面前。
<br/>　　轰…
<br/>　　巨热的气浪，迎面灼来。
<br/>　　萧逸身上，霎时火焰加身，被气浪掀飞百米。
<br/>　　“就这点本事？”老者冷笑一声。
<br/>　　“紫炎易霄，今日便在我火牙宗陨落吧。”
<br/>　　老者脚踏火鸦，再次瞬间而来。
<br/>　　“炼狱苍炎。”老者一掌打出，身后再次虚影凝聚。
<br/>　　虚影，不再是巨大白骨，而是火鸦虚影。
<br/>　　火鸦虚影，大口一张，一股炙热火焰喷发而出。
<br/>　　老者一掌打出，掌中黑气幻化龙卷。
<br/>　　黑气环绕翻腾，与喷发的炙热火焰融合而至。
<br/>　　“怒炎指。”萧逸怡然不惧，一指点出。
<br/>　　“又是同样的招数？笑话罢了。”老者冷笑一声。
<br/>　　轰…
<br/>　　指与掌，瞬间接触。
<br/>　　萧逸指尖火光，顷刻挡下了喷发而至的火焰与黑气龙卷。
<br/>　　“你败了，便也可以死了。”老者眼眸一冷。
<br/>　　“准确来说，是你没有机会了。”萧逸眼眸中，蓦地闪过一丝戏谑。
<br/>　　“嗯？”老者一惊，心头闪过一丝不妙。
<br/>　　萧逸手中手套，忽然光芒大作。
<br/>　　滔天气息，霎时凝聚指尖。
<br/>　　指尖火光，在这一瞬间暴涨。
<br/>　　看似点点火光，却以肉眼可见的速度不断压过了喷发而来的火焰与黑气。
<br/>　　“怎么可能。”老者大惊失色，“区区中品巅峰圣器，怎可能发挥这般威力…”
<br/>　　“谁告诉你这是中品巅峰圣器的？”萧逸冷笑一声。
<br/>　　“火牙宗主那个傻瓜吗？”
<br/>　　萧逸话音落下，指尖一突。
<br/>　　指尖，顷刻破去了老者手掌黑气，随后势如破竹，连带着火鸦虚影亦一并消散。
<br/>　　萧逸的指尖，最后轻轻抵在了老者心房之上。
<br/>　　哒…轻轻的点声。
<br/>　　嘭…嘭…嘭…嘭…
<br/>　　连绵不断的爆响声，在老者身上爆发。
<br/>　　那是数以千计的火焰在他体内暴走。
<br/>　　这一指，代表着萧逸控火一道的最强手段；在五色火焰融合，以及星幻手套的加持下，此刻达至最强威力。
<br/>　　待得爆响声消散，老者的身影，已然僵立于高空。
<br/>　　“怎么可能…”
<br/>　　老者脸上，尽是不可置信之色，嘴中艰难吐出寥寥几字。
<br/>　　下一秒，身躯就此从高空无力跌落。
<br/>　　老者身上气息，已然全无。
<br/>　　“太上长老…”远处，火牙宗主等人惊呼一声，脸色大变。
<br/>　　“该你们了。”萧逸看向火牙宗主。
<br/>　　“不好…”火牙宗主只来得及惊呼一声，下一秒，只觉身体一疼，眼前，已出现了一双冷漠的眸子。
<br/>　　高空中，萧逸的身影，忽然而消。
<br/>　　空气中，只余一道火焰残痕。
<br/>　　他的拳头，已洞穿了火牙宗主的胸膛。
