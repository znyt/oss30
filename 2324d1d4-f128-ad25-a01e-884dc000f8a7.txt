
<br/>　　<br/>　　    林文赫和王令歌到了没多久，午桥和菜刀文也各自被警车送了过来，四个人一到位，这算是王惊蛰把团队给支了起来。<br/>　　<br/>　　    当王惊蛰领着他们几个去到指挥部的时候，人往那一站，直接就让黄詹，张生利品出这都不是一般人了，就拿气质这一块来说，他们几个谁都比王惊蛰要出类拔萃得多了。<br/>　　<br/>　　    林文赫一出场浑身上下透着股死气沉沉的感觉，菜刀文江湖气息浓重，至于王令歌那就更不用说了，妥妥的大家气质，午桥出轨多年，走阴阳路，很有种老气横秋的稳重味，所以说人不可貌相这话说的并不太准确，因为总有那么一小措人是出场自带背景音乐的。<br/>　　<br/>　　    王惊蛰给双方介绍了下，特别是在四人身份上，他是一点都没含糊的朗朗的吹起了牛逼，不过吹的也不算多离谱，毕竟有些事都是有据可查的。<br/>　　<br/>　　    就像王惊蛰说午桥是东三省的出马仙时，黄九郎就点头说了声听过久仰，提起王令歌是岭南王家大公子，他就更加动容了，还主动套了两句近乎，说很想结实令尊但却无缘一见什么什么的。<br/>　　<br/>　　    有的时候，你不吹吹牛逼，人家也真不重视你啊，但主要还得是你自身本事过硬才行。<br/>　　<br/>　　    最重要是这几个人一来，就相当的云淡风轻了，一看就是见过世面的那种。<br/>　　<br/>　　    人员一到位，黄九郎就已经把封存的电脑拿出来开机，直接把画面就切到了昨天水下探古城，进入内部的那一段上，他跟王惊蛰说道：“你来介绍下吧？”<br/>　　<br/>　　    “啪”王惊蛰点了根烟，咬着烟嘴，一根手指指着电脑屏幕说道：“我长话多说，其实也没多复杂……”<br/>　　<br/>　　    王惊蛰简单的点了下抚仙湖水下古城的概况，历史方面他不是专家，随意的提了几句，然后话题一转就到了水下古城上，从水中尸体开始，一直讲到了那座祭坛。<br/>　　<br/>　　    虽然这几人都挺有见识，个个都一身精彩，但听闻抚仙湖下的尸库，和一座偌大的古城也着实惊了够呛，仅仅属于勉强能接受的范围而已，也惊叹的半天没回过神来。<br/>　　<br/>　　    视频画面中播放的，也是王惊蛰和耿明还有李菲儿等人深入古城内部的这一段，看的他们都挺皱眉的。<br/>　　<br/>　　    到了祭坛那一段的时候，王惊蛰按了下暂停，画面放大，他接着说道：“这就是出事的地方了，当时我在祭坛上，这个圆鼎上面有一副祭祀图，讲述的应该是两千多年前，祭祀的过程，察觉到血祭的时候，我就觉察到身后有气息在波动，虽然不确定要发生什么，但我的感觉告诉我，得先离开才行……”<br/>　　<br/>　　    “唉？你等下”林文赫忽然挥手打断了王惊蛰。<br/>　　<br/>　　    “嗯？”<br/>　　<br/>　　    林文赫指了指画面中祭坛四周那队士兵示意他放大，然后就凑了过去，仔细盯着士兵身前军服上的图腾，这幅画面其实照的比较清晰，当时王惊蛰来到祭坛前时，曾经打量过士兵的尸体，他身上携带的摄像头正好冲着对方。<br/>　　<br/>　　    “啪”林文赫敲了下屏幕，抬头笃定的说道：“这是尝羌王的亲兵”<br/>　　<br/>　　    “唰”黄詹，张生利还有考古队的人，全都齐刷刷的望向了他，之前他们曾经判断这些士兵有可能是滇国过往的卫戊士兵，但具体是哪一位国王的，则就不知道了，但没想到被林文赫一句话就给点了出来。<br/>　　<br/>　　    黄詹当即惊愕的站了起来，来到他面前皱眉说道：“你怎么知道这是尝羌的亲兵？”<br/>　　<br/>　　    林文赫说道：“滇国的图腾国兽为蛇，前几任滇国国主的黄袍和亲兵上都有这种图腾，但到了尝羌时期，汉武帝派使臣王然于，柏始昌等为开辟南方丝绸通商路到了滇西一带，尝羌为了整个族人的和睦不许通道经过滇国，于是西汉王室派兵数十万征伐西南蛮夷，滇国大败……再往后，柏始昌又再次出使滇西，说服尝羌归顺，他最后不得不答应了，于是便被赐予了云南滇王王印，其中有一条就是允许他们国兽蛇腹下生爪，你看那些亲兵衣服上的图腾，都有两爪，这就是尝羌的亲兵无疑了”<br/>　　<br/>　　    林文赫一席话落，这帮考古和历史专家全懵了，因为这段历史他们并不知晓，也从来没有在哪本古书上看过。<br/>　　<br/>　　    文峰皱眉说道：“史书和一些典籍上并没有这一段，你是从何得知的？”<br/>　　<br/>　　    林文赫反问道：“你们能掌握所有的滇国史书，典籍？”<br/>　　<br/>　　    文峰顿时被问的哑口无言，黄詹没有接着这个话题，而是往下问道：“那这么说来，抚仙湖水下的这座古城，应该就是尝羌这一任的了？”<br/>　　<br/>　　    林文赫想了想，随即伸手敲了下电脑，画面再次转动，当视频中出现那座金字塔的时候，他又按了下暂停，仔仔细细的看了片刻，点头说道：“八九不离十了，现在发现的六七座滇王墓都搞不清楚墓主和年代，因为记载的太少，不过关于尝羌的记载却有挺多，他曾经统领羌人往云贵川一代开发，一直推到了广汉鸭子河南岸，也就是现在的三星堆遗址……”<br/>　　<br/>　　    黄詹顿时低下头，皱眉看着画面里的金字塔，脑袋里面似乎有点什么念头但是没抓住，这时李菲儿忽然插嘴说道：“三星堆中曾经有一处遗址，和这个金字塔很像，你的意思是尝羌墓是仿造这个所建造的？”<br/>　　<br/>　　    林文赫耸了耸肩膀，说道：“证据我拿不出来，我只会推论，从年代上来看三星堆和古滇国尝羌时期，是差不多在同一个年代的”<br/>　　<br/>　　    黄詹听闻，叹了口气，点了点头后深深的看着林文赫，说道：“你是怎么知道这些的，我就不问了，料想你也不会说的，我们考古和历史圈里都流传一句话，说一个最顶级的摸金校尉要是论知识储备的话，可能比我们这些专家教授都要丰富了不知多少，你要是想进入我们的领域，可能自己的饭碗都保住了”<br/>　　<br/>　　    林文赫讪笑着呵呵了一声，没有接话，他自然不可能告诉对方，他知道的这些有一部分是从祖上盗墓中挖掘出来的，有一些是自己走南闯北发现的，这玩意儿你哪里能随便往出所，一说就有毛病了。<br/>　　<br/>　　    这一点确定完了后，王惊蛰又把视频点开，直到他自己出现在祭坛上，圆鼎中那幅祭祀图画露出来时，几人才眯着眼睛仔细的打量着。<br/>　　<br/>　　    “这个血祭，有点不太对劲啊……”午桥忽然凑到电脑前，狐疑的指着屏幕说道：“这不是传统的血祭，这个祭坛上的花纹是一种法阵，萨满教中曾有过记载”
