        &lt;h3&gt;第三卷第九十四章  寒伯安的小幸福&lt;/h3&gt;
<br/>
<br/>        寒伯安开着车，他要去见他的安男。
<br/>
<br/>        安男说了，让他元旦再过去，他心里还是很不舍的，和安男在一起，心里真的有一种甜甜的感觉。
<br/>
<br/>        元旦还得几天。
<br/>
<br/>        想着安男见他那羞羞的温柔似水的样子，他的心情有点飘。
<br/>
<br/>        现在他倒是越发的喜欢她黏着他了。
<br/>
<br/>        让轩轩帮忙寻找手镯的事情不知道怎么样了？
<br/>
<br/>        郝景文和陈思宇见面之后陷入沉思中，看样子陈子昂的公司遇到了麻烦，他们也有小量的投资，他们的app用得很顺手，推出市场也有一段时间了，效果确实还不错，原本后期还想和她们合作，可是如果陈子昂的公司资金链断裂会面临什么样的结果？
<br/>
<br/>        她们的耗材到底有没有市场？
<br/>
<br/>        郝景文回到公司之后让人发起了一份民意调查，有些人持怀疑态度，有些人认同。比例各占一半，而且他查了相关的资料，发现耗材每年都成递增状态。
<br/>
<br/>        他写了一份详细的资料，将资料分别发给江俞轩和寒伯安，后面还分析了陈子昂公司目前状态。
<br/>
<br/>        寒伯安虽然怼郝景文说过：“看着我的如花美眷飞舞着三千青丝奔向我的情景那将是世界上最美好的画面。”这句话，但是他知道他的安男不会飞奔向他，她会浅浅而笑、满含深情的站在门口等他，对他说：“你不嫌累吗？经常这样跑来跑去？”然后伸出小手，扬起明媚的笑脸，香一下，再十指紧扣，相依着走回家。
<br/>
<br/>        安男做得一手好饭，没有在一起之前他并不知道安男会做饭菜，他以为安男和众多的富家女孩一样十指不沾阳春水，但是他的安男是一个宝藏，不但是琴棋书画，文采斐然，而且还是一个美食家。
<br/>
<br/>        正是当下流行的那种：上得厅堂下得厨房的新时代女性，独立、个性、又有情趣。
<br/>
<br/>        想想当初自己嫌弃她的时候，他都想骂自己一句，安男其实也就是和他在一起的时候，才会黏着他，平时各自上班，忙得不亦乐乎，哪有时间黏在一起？
<br/>
<br/>        他也想到了郝景文说的，不如让安男和他一起回上海？反正寒家有些媳妇是不用上班，只负责打扮的漂漂亮亮，在家当太太就好，偶尔陪着老公出去应酬一下。
<br/>
<br/>        “嗯，今天见了安男可以试探一下，估计她是不会答应的。”寒伯安还是了解安男，看似温柔，其实是非常有主见的人。
<br/>
<br/>        当他到家门口的时候，果然，他的男男站在门口张望着，
<br/>
<br/>        “累吗？”安男扬起小脸，眉眼温柔得就像盛开的迎春花，更像是四月的清风，轻轻的划过他的心田，激起点点涟漪。
<br/>
<br/>        “见到你，所有的疲惫就会烟消云散，你是我温暖的港湾。”寒伯安低下头轻轻的抵在安男的额头。
<br/>
<br/>        他是寒家大少，掌管着寒家的一切，整天面对着各式人等，工作、应酬、决策，无论何时何地都要保持着一颗清醒的头脑，应付着各种人。
<br/>
<br/>        只有在安男这里，他的心是安宁的，平静的，满足的。
<br/>
<br/>        安男轻轻的拉起他的手：“回家了。”
<br/>
<br/>        这个世界上所有的情话都抵不过深爱的人的一句：“回家了。”
<br/>
<br/>        庭院青翠，夜灯浅暖，十二月份的天气在寒伯安来说就是温暖的春天。
<br/>
<br/>        俩个人相携着走回房间。
<br/>
<br/>        “饿了吧？”安男看着寒伯安那手中涂满泡沫的修长白净的手。
<br/>
<br/>        “饿了，想吃。”寒伯安含着笑意的眼让安男的脸又飞起了两朵红云，在灯光下越发妩媚。
<br/>
<br/>        安男嗔怪的看了一眼寒伯安低声说：
<br/>
<br/>        “饭已经准备好了。”每次吃个饭在他的嘴里都会变成另外的一种意思。
<br/>
<br/>        她转身准备去厨房，却被寒伯安轻轻的拥住。
<br/>
<br/>        庭院清幽，摇曳着的青翠的万年青舞动了纷纷的雪花，那洁白的精灵在欢乐的轻声呢喃，世界裹在温暖的柔情蜜意之中。
<br/>
<br/>        饭凉了。
<br/>
<br/>        安男重新热了饭菜，虽然热了一遍，但是饭菜看起来清爽可口。
<br/>
<br/>        “男男，我让人打听了另外一只手镯的下落，还没有给反馈过来。”
<br/>
<br/>        “不着急，妈妈已经等了这么多年，也不在乎再等几年。茫茫人海如大海捞针，妈妈不是没有找过，只是没有任何消息。”安男柔声说。
<br/>
<br/>        “元旦放假准备怎么过？”
<br/>
<br/>        “你说呢？”
<br/>
<br/>        “我准备去北京，和轩轩、文文商量一下事情，刚好大家都放假了，也能聚一聚，你随我一起吧。”
<br/>
<br/>        “好的，很想见一下你的兄弟们，还有你说的那个眼睛和我有点像的人呢。”
<br/>
<br/>        “那就这样说定了。”寒伯安定定的看着安男。
<br/>
<br/>        “怎么了？”安男摸了一下脸。
<br/>
<br/>        “好看。”寒伯安接着说：“结婚后我们住一起？”
<br/>
<br/>        “难道现在不是住一起，证都领了啊。”越是受过高等教育的人，越是对婚姻忠贞，她觉得他们现在已经是结婚的人了。
<br/>
<br/>        “我是说去上海。”寒伯安看着安男，不放过她的神情。
<br/>
<br/>        安男看了一眼寒伯安，其实有时间寒伯安很霸道的。如果寒伯安真的让她过去，她也不能不去。但是她可不想做金丝雀。
<br/>
<br/>        “我可以尝试着将工作调过去。”安男轻声说。
<br/>
<br/>        “还是你聪明，那我就准备看看你调动的事情。”寒伯安眉开眼笑，其实安男只要答应去上海就可以了，她爱干什么就干什么。
<br/>
<br/>        她是一个独立的女性，她有自己的事业。她是飞翔、自由的鸟儿而不是笼中的金丝雀。
<br/>
<br/>        俩个人的世界是温馨的甜蜜的。
<br/>
<br/>        郝景文将邮件发给寒伯安和江俞轩的时候，江俞轩因为工作的原因，直到很晚才给他回信息。
<br/>
<br/>        江俞轩详细的问了郝景文有什么打算，并且说公司的事情他还是要管的，不管怎么样，这个公司已经和他有了感情。
<br/>
<br/>        “我们先不要决定，等大哥的消息，原本我们打算元旦和你们聚一聚的，等元旦我们在商量吧。”郝景文说。
<br/>
<br/>        “ok，我已经买了回北京的机票，回去看看情况，陈子昂也在返程的路上。”
<br/>
<br/>        “那我们元旦见。”
<br/>
<br/>        寒伯安回家之后就想和安男在一起，天大地大，老婆最大，他们最近都是各自忙各自的，都没有时间好好在一起，这和安男在一起便将手机和电脑一股脑的都关机了。
<br/>
<br/>        他没有看到郝景文的邮件，也没有看到江俞轩和郝景文的聊天记录。
<br/>
<br/>        但是他们都会在元旦相聚北京，而陈子昂和安男也会相见，寒伯安口中说的眼睛特别像的俩个人会有什么样的联系，他们谁都没有想到。
