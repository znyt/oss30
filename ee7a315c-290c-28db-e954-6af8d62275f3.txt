　　大唐科学院龙头老大遇袭，重要资料丢失，这么大的事情根本瞒不住人，还没有等到第二天早上，来到咸阳的那些大佬们几乎全都收到了消息。
<br/>　　这下子谁也没有心思去参加什么狞猎活动了，一个两个急吼吼的就开始往长安赶，在第一时间便回到了长安城。
<br/>　　毕竟大唐高层只要是个人就知道，大唐科学院几乎就是皇帝的命根子，现在有人在皇帝的命根子上踢了一脚，虽然不知道蛋碎没碎，可是被迁怒绝对是免不了的，就算是心再大的人也不可能在这个时候不有心情去狞猎了。
<br/>　　另外不有一点就是，事情当时发生在咸阳火车站，偏偏好死不死那些世家和士族之人又全都集中在那里，这样一来所有去到咸阳的人都是黄泥掉进裤裆里，不是屎也是屎。
<br/>　　所以他们必须尽快赶回长安，证明自己的清白，若是回去的晚了，被皇帝怀疑上，说不定又是一轮大清洗。
<br/>　　但事实证明，李承乾并没有那么疯狂，事情发生以后他只是在第一时间赶到了医学院，见了探望了重伤的唐善识，并且在小唐的病房里面待了很久。
<br/>　　而等到李承乾出来的时候，大唐科报科便收到一条命令，全国缉拿独孤玉凤，黑子就地免职囚禁到皇室大牢之中，情报科由杨天接手。
<br/>　　没想到，真的没想到，下手的人竟然是独孤玉凤，这个女人难道是疯了么？身为情报科副科长的她怎么会干出这样的事情？
<br/>　　不管是李世民还是李承乾，说起来对待独孤家都不薄，尤其是太上皇，这次出海旅行还专门带上了独孤青云和他的儿子独孤玉林。
<br/>　　结果就算是这样，独孤玉凤还是干出了这样的事情，这到底是为什么？
<br/>　　正所谓世上无难事，只怕有心人，在没有人肉搜索的大唐，尽管事情已经时隔多年，但一些有心人还是把独孤玉凤与李承乾当的龌蹉翻了出来。众人也都想起了当年独孤家被李承乾恶意陷害，独孤青云被流放的事情。
<br/>　　事情的前因后果都找到了，独孤玉凤的行为也不变的可以理解了，只不过众人现在想要知道的并不是那个女人会面对什么样的惩罚，他们真正关心的是独孤玉凤到底拿走了什么？人在哪里？是否有机会把那些东西拿到手。
<br/>　　……
<br/>　　称心在事情发生的第二天一早同样接到了消息，飞鸽传书在大唐眼下已经得到了很好的应用，咸阳距离长安又不是很远，所以他知道这件事情的时间并不比任何人晚。
<br/>　　只是让他有些不解的是，为什么独孤玉凤会在这个时候发动，按说这个女人其实有很多机会接近李承乾，如果好真的想要报复，完全可以直接杀死自己的仇人，根本没有必要这么麻烦。
<br/>　　李毅恒接到消息的时间并没有比称心晚多少，他同样对这件事情抱有一种怀疑的态度，为什么那个独孤玉凤会在这个时候发起行动？难道直接杀死皇帝不是更好么？又或者这里面其实另有隐情，是众人所不知道的？只是被忽略了过去？
<br/>　　混在拜占庭人队伍中的他思前想后，与自己身边的几个随员讨论了很长时间，都没有得出一个有用的结论。
<br/>　　但有一点可以肯定的是，他们一定要先一步找到独孤玉凤，无论如何也要从她的手中拿到那份资料。
<br/>　　毕竟能让唐善识随身带着的资料一定不是什么简单的东西，搞不好就跟大唐那些先进的装备有关系。
<br/>　　而且就算不是那些先进的资料，其它东西也不是没有任何意义，拿到手之后就算是自己不能用也可以卖给别人，反正这个世界上最不缺的就是有钱人。
<br/>　　……
<br/>　　杨天在接手了情报科之后，同样也是一脸的茫然，虽然他回来之后一直在情报科里任职，可再怎么样他也想不到以前教过他很多东西的那个女人竟然会干出这样的事情，同样他也不相信黑子会与独孤玉凤合谋。
<br/>　　所以杨天第一时间便进了宫，找到李承乾：“陛下，臣觉得此事并不是表现上看着那么简单，关于墨科长和独孤副科长的事情您是不是再重新考虑一下？”
<br/>　　“还要考虑什么？唐善识现在就躺在医学院里，那十个护卫同样在那里，你觉得他们会撒谎？就算是他们在撒谎，那你告诉我独孤玉凤又在哪里？为什么她会一直不出现？”李承乾面沉似水，直视着杨天：“杨天，从大的方向来讲，你在倭国的任务完成的很好，深得朕的信任，从小的方面来讲，你是朕的舅兄，说来也不是外人。你能不能用你多年的经验来告诉朕，这件事情到底哪里不简单，朕到底还要重新考虑什么！”
<br/>　　杨天可以看得出来，自己的这个妹夫现在真的很生气，如果不是因为自己妹妹的关系，很有可能，因为刚刚自己的那一句话，就会被拖出去狠狠抽上一顿板子。
<br/>　　可是，杨天也有自己的原则，他是搞情报的不假，但他却不是冷血无情之人，在倭国的时候因为身边没有任何人可以信任，所以他表现的心狠手辣。
<br/>　　但是在回到大唐之后，在情报科里待了一段时间之后，他发现不管是黑子还是独孤玉凤，其实都是很好的人，对他也都很不错。
<br/>　　他相信自己对人的判断，也相信黑子和独孤玉凤都不是那种会为了私利而损害大唐利益的人，至于说独孤玉凤想要报当年的仇，这话除了忽悠一些外人之外，只要是与当事人有过接触的人都清楚，这根本不可能。
<br/>　　所以在考虑了良久之后，杨天还是硬起头皮对李承乾劝道：“陛下，就算是您不想考虑，那您能不能让我见墨子羽一面？”
<br/>　　墨子羽是黑子的本名，不过因为不常用，李承乾反应了半天才想起杨天说的是谁。
<br/>　　看着自己这位舅兄眼中的那份执着，李承乾最终点了点头：“可以，不过你只有一次机会，好好把握吧！”
