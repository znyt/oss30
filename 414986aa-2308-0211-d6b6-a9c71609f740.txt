　　“够了！”再一次将两人的争论喝止，看着歇斯底里的李元昌，老李渊眼中闪过一抹失望。
<br/>　　再看看二儿子，现在李二像是要睡着了一样，脸上全是满漠不关心，似乎完全不担心李承乾会被驳倒一般。
<br/>　　直到此时，李渊才明白，自己还是过于冲动了，自己家这个老二本身就不是个好对付的角色，他的大小子，也就是自己这个大孙子，似乎比儿子更难对付。
<br/>　　如果……，如果李承乾所作的一切完全是脱离开李二的指导的话……，李渊突然有一种想笑的感觉。
<br/>　　看着老神在在的二儿子，李渊决定不把自己现在的想法告诉他，让他等着吧，再过十年，当自己的大孙子真正的成人之后，估计老二就知道自己现在是什么感觉了。
<br/>　　“皇爷爷，父皇，今天既然元昌王叔说起孙家血案，承乾也正有一件事情需要禀报。”沉默中，李承乾突然开口说道：“昨天夜里，情报科‘第七小组’得到线报，说城外有一处庄园被歹徒围攻，他们连夜赶去的时候，在庄园外面截住了一队歹徒，将其击杀之后，解救了十八个人质出来。“
<br/>　　“哦？人质？都是什么人啊？”李二突然来了兴致，从椅子上直起身子，正色问道。
<br/>　　李承乾看着脸色隐隐有些发白的李元昌，淡笑着说道：“孙家家主孙辉等一十八人。”
<br/>　　“不，不是，你骗人，你在骗人。舅舅他们已经死了，他们已经死了！”脸色惨白的李元昌疯了一样咆哮着，用手指着李承乾，对李渊说道：“父皇，你快点让人把他抓起来，这一切都是他在其中搞鬼，都是他在搞鬼，人都是他杀的，都是他杀的！”
<br/>　　“闭嘴，滚，滚出去，给朕滚出去。”李渊已经不想听李元昌说什么了，他根本就斗不过李承乾，却非要自不量力的去比划，现在丢人现眼不说，连带着自己也跟着臊的慌。
<br/>　　“父皇……”李元昌被骂的一愣，眨眨眼睛，半晌之后才看看冷笑的李承乾，“噗通”一声跪了下去：“承乾，是我错了，是我错了，是我鬼迷心窍，我不该收胡老八当护卫，我应该早点杀了他。承乾，叔错了，叔真错了，你别说了行么？别说了。”
<br/>　　李元昌已经预料到李承乾接下来会说什么，他想要阻止，因为如果李承乾把后面的话说了，那么他这辈子的名声就算是彻底的完了。
<br/>　　虽然他现在的名声也不怎么好，但总好过……。
<br/>　　而且，他是叔，李承乾是侄，反过来这一跪就是逆转人伦，这也是给李承乾出了一个难题，看他到底会怎么接。
<br/>　　结果，李元昌错误的估计了自己的份量，李承乾根本鸟都没鸟他，反而直接问道：“元昌叔，据孙家家说孙辉说，他们是你亲自绑起来交给贼人的，这事是不是真的啊？如果元昌叔否认的话，小侄可以把他们都带过来，咱们当面对质，也好给元昌叔正正名声，如何？”
<br/>　　毁人不倦，是的，这就是毁人不倦，李承乾此话一出几乎把李元昌的一切都给毁了。
<br/>　　母族被人屠杀，族人到他的地盘去避难，结果他不保护好不说，反而把人绑起来送给贼人，这是一种什么行为？说是狼子野心也不为过。
<br/>　　而且如果按照他前面所说的，是李承乾想要杀他，也是李承乾想要屠灭他的母族的话，那么他正应该与族人一起同仇敌忾才对，怎么可能把族人献出去呢。
<br/>　　李元昌彻底的蔫了，连一句反驳都没有，这已经可以证明，李承乾说的话句句属实。
<br/>　　李渊也没什么可说的了，自己的儿子干出这种事情，他这个当爹的不打折他的腿就不错了，怎么可能再去找孙子的麻烦，更不要说孙子还是救人的一方。
<br/>　　甚至李渊心里已经很清楚的知道，这事儿一定就是李承乾干的，孙家就是李承乾屠的，围住李元昌庄子的人也一定是李承乾派的。
<br/>　　可是没有证据，唯一的证人李元昌还像死狗一样被驳了个体无完肤。
<br/>　　最后，无奈的李渊摆了摆手，示意手下侍卫把李元昌带去宗正寺，为他的‘诬告’行为去承担自己的责任。
<br/>　　自己怎么就生了这么一个怂货呢？
<br/>　　按当时事情的发展开看，如果李元昌真敢走出来的话，李承乾的人未必敢杀他，就算是李承乾亲自在场，也未必敢杀他，这样一来所有的事情就会发生转变。
<br/>　　或者就算是李承乾真的把他给杀了，自己这个当爹的也可以有理由来发飙，甚至借此来废掉李承乾的太子之位也不是没有可能。
<br/>　　可是现实就是现实，李元昌怂了，他没敢出来，没敢拿自己的命去赌，那么有现在这样的后果他就只能自己承担。
<br/>　　这并不是皇家无情，这是现实，不单单是皇家会有这样的事情发生。
<br/>　　举个例子来说，就像现在银行贷款一样。
<br/>　　企业越好，越不需要贷款，就越是有银行上门问询问，企业需要需要额外的资金，这是因为企业效益好，有还款能力，所以银行十分希望这样的企业来贷款。
<br/>　　而反之一些濒临破产的企业，他们十分需要银行的贷款来救活企业，可是银行却推三阻四，十分不愿意把钱贷给他们。为什么？因为银行怕企业还不起钱，那样贷出去的钱就没办法收来。
<br/>　　往大了说这是为国家负责，为储户负责，必竟银行的钱不是天上掉下来的，也不是海里捞上来的，那是都是老百姓的血汗钱，舍不得花存起来的。
<br/>　　如果被那些效益不好的企业给败光了，这就是对国家和人民的不负责人。
<br/>　　道理就是这样，现实就是这样，并不是说弱者就一定会有人去帮助，有时候帮助强者才是这个世界上最根本的规则。
<br/>　　人们会同情弱者，会可怜弱者，但却不会全力以赴，豁出一切去帮助弱者。
<br/>　　所以才有‘锦上添花易，雪中送炭难’，这样的词句诞生。
<br/>　　皇家，也不过是一个微缩的小社会罢了，关系再复杂，也不会逃出规则的泛围。
