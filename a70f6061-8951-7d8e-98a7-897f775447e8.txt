　　当南署基地的大战进行中时，也是于南部联盟军北上的最后阶段。
<br/>　　得知南署基地的大战竟然逆袭时，这些将领都像打了鸡血一般，一连攻下数座基地。
<br/>　　北部地区的大多数基地早已招降，仅剩还在坚持的基地也在南部联盟军这一鼓作气的猛攻气势中败下阵来，招降接二连三，一路都对南部联盟军敞开了基地大门。
<br/>　　上京基地就更别说了，叶子博的无故失踪就是一个导火线，南部联盟军还未靠近的时候，其内的管理层就已经大乱，狼牙暗部的成员几乎没花什么力气，就完全将其渗透。
<br/>　　紧接着便没有任何悬念的，第一批速度最快的南部联盟军队伍，就已经来到了上京基地。
<br/>　　平定内乱，重新建立管理体系，一顿没日没夜的忙活之后，短短三天的时间牧司令和洛冥就已经重回上京基地！
<br/>　　只是这一过程中，狼牙暗部的成员还是没有暴露，而是通过了各种迂回方式将事态推波助澜。
<br/>　　“没想到我们这么快就回来了。”牧司令站在他用了两年的办公室内开口，望着屋内陌生的陈设语气复杂。
<br/>　　洛冥则是看着窗外，声音带着冷嘲：“看看这个上京，看看！乱成了什么样子，这城内城我看还不如推了，这还是基地么？这简直就是一个乱城！把我们赶走，叶子博就把上京管成这样？”
<br/>　　听到叶子博的名字，牧司令的声音也随之变化：“他的失踪你怎么看？”
<br/>　　洛冥摸了摸下巴：“叶子博在南署基地的大尸潮开始之前就失踪，应该不是被楚涵那小子吓的，大概是看到了大势已去？”
<br/>　　牧司令摇头：“未必，他的身后不是站着白家么？”
<br/>　　“白家也不是傻，不可能永远扶持一个废物。”洛冥毫不掩饰对叶子博的厌恶，翻着白眼道：“分战界线的战败失利，南部联盟军北上攻势凶猛，他这就是看清了状况，想逃啊！”
<br/>　　牧司令沉默了一会儿，在此摇头道：“总觉得这事有古怪，白家也太安静了。”
<br/>　　洛冥点头：“的确，我们内战的动静这么大，四大神秘家族一家都未曾出面，一点风声都没有，实在让人心慌啊！”
<br/>　　牧司令眼神带着微寒：“可能是觉得还不够资格让他们关注吧？华夏历史这么悠久，战乱纷纷这么多次，神秘家族什么时候管过？”
<br/>　　洛冥苍老的面容上忽的勾起一个贱贱的笑容：“嘿嘿！不过话说我们回到上京的速度也太快了点吧？是不是有猫腻啊？”
<br/>　　牧司令看了他一眼：“你是想说狼牙暗部？”
<br/>　　“可不是嘛！”洛冥眼珠子一转：“要不要问问你外孙？”
<br/>　　牧司令摆摆手：“他是暗部成员，不可能对无关紧要的人透露消息，哪怕我是他的外公。我也觉得这其中有狼牙暗部在运作，他们不暴露的理由你也不是不能理解，何必打破砂锅问到底？”
<br/>　　“好奇啊！”洛冥耸耸肩，紧接着眼底透露出了一丝谨慎：“外加有些忌惮。”
<br/>　　“楚涵的狼牙的确太霸道，无论是基地还是战团都首屈一指，这暗部也很逆天，我都不知道从分界线一战开始，我们一路走来有多少场战争都有狼牙暗部的参与。”牧司令说出这番话的时候平淡无比，不带一点其余色彩：“你是担心哪怕狼牙战团在南方边界线抵挡尸潮，光是凭借狼牙暗部这一组织，就可以搅动整个华夏的风云？”
<br/>　　洛冥收起了玩世不恭，神态认真：“不错，人都是会变的，楚涵……还是太年轻了！”
<br/>　　牧司令却忽然一笑：“就算他真的这么做又怎么样？这和你我待在上京管理各大基地，有什么区别？说不定他会做的比我们更好。”
<br/>　　洛冥愣了愣，良久后才忽然一笑：“说的对，是我白担心了，楚涵真要想做些什么早就得手了，当下却只有防守姿态，显然对这些不感兴趣。”
<br/>　　牧司令走到桌边，随手将一份档案袋内的资料拿出：“叶子博的情报还真不少，这里竟然有楚涵的全部事件记录。”
<br/>　　洛冥走过去随手翻阅起来，语气带着一丝感叹：“咱俩也算是很早就认识了楚涵这小子，却没有客观分析过这个人，这叶子博的调查倒是全面！”
<br/>　　牧司令挑挑眉：“你发现了什么？”
<br/>　　“你看这里。”洛冥指着资料内的某一页：“这篇是记录了狼牙基地的成立和发展模式，甚至管理体系的细节都有。”
<br/>　　牧司令点着头，目光中带着欣赏：“能想出这套体系，可见楚涵的确是大才，华夏……他应当有能力接手。”
<br/>　　这番话已经彰显出了某种趋势，若是其他人听到恐怕会慌张无比，但对牧司令熟悉至极的洛冥，在之前的交谈之中就已经察觉出了他这位老搭档的想法。
<br/>　　于是洛冥没有半点意外，只是忽然一声惊疑：“矮？不对啊！”
<br/>　　牧司令笑了笑：“一把年纪了还一惊一乍的。”
<br/>　　洛冥抬起眼，目光有些呆滞：“我觉得你可能需要另选一名继承人了。”
<br/>　　牧司令皱眉：“怎么说？”
<br/>　　“虽说这狼牙从上到下的管理体系，包括两座城池的大致系统，都是楚涵拟定的初步方案，管理能力可见一斑。”洛冥说着声音一顿，道：“但是你有没有发现，楚涵每次都是把大纲定下之后，就撒手不管！”
<br/>　　“恩？”牧司令有些没明白。
<br/>　　“看这里。”洛冥一边翻阅一边说道：“狼牙基地的刚成立楚涵就让尚九娣代管，自己满世界瞎跑，又是打架又是杀人又是搞出一连串动静，根本对狼牙内部的情况不闻不问，也半点不担心尚九娣会不会将他的功劳取而代之。”
<br/>　　“那还不是因为尚九娣对他深情，取而代之这完全不可能。”牧司令随口一答。
<br/>　　“是这样没错，但是你再看这里。”洛冥继续道：“狼牙战团第一批成员是楚涵亲自带出来的，手把手的教，各方面的训练全都包含，还搞出了一套极为可怕的训练系统。”
<br/>　　牧司令继续点头：“对，第一批的成员还是他从上京带走的，很多人还是你我亲自挑选，眼下这些老兵还健在者，最差的都是少将，所以呢？”
