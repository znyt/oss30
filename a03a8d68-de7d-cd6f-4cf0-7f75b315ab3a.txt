        他们两之间没有任何话题上的交流，也没有过任何的肢体语言，明明互相认识，却像陌生人一样，各自做着各自的事情。
<br/>
<br/>        诺筱颖从未将马智杰放在心上，她只知道，马智杰这个男生很安静很内向而且不大爱说话。
<br/>
<br/>        但他俩一下午都独处在一个宿舍里，哪怕宿舍大门是敞开的，不知情的人，还以为她诺筱颖才是他马智杰的女朋友。
<br/>
<br/>        到了下午四点钟的样子，诺筱颖就会收拾自己的东西只身离开。
<br/>
<br/>        她前脚刚走，马智杰后脚就跟了过去，直到他看到她安全地上了公交车，他才放心地返回自己的宿舍里。
<br/>
<br/>        这样的周末，一过，就是一个学期。
<br/>
<br/>        半年的时间里，诺筱颖对何明旭几乎是随喊随到，百依百顺。
<br/>
<br/>        何明旭感冒发烧不想去看医生开药，诺筱颖逃课亲自跑过来，给他看病开药抓药熬药。
<br/>
<br/>        何明旭胃痛食欲不振，诺筱颖锅碗瓢盆，在宿舍里开小灶，给何明旭熬粥、炖汤。
<br/>
<br/>        甚至，何明旭逃课，诺筱颖还给他去代过课，只为了让他修满学分。
<br/>
<br/>        而何明旭搬出宿舍，哪怕已经和林若琴在校外租房同居了，却仍旧在欺骗着诺筱颖的感情。
<br/>
<br/>        这么好的女孩子，何明旭怎忍心去欺骗她的感情？
<br/>
<br/>        方浩博和章海昌可以对何明旭的行为视而不见，但他马智杰已经忍到了极限。
<br/>
<br/>        在童筱颖的记忆里，有关于马智杰的回忆与印象并不多。
<br/>
<br/>        一整夜，童筱颖都在浑浑噩噩中度过。
<br/>
<br/>        翌日。
<br/>
<br/>        傅老太爷的早膳时间，在家的人必须每一位都要到位。
<br/>
<br/>        动筷子之前，傅老太爷傅炎廷发现童筱颖未到，不禁皱起了眉头：“既然入了这园子，就要讲究这园子里的规矩。”
<br/>
<br/>        “爷爷，筱颖她身体不舒服，我已经吩咐管家给她单独送早膳过去了。”傅夜沉坐直了身子，一脸认真地回答道。
<br/>
<br/>        苏漫雪趁机火上浇油，添油加醋地说道：“爷爷，待会还是请个医生去看看弟妹吧！这弟妹和弟弟都领证这么久了，万一这个时候怀上了，只怕会影响上大学啊！”
<br/>
<br/>        “……”傅夜沉冷着脸，瞥了苏漫雪一眼，剑眉微蹙。
<br/>
<br/>        冷夫人倒是听不出苏漫雪的话外音，只顾着自己的心情，打趣地说道：“若是筱颖怀了小景的孩子，上不上大学都不打紧。倒是小雪你，你得加把劲，和我家小沉，也给我这老太婆生个大胖孙子出来。让你爷爷，抱一抱小曾孙。”
<br/>
<br/>        “阿沉说了，等我念完大学，再跟我要孩子。”苏漫雪故作娇羞地说。
<br/>
<br/>        傅夜沉耐着性子听着，不做任何解释。
<br/>
<br/>        “那待会请个医生，去筱颖屋里给筱颖瞧瞧。”傅老太爷傅炎廷一声令下完后，这才拿起筷子用早膳。
<br/>
<br/>        对于傅家这种复古家规，苏漫雪早已习惯。
<br/>
<br/>        就算她在傅夜沉的面前再造次，到了傅家大庄园里，在傅老太爷和冷夫人面前，她还是规规矩矩地不敢乱来。
<br/>
<br/>        她苏漫雪都知道要讨好傅老太爷和冷夫人，而童筱颖却这点意识都没有，苏漫雪不禁暗自在心底窃喜。
<br/>
<br/>        不过，话说回来，苏漫雪也能理解童筱颖的所作所为。
<br/>
<br/>        毕竟，这傅家大庄园和傅氏集团日后都与她童筱颖无关，所以，童筱颖确实没必要去讨好傅老太爷和冷夫人。
<br/>
<br/>        殊不知，在童筱颖的心里，家里的长辈不是表面上用来去讨好的，而是打从心底里用来去尊重的。
<br/>
<br/>        童筱颖和苏漫雪的三观不合，早在她两还是闺蜜的时候，童筱颖就已经体会到了。
<br/>
<br/>        屋外，有两个年轻的女佣在议论她和苏漫雪。
<br/>
<br/>        说苏漫雪如何招人傅家两位“大佬”的喜欢，说她童筱颖如何惨遭两位“大佬”的嫌弃。
<br/>
<br/>        童筱颖听着倒也未往心里去。
<br/>
<br/>        反正，她和苏漫雪是两个不同世界的人，她没必要去计较傅家的女佣对她的是非争议。
<br/>
<br/>        昨天晚上，她就没吃什么东西，而且又一整晚未睡，再睁眼的时候，已经临近中午。
<br/>
<br/>        童筱颖本想开门透透气，刚走到门口，门栓子都未来得及拉动，就听到门外那两个女佣的对话。
<br/>
<br/>        “这二少奶奶真的不如大少奶奶！”
<br/>
<br/>        “就是！”
<br/>
<br/>        她们两说话的声音渐行渐远。
<br/>
<br/>        为了避免尴尬，童筱颖真的是等她们走远了，她才将房门打开。
<br/>
<br/>        “咕——”肚子饿得咕咕叫。
<br/>
<br/>        童筱颖下意识地摸了摸肚子，又将房门的两扇门板给随手合上，转身进屋准备换衣服去找东西吃。
<br/>
<br/>        此时，傅夜沉忙完自己的事情，从院门外走了进来。
