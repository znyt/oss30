陆寒霆那样救了夏夕绾，蓝烟心里是很感激的，但是这不影响她讨厌他，大渣男一个。
<br/>
<br/>        “我不管陆寒霆和绾绾之间曾经发生过什么，他明天都要跟厉嫣然订婚了，那就不该再来招惹我家绾绾，况且他自己都说了，不要将他救人的事情说出来，那我们更不要让绾绾知道是他救了人，这件大衣丢了，丢在垃圾桶里！”蓝烟道。
<br/>
<br/>        夏邦有点犹豫，“夫人，真的要丢垃圾桶？”
<br/>
<br/>        蓝烟一把抢过了黑色大衣，作势就往垃圾桶里丢，“丢了！现在就丢！”
<br/>
<br/>        可是下一秒蓝烟又犹豫了，她看了看夏邦，小声道，“我觉得绾绾好像真的喜欢那个陆寒霆…”
<br/>
<br/>        “夫人，我也觉得那个陆寒霆很喜欢我家绾绾…”
<br/>
<br/>        两个人交换了一个眼神，“要不，这件大衣先送去干洗，留着看看？”
<br/>
<br/>        “行。”
<br/>
<br/>        两个人就这么愉快的决定了。
<br/>
<br/>        ……
<br/>
<br/>        陆家。
<br/>
<br/>        陆寒霆带着厉嫣然回去了，女佣们在准备晚餐，柳招娣笑道，“嫣然，你过来陪我说说话，寒霆，你爸叫你去一下书房，有事找你。”
<br/>
<br/>        陆寒霆去了书房，陆司爵并没有批阅文件，他沉稳挺拔的伫立在落地窗前，修长的手指里夹了一根雪茄在抽。
<br/>
<br/>        “回来了？右手怎么样了？”陆司爵单手抄裤兜里，俯身在烟灰缸里叩了叩雪茄的烟灰。
<br/>
<br/>        陆寒霆那张俊脸上没有什么表情，只是淡漠的掀了一下薄唇，“挺好的。”
<br/>
<br/>        陆司爵漫不经心的勾了一下薄唇，幽幽的目光看了过去，“你的手臂都废了，落下一个残疾，哪里好了？”
<br/>
<br/>        陆寒霆蹙了一下英气的剑眉，“没事的话我就出去了。”
<br/>
<br/>        他完全不想聊，转身就走。
<br/>
<br/>        这时陆司爵低沉的嗓音在身后传来，“你很喜欢那个夏夕绾？”
<br/>
<br/>        陆寒霆脚步倏然一顿，人转了回去，鹰隼般的狭眸盯着陆司爵，“你想干什么？”
<br/>
<br/>        “这么紧张干什么，我就提一下她的名字。”
<br/>
<br/>        陆寒霆还是很了解这个父亲的，他从来不会无缘无故的提到一个人的名字，“我是喜欢她，很喜欢的那种，为了救她我可以废了一条手臂，所以你别动她，如果你动了她，我也不知道自己会做出什么事情。”
<br/>
<br/>        陆司爵有点后悔了，从知道陆寒霆不要命的去撞车，他就后悔了，他不该放任他们自由的。
<br/>
<br/>        陆寒霆对夏夕绾的爱，比他想象的更加炙烈疯狂。
<br/>
<br/>        陆司爵将嘴里的一口烟雾吐出来，然后在烟雾缭绕里看向陆寒霆，“那你怎么不取消明天的婚约，你还有选择的机会。”
<br/>
<br/>        陆寒霆缓缓敛下英俊的眼睑，像一头危险的野兽突然收敛了一身的锋芒，透出几分自嘲来，“她不喜欢我。”
<br/>
<br/>        她不喜欢他。
<br/>
<br/>        她还玩弄他的感情。
<br/>
<br/>        陆司爵挑了一下英眉，没说话。
<br/>
<br/>        “她不喜欢我，我也在心里发过誓，再也不会将自己送上门让她践踏，不要告诉她是我救了她，也不要告诉她我为了她废了一条手臂，不然她又可以仗着我对她的喜欢肆意欺负我，我不想让自己活成一个笑话。”
