　　第232章
<br/>　　“程贤弟舍身杀敌，宁死不退，一如我们初见时定下的信念：精忠报国，死而后已。今日我仍是阳世一人，程贤弟已是阴间一鬼，人鬼永隔，此生难见。”
<br/>　　“今日我有何面目立于于天地间，有何面目见我九泉之下的程贤弟。和程贤弟相比，我得到的已经够多了，实在无颜再受一分一毫。”
<br/>　　“功虽成，身已殒，唯留正气长驻人间。”
<br/>　　“呜呼贤弟，哀哉贤弟，你若在天有灵，就睁眼看看，这帝国是否如你所愿。”
<br/>　　杨龙停伸开双手，朝向天空，白发披散，已然老泪纵横。
<br/>　　特么的！这是给我念悼文呐，你给我等着，给我等着。
<br/>　　看台下，程大雷把牙咬得嘎吱嘎吱响。
<br/>　　明帝挥挥手：“好了，好了，人死不能复生，你也要节哀。”
<br/>　　“臣祈陛下，请封赏程大雷，以告慰程贤弟的在天之灵。”杨龙停突然跪倒。
<br/>　　“这个自然，这样的勇士我不赏，岂不是让天下人寒心。”
<br/>　　台下的士子也被感动得热泪盈眶，瞧瞧，瞧瞧什么才叫干大事而不居功，什么叫一声朋友一起走，那些日子不再有……杨龙停就是天下做朋友的榜样呐。还有人居心险恶的映射战神大人，这种就是以小人之心度君子之腹，以萤烛之火揣测日月光辉。
<br/>　　明帝起身，攥住杨龙停双手：“你也莫要再推辞，帝国还要靠你们这些虎将。”
<br/>　　杨龙停低头：“陛下深恩，臣万死难偿。”
<br/>　　“还要为帝国多培养一些像你一样的英雄，如你这样的人多了，帝国才能长治久安，朕也可以安枕无忧。”明帝指了指台下的程大雷：“牛校尉的军符令信，便由你亲自交给他。”
<br/>　　“臣遵旨。”
<br/>　　视线落在程大雷身上，此刻他依旧站在看台下，而许多人看向他的眼神，难免就露出一丝羡慕。
<br/>　　这个仪式更像是某种传承，杨龙停是今日战神，程大雷会不会成为明日战神，未来的帝国朝堂，必然有程大雷的一席之地。
<br/>　　在艳羡嫉妒的眼神中，程大雷缓缓走上看台，他终于从看台下的观众，一步一步走上舞台。
<br/>　　杨龙停身边立着几名太监，他们手中托着官府，将刀，军印，以及给程大雷的官凭。
<br/>　　不知怎的，杨龙停感觉有些头晕，大概是太热的缘故吧，毕竟自己在太阳下立得太久了。好在，过了今天自己就可以回去了，远远的离开这个鬼地方。
<br/>　　此时程大雷已经立在他面前，垂着头，二人相距大概三步之遥。
<br/>　　这种不安感更加强烈，太阳穴砰砰跳着。杨龙停勉力遏制着，看了一眼面前的程大雷。他自然听说过牛三斤，据说明帝很看重他，将是未来帝国冉冉升起的将星。
<br/>　　同这种人还是要打好交道，毕竟他还年轻，未来不可限量。
<br/>　　杨龙停伸出手拍拍他的肩膀：“好男儿，帝国缺的就是你们这些少年英雄呐！”
<br/>　　台下的士子露出艳羡的眼神，如今的帝国，杨龙停不知是多少人的偶像，自从他来到京城后，许多人想见他一面而不得，但在今日，他竟然亲口夸赞程大雷。
<br/>　　杨龙停看着程大雷，微微皱起眉头。
<br/>　　照道理来说，此时程大雷应该跪下，先拜明帝，再拜自己。然后自己将军印交给他，再给他一些安慰鼓励的话。
<br/>　　可是程大雷垂手站在自己面前，一动不动。
<br/>　　气氛微微干了干，杨龙停笑了笑，道：“陛下大恩，我辈做臣子的当铭记在心，日后多为陛下做事，多为帝国做事。”
<br/>　　“我有一样东西，想让阁下还给我。”一直沉默的程大雷开口。
<br/>　　这声音隐隐有些熟悉，那种不安感更加强烈，杨龙停一时间有些失神，几乎是下意识道：“什么东西啊？”
<br/>　　“公道。”
<br/>　　程大雷已抬起头，目光一动不动的盯着杨龙停。杨龙停只看到一张长满胡须的黑脸，他愣了愣，注意到程大雷的眼睛。
<br/>　　他认出来了。
<br/>　　杨龙停和程大雷本来就见面不多，更何况已经时隔一年，大街上走个对脸，他都未必能认出程大雷。可此时此刻，此情此景，那砰砰跳动的心脏，一切的一切告诉杨龙停。
<br/>　　噩梦找来了。
<br/>　　公道！
<br/>　　世人都想要公道，世世代代人渴望明君，清官，都渴望不到，转而渴望侠客出世，侠客也求不到，就说公道自在人心。
<br/>　　想要公道的人是有福的，因为他们相信世间还有公道。可干干净净的人世间，为何会需要明君，清官，侠客，而人心这种东西，又常常和愚蠢相依相伴。
<br/>　　或者，世间根本无所谓公道。
<br/>　　那就靠你一手抓得到的东西，抓起它，狠狠的砸向那些不讲公道的人。
<br/>　　程大雷一手抓得到的东西是一柄剑，于是拔剑出鞘。
<br/>　　这一剑名曰直来直往。
<br/>　　变故来得太快，杨龙停也不过刚刚认出程大雷而已，就看到一柄闪着寒光的铁剑扑面而来。
<br/>　　来了，你就挡不住。
<br/>　　一剑刺中杨龙停喉头，然后，没扎进去。
<br/>　　程大雷愣了愣，今日杨龙停穿着布甲，这种布甲是仿照盔甲的样式，专门就是为出席各种礼仪场合用的，就连脖子都护在布甲内。杨龙停里面穿的软胄挡住了程大雷这必杀一剑。
<br/>　　程大雷僵了一刹那：呃，似乎有点尴尬。
<br/>　　可一刹那已能决定很多事情，在反应过来后，杨龙停探出一掌拍向程大雷胸口。程大雷毕竟小看了杨龙停，他从前也是在沙场上摸爬滚打，从小兵一点点做起，直到走到今日。
<br/>　　程大雷的反应也是半点不慢，没时间再刺出第二剑，程大雷抓住杨龙停手掌身体往后退，借着坠势，带着杨龙停一起坠下高台。
<br/>　　咚！
<br/>　　两个人一起砸下看台，狠狠扑在黄土中，沾满一身尘埃。
<br/>　　说时迟那时快，一切只在极短时间发生，看台上的明帝，文官武将，观战的士子，包括以呼延力为首的戎族谈判使节，也都没明白发生了怎么回事。
<br/>　　程大雷眼看就要一步登天，可帝国未来战神为什么要将如今战神狠狠拽下来。
<br/>　　看台下，程大雷握紧剑扑向杨龙停。
<br/>　　“匹夫，受死！”
<br/>　　
<br/>(本章完)
