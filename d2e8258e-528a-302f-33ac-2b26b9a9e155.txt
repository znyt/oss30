        “你怎么什么都懂？这也是你外婆教的吗？”傅夜沉嘴角微扬，好奇地问道。
<br/>
<br/>        童筱颖不假思索地回答：“那是因为我当初为了……”
<br/>
<br/>        话说到一半，她才意识到这种话不能说给大哥听。
<br/>
<br/>        她曾经和傅昼景在一起的时候，因为有想过两人是要孩子的，才去图书管里借阅过一些与备孕有关的书籍。
<br/>
<br/>        现在，傅昼景已经是过去式了，她就不应该再在大哥面前提及他了。
<br/>
<br/>        “为了什么？”傅夜沉顿时敏感起来，会不会是为了阿景？
<br/>
<br/>        “我不是吃过事后药嘛！怕对今后的怀孕造成影响，就特意为了这事去查阅了一些相关的书籍。”童筱颖干笑着打了个圆谎。
<br/>
<br/>        傅夜沉忽然意识到了什么，一脸认真地应和道：“也对，怀宝宝这事似乎不能太马虎了。我改天抽空，也去买一些这方面的书籍回来研究研究。”
<br/>
<br/>        童筱颖顿时语塞。
<br/>
<br/>        他若是研究了那方面的书籍，以后她就没法在他面前敷衍了。
<br/>
<br/>        “大哥，你那么忙，这种事情，就交给我去研究好了。你只要负责配合就行了。”童筱颖咧嘴一笑。
<br/>
<br/>        傅夜沉总觉得哪里不对劲，总觉得这小女人葫芦里卖的不是他想要的那种药。
<br/>
<br/>        “我是认真的！”见他一脸狐疑的样子，她很认真地补充。
<br/>
<br/>        傅夜沉这才放下心来，吻了吻童筱颖的唇。
<br/>
<br/>        当落以柔亲眼看着傅夜沉将童筱颖从跑车上抱下来，一路又抱上楼去后，站在她身边的郑忠威终于忍不住发话了：“小落，感情这种事情真的强求不来。学会放下，才是最好的解脱。”
<br/>
<br/>        “师父，我到底哪里比不上那个童筱颖？我觉得我比她更优秀。”落以柔毫不忌讳地坦白。
<br/>
<br/>        郑忠威会心一笑：“感情这种事情，跟一个人优秀还是不优秀，真的没有任何关系。”
<br/>
<br/>        “我有想过要放下，但是就是放不下。”落以柔苦笑，她是真的忘不了傅夜沉。
<br/>
<br/>        “如果真的放不下，那也要做最骄傲最理性的女人。”郑忠威接着说道。
<br/>
<br/>        落以柔微微偏头，疑惑不解地看着郑忠威。
<br/>
<br/>        郑忠威语重心长地回答道：“你可以在心底爱着大少爷，但是不要去破坏大少爷的家庭，不要趁人之危地去向大少爷告白，不要去争取不属于自己的东西，不要试图融入大少爷的婚姻中。你从第一天起，是大少爷的得力助手，这一辈子，就只能是大少爷的得力助手。不要试图去越界，不要对大少爷有任何暧昧的举动，你做好自己的本分即可。你可以爱大少爷，但也可以爱得骄傲，爱得理智，爱得高尚。你觉得我说得对不对？当然，于我来说，作为你的师父。我更希望你另觅良人，嫁个如意郎君，而不是一直吊在大少爷这棵树上。”
<br/>
<br/>        “我明白了，师父。”落以柔释然地微微一笑。
<br/>
<br/>        其实，傅夜沉在察觉到落以柔对他有“异心”之后，本想将她调走的。
<br/>
<br/>        是他郑忠威在背后替她落以柔说了不少的好话。
<br/>
<br/>        他觉得落以柔心思不坏，只要稍稍教导，她不会走上不归之路。
<br/>
<br/>        童筱颖来了后，傅夜沉的每日行程里，得加上“陪老婆时间”这条。
<br/>
<br/>        时间是傅夜沉自己定的，落以柔得将其他的工作行程，为这条时间挪位调整。
<br/>
<br/>        以前她还曾抱怨过，但习惯了以后，她连抱怨的心思都没有了。
<br/>
<br/>        这半个月的时间里，傅夜沉趁机把工作和蜜月给一起渡了。
<br/>
<br/>        落以柔对这个男人宠老婆的程度，真的是佩服得五体投地。
<br/>
<br/>        他确实把忙工作与陪老婆的时间，处理得很好。
<br/>
<br/>        前提条件，那也得她童筱颖配合。
<br/>
<br/>        法国的十大旅游景点，傅夜沉带着童筱颖在这半个月的时间里都去了一遍，同时也把这景点里需要会见的贵客也都见了。
<br/>
<br/>        傅夜沉会见贵宾谈工作的时候，她童筱颖就在外先一个人闲逛，一边打发时间，一边默默地等待。
<br/>
<br/>        谈生意和谈恋爱两不误的男人，也就只有他傅夜沉能做到了。
<br/>
<br/>        落以柔很是羡慕童筱颖，但已经学会了不再去嫉妒。
<br/>
<br/>        她算是见证了他们两的大吵大闹吧！
<br/>
<br/>        好像不管童筱颖怎么闹，他傅夜沉依旧舍不得放弃她。
<br/>
<br/>        在这半个月的时间里，童筱颖从傅夜沉那儿学会了跳华尔兹，只是她学会了的结果，是让傅夜沉的脚趾肿了好几个。
<br/>
<br/>        到了晚上夜半无人私语时，她还得私底下悄悄地亲自给他那受了伤的脚趾抹药。
<br/>
<br/>        毕竟是他是大Boss了，在外她得给足他面子才是。
