　　“这是境王布袋，内有七百丈须弥空间。”
<br/>　　陈进把这境王布袋握在手心之中，一股灵力注入其中，顿时打开了一处空间，一眼望去，有着七百丈的区域。
<br/>　　七百丈的空间，已经是无比惊人了，在须弥空间之中，这境王布袋的品质都算得上是上等了。
<br/>　　毕竟达到千丈的须弥空间，哪怕是在一些品级更高的星域大陆之中，都难以寻找得到。
<br/>　　楚风眠手心一动，便是将这境王布袋拿在了手心之中，旋即放入到了空戒里。
<br/>　　这境王布袋想要盛下太古蜈蚣，还有些太过勉强，不过楚风眠心中却是有着另一个想法。
<br/>　　重炼境王布袋。
<br/>　　境王布袋，乃是须弥空间的灵器，蕴含的空间之道无比玄妙，除非是对空间之道造诣极高的人物，才可以炼制。
<br/>　　纵然是在天人境强者之中，能够炼制须弥空间的炼器大师，都是寥寥无几。
<br/>　　对常人来说，想要重炼这境王布袋，简直是在白日做梦一样的想法。
<br/>　　但是楚风眠现在身躯融入始祖天龙之爪的太初之气，得到的可不只是龙之本源那么简单。
<br/>　　始祖天龙，乃是天空的霸主。
<br/>　　空间之道的玄妙，也随着太初之气，融入到了楚风眠的身躯之中去。
<br/>　　不过这空间之道，并不完整，楚风眠想要自己去炼制须弥空间，还远远无法做到。
<br/>　　可只是重炼的话，以楚风眠现在的空间之道，应该足以将这境王布袋重炼，再一次扩大其中的空间。
<br/>　　只要是可以扩大到千丈的程度，楚风眠就可以用这境王布袋，带着太古蜈蚣直接通过星域传送阵前往沧海星域了。
<br/>　　日后不管是楚风眠想去哪里，都可以随身带着太古蜈蚣，轻松无比。
<br/>　　正是因为楚风眠决定要重炼境王布袋，如今才会选择从这陈家商会的手中，买下境王布袋来。
<br/>　　“对了，不知道陈家商会之中，有没有仙石？”
<br/>　　楚风眠突然想到了什么，开口道。
<br/>　　对现在的楚风眠来说，想要提升实力，最为需要之物，不是什么诸天奇物，什么灵器，而是仙石。
<br/>　　只要是有着大量的仙石，楚风眠就可以借助其中蕴含的仙力，一举踏入半步天人境。
<br/>　　楚风眠前去沧海星域这一趟，也是有着想要去其他星域，寻找仙石的想法。
<br/>　　“仙石？”
<br/>　　陈进眉头一皱道。
<br/>　　“我们陈家商会，很少可以收到仙石，之前得到的几枚，也都出售了，现在陈家商会内，只有一枚仙石，如果楚公子需要的话，我们陈家愿意送给你。”
<br/>　　陈进一边说着，一边身形一闪，似乎是去了其他地方，再一次回来，从他的手心之中，便是多了一枚青蓝色的石头。
<br/>　　正是仙石。
<br/>　　不过楚风眠感觉得到，陈进手中的这一枚仙石的品级，似乎是比起楚风眠从九华天武者手中得到的品级要低一些。
<br/>　　其中蕴含的仙力，也驳杂一些。
<br/>　　陈家商会，毕竟只是阎罗大陆的商会，属于三级大陆，而那些九华天武者，可是来自一级大陆，圣域九华天。
<br/>　　得到的仙石品质有些差距，倒是在楚风眠的意料之内。
<br/>　　不过看来这仙石，的确是无比珍贵，在这陈家商会之中，居然就只有一枚。
<br/>　　楚风眠要突破半步天人境，至少需要一千枚仙石，这样一枚一枚的积攒，只怕是楚风眠耗费数十年都积赞不齐。
<br/>　　“看来必须是要去一些更高级的大陆，一些二级大陆，甚至是圣域之中，才可能得到大量的灵石。”
<br/>　　阎罗大陆已经算是三级大陆之中贸易无比繁华的一座大陆了。
<br/>　　既然从这阎罗大陆之中楚风眠都难以找到仙石，那也就代表其他的三级大陆之中，楚风眠找到仙石的几率也是寥寥无几。
<br/>　　只有前往更高级的大陆，才有希望寻找仙石。
<br/>　　不过这一次，楚风眠要前往的沧海星域之中，便是有着二级大陆的存在。
<br/>　　倒时候楚风眠可以前去寻找仙石。
<br/>　　“这一枚仙石，什么价格？”
<br/>　　楚风眠看着陈进开口问道。
<br/>　　一枚仙石，也是仙石，楚风眠自然来者不拒，毕竟楚风眠的手中，也就只有九枚仙石。
<br/>　　“楚公子需要，拿去就是。”
<br/>　　陈进笑了笑开口道。
<br/>　　仙石的确罕见，可一枚仙石的价值，倒是没有太过惊人。
<br/>　　毕竟跟灵石一样，仙石消耗起来，一下子也是几百枚，上千枚。
<br/>　　一枚仙石，语气是卖给楚风眠，不如送给楚风眠，留个人情。
<br/>　　“那好。”
<br/>　　楚风眠也不客气，手心一爪，这一枚仙石也落入到了楚风眠的手心之中，被楚风眠放入到了空戒之中。
<br/>　　十枚仙石，都被楚风眠放入到了空戒的最深处，以防止被其他人窥探道。
<br/>　　境王布袋已经得到，楚风眠也该离开了。
<br/>　　“楚公子等等。”
<br/>　　不过就在楚风眠准备离开的时候，那陈进却是突然开口了。
<br/>　　“何事？”
<br/>　　楚风眠停下脚步，回过头问道。
<br/>　　“之前殿王城之外的太古蜈蚣，可是被楚公子降伏了？”
<br/>　　陈进目光闪烁，似乎是想到了什么，开口道。
<br/>　　“不错。”
<br/>　　楚风眠点了点头道。
<br/>　　“这太古蜈蚣，是我在碎星带之中偶遇的，我便是出手将他降伏了，作为我的坐骑，怎么，这太古蜈蚣，是你们陈家之物？”
<br/>　　这太古蜈蚣的脑袋之中，本来可是有着禁制的，那也就证明曾经有人降伏过这太古蜈蚣。
<br/>　　莫非太古蜈蚣，是被陈家降伏的？

<br/>　　“不，这太古蜈蚣，跟我们陈家没有关系，不过这太古蜈蚣，却是跟吴家有些关系，所以我这才想要提醒楚公子，小心吴家的人。”
<br/>　　吴家？
<br/>　　阎罗星域的四大家族，吴，严，慕，陈。
<br/>　　以吴家为首，陈家最次，所以这陈家之中，仅仅只有两位半步天人境强者。
<br/>　　比起大罗天星域的四大势力，都远远不如。
<br/>　　而吴家，可是阎罗星域的第一大家族，实力远超陈家。
<br/>　　没想到这太古蜈蚣，居然是属于吴家的。
<br/>　　“这太古蜈蚣，是我的坐骑，谁敢来动，谁就得死！”
<br/>　　楚风眠也听出了陈进话中的意思，冰冷的开口道。
