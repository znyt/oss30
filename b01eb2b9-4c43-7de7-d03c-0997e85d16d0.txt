　　沉默半晌，张无生用筷子点着他，说道：“你小子不地道，你就知道我会主动开口？”
<br/>　　林三生低下头，喃喃道：“我甚至希望你不说。”
<br/>　　张无生倒是怔了一下，看他眼神中一片赤诚，心中有些感动，喃喃道：“叶少阳有你这号朋友，是他一生之幸。”
<br/>　　“不敢，其实我也什么都没做，能做选择的……是您自己。”
<br/>　　张无生用筷子指着饭桌上的菜，说道：“我这不就是已经选了吗？”
<br/>　　林三生暗暗吸了口气，道：“之前我跟少阳说的那些，您都听到了？”
<br/>　　张无生微微颔首，表示默认。
<br/>　　“大家都知道，我是张道陵嫡传子孙，是道门中的皇亲贵胄……人们觉得我理应当上龙虎山掌教，却不知道我是怎么走到这一步的。林大帅，我跟你说个故事，希望你能对任何人保密，你能让我相信吗？”
<br/>　　林三生点头，“若是您需要，我可立誓。”
<br/>　　“那倒不必，都是陈芝麻烂谷子的事，估计你也没兴趣说给人听……我年少时候，曾经跟在星月奴身边修道过……”
<br/>　　此言一出，林三生和曲波都怔住了。这可是从来没听说过的啊，如果不是张无生自己说，那真是打死都不会相信。
<br/>　　“我是被她拐走的，在她身边三年，那一年我十五岁。原因……”张无生迟疑了一下，道，“罢了，我从头说起吧，我道渊师叔，曾与星月奴有过一段感情……”
<br/>　　什么！
<br/>　　林三生跟曲波听得下巴都快掉了。
<br/>　　道渊真人……跟星月奴？这……林三生感觉自己的三观要碎了。
<br/>　　“那、那个，师父，我还是出去吧，我不敢听了……”曲波按捺着强烈地好奇，战战兢兢地恳求着。
<br/>　　“你他妈就给我站着，一会还有你的事！”张无生呵斥了一声他才老实了。
<br/>　　“这段往事，因为涉及到长辈，我不便细说具体，当时的情况是，我师叔拒绝了她，一个人回到玲珑塔下，他立誓镇守宝塔，一生不出山，跟这段感情也有关系……”（详见番外《茅山捉鬼人前传：起源》，不要搜，还没写）
<br/>　　原来是这样……林三生心中也算解开了一个谜团：之前叶少阳穿越到民国，跟年轻时候的道渊真人相处过一段时间，那时候的道渊真人还是充满理想抱负的，后来他们回来之后也讨论过，道渊真人性格并不怪异，为什么后来甘愿一生待在玲珑塔下，过青灯枯卷隐世生活？
<br/>　　虽然他后来也解释过原因，但是最初的动机，其实并没有说的。
<br/>　　这在大伙心中，一直都是个谜，今天终于知道根源了，居然是……为情所困。
<br/>　　只是没想到，看上去如道学先生一般的道渊真人，居然跟星月奴有过感情……两人的身份差距，道渊这么做就算不是大逆不道，也很是惊世骇俗了。
<br/>　　“张掌教，星月奴有几百岁了吧，道渊祖师才……这年纪差距也太大了吧。”
<br/>　　张无生瞥了他一眼，“星月奴看上去很老吗？”
<br/>　　好吧……
<br/>　　林三生没话说，邪物的心性，的确跟年龄没关系，毕竟光是修成人形就需要很多年。
<br/>　　“我师叔，是喜欢星月奴的，最初不排斥两人在一起——道渊你们也熟的，他这人也一向不在乎礼法，他跟星月奴在一起了一段时间，让他们分开的原因，是星月奴的野心，星月奴不甘心只做轩辕山的长老，她要统治人间，这也是她的使命，我师叔这人，刚直不阿，自然不能容忍，于是劝她放弃这念头，为此，他愿意跟她远走高飞……
<br/>　　星月奴反倒劝他里应外合，利用龙虎山的影响力，一起拿下人间。我师叔于是跟她决裂，回到龙虎山，把自己关在玲珑塔下，那一年，我十二岁，每天去给他送饭，跟他后面学习道法。
<br/>　　一日，他正在教我道法，星月奴突然赶来，要师叔跟他走，两人话不投机，星月奴劝不动他，两人大打出手，星月奴恼怒之下，将我抓走了……”
<br/>　　说到这，张无生摇头苦笑。
<br/>　　“星月奴这么做，不是报复，他是要逼我师叔去轩辕山见她，但我师叔知道那是个坑，就没去……”
<br/>　　“这……他就放着你不管了？”
<br/>　　“他知道，星月奴必然不敢杀我，一直在想办法营救，只是一直没找到星月奴而已。”林三生接着说，“我有兄弟四人，我是老四，是家门这一代子弟中天赋最高的，因为事关我师叔的名声，我爷爷当时非常生气，也剥夺啊了师叔的掌门继承权……外面那些争夺掌门的传言，就是这么来的。
<br/>　　不过我爷爷也只是想惩戒他一下，师叔自己傲气，加上情场失意，一直蹲在玲珑塔里，是他自己的选择。当时我失踪这件事，只有龙虎山核心的几个人知道，自然不可能声张，否则道渊和星月奴的事一旦传出去，龙虎山的名声就完了。
<br/>　　那边，星月奴为了自己的名声，当然也不敢声张，为了掩人耳目，她一直把我带在身边，除了他那几个核心弟子和师兄弟外，没人知道我的存在。”
<br/>　　说到这里，他望着林三生，加强语气：“我排行老四，星月奴让他几个弟子叫我‘小四哥’，这么叫而不叫名字，也是为了掩人耳目，鲲鹏当时闭关，我没见过他，但只要他稍微打听，立刻就会知道，这个‘小四哥’是我！”
<br/>　　原来……一切竟是这样。
<br/>　　现在所有的谜团都解开了。林三生松了一口气的同时，内心也越来越沉重。
<br/>　　“林元帅，我真的很佩服你，凭那么点线索，你就能推测出是我……”张无生摇头感叹，“你将来不可限量，当然，你现在已经很厉害了……”
<br/>　　曲波在边上听他们对话，感觉跟听天书似的，一脸的茫然。
<br/>　　“我很好奇，后来你为什么又回了龙虎山？”
<br/>　　“星月奴等了三年，也不见我师叔来见她，心灰意冷，她也不能真把我杀了，于是就放了我……因为我嘴甜，人乖，她是挺喜欢我，她放我回去，也是试图让我当一个棋子……毕竟相处过三年，我对她也一直示好，这让她觉得，将来如果我当了掌门，对她统治人间肯定有所帮助……”
