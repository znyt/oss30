        &lt;h3&gt;第三卷第一百三十三章  宁雅的改变&lt;/h3&gt;
<br/>
<br/>        宁雅和陈子昂聊完天以后，心情好多了。
<br/>
<br/>        她本来就不是那种跌倒了就爬不起来的人，家里虽然说有一些重男轻女的思想，但是对她的生活并没有限制，即使她不工作也还是可以当一个寓公。
<br/>
<br/>        只是人活着总该有一些事情可做吧。
<br/>
<br/>        应聘到陈子昂的公司里她充实多了，慢慢的也从家族的阴影中走出来了，原本她怀着对凌辰美好的希望，希望看着凌辰成长起来，能牵着她的手陪着她走过今后的人生。
<br/>
<br/>        他们还真是她大学开玩笑时候说过的话呢：“俩个冰坨子是擦不出火花来的。”
<br/>
<br/>        等年后她就辞职自己干一份事业，至于凌辰，祝他幸福吧。
<br/>
<br/>        每次想到凌辰，她心里就难过，和凌辰的聊天记录她看了又看，总是舍不得删除。
<br/>
<br/>        她想约胡婷周末出来，但是胡婷要和她家狸猫腻歪，就把她这个死党闺蜜给抛下了。
<br/>
<br/>        “胡婷，你一定是假闺蜜。”宁雅打电话抱怨着。
<br/>
<br/>        “小雅，闺蜜重要，狸猫也重要，狸猫出差都一个多礼拜了，今天才回来，人家想香香嘛。”胡婷在电脑那头撒着娇。
<br/>
<br/>        “你个重，色，轻友的家伙，以后别拖着行李箱来我家，不收留。”宁雅笑着说。
<br/>
<br/>        “小雅，你听我说啊，我就这个周末陪狸猫，以后我都陪你啊。”胡婷在电话那头笑嘻嘻的说。
<br/>
<br/>        “别扎我心了，以后你要是都陪我的话，你家狸猫就会将我写成十足的渣女，我可不敢。”宁雅说。
<br/>
<br/>        “哼，他敢？他要是敢将你写成渣女，我就偷偷的混进他的书友群黑他，说他和女粉丝结婚。”胡婷在那边胡说八道的笑着。
<br/>
<br/>        “那你就成了公敌了。”宁雅被胡婷的笑感染了，有这样一个活宝闺蜜其实真的不错。
<br/>
<br/>        “那我就让他跪键盘，跪在键盘上码字，哈哈哈。”胡婷似乎看到了她家的狸猫跪在键盘上码字的情景，开心的大笑。
<br/>
<br/>        “你让他跪啥都行，只要你舍得，就怕到时候被狸猫一哄，就忘了我是谁了，说不定狸猫要将我卖了，你还帮着数钱呢。”宁雅边打电话边挑选东西。
<br/>
<br/>        “放心，卖了分钱给你，哈哈哈。好了，我要去机场接狸猫了。你先考察市场，下周我就陪你。”胡婷虽然说是要去接人，但是还是不挂电话。
<br/>
<br/>        “对了，凌辰最近还打扰你了吗？”胡婷听宁雅提起行李箱的事情就想到了她接凌辰电话，她嗲着声音说完那句话，对面就没有了声音。
<br/>
<br/>        后来她也就没有给宁雅说这件事情，也不知道俩个人最近怎么样了。
<br/>
<br/>        “没有了。”凌辰算不算打扰她？他都有女朋友了，还经常给她发那么暖昧的情话也算是打扰她吧。
<br/>
<br/>        “嘿，男人。碰到一点点事情就放弃了，不值得。”胡婷自言自语。
<br/>
<br/>        “你说什么呢？”宁雅不知道胡婷指的啥。
<br/>
<br/>        “没什么，我说凌辰，撩了你这么多年，碰到一点点事情就放弃了。”胡婷叹了口气，其实，那天她就是恶作剧。
<br/>
<br/>        想到凌辰和他家狸猫拍的照片，她就故意的那样接了电话，想气一气凌辰，本来他们都是同学关系。
<br/>
<br/>        “他有女朋友了。”宁雅低着嗓子说。
<br/>
<br/>        “你说什么？他有女朋友还和你谈情说爱？渣男！”胡婷在电话那头生气的骂。
<br/>
<br/>        “不说了啊，我周末去考察市场。”宁雅不愿意再提起凌辰。
<br/>
<br/>        “真的要自己干了啊？”胡婷问宁雅。
<br/>
<br/>        “不开玩笑的，年内我先考察好，年后就准备行动。”宁雅那天和陈子昂聊了一下，其实陈子昂也觉得开个美容医院很不错的。
<br/>
<br/>        现在的女孩子们都很爱美，做个小小的微创就能变得漂亮一些，小到纹眉、漂唇、种睫毛、双眼皮，拉皮，大到整形，反正都是很流行的，也是很赚钱的。
<br/>
<br/>        陈子昂其实很鼓励她创业的，并且要让她一定要考察好项目，让她感到暖心了。
<br/>
<br/>        温暖的人走到哪里都会自带热量，给周围人以温暖。
<br/>
<br/>        她决定安下心来在陈子昂的公司好好干到耗材推上市场的那一天，利用周末的时间考察市场。
<br/>
<br/>        宁雅也开始忙碌起来了，周内和陈子昂一样忙得连轴转，周末到处跑，有时候周五买了机票周天晚上回来，周一接着上班。
<br/>
<br/>        时间是治愈心灵创伤的良药，充实的工作更是治愈心灵创伤的良药。
<br/>
<br/>        宁雅没有时间去考虑她和凌辰之间的事情了。
<br/>
<br/>        每天满脑都是工作规划，会经常在梦里梦见凌辰那些甜蜜的情话，醒来两行泪，到了天明又是一个风风火火的人。
<br/>
<br/>        为了控制着不去打扰凌辰，摔碎了两部手机，但是还是将之前的手机修好，舍不得删掉和凌辰的聊天记录。
<br/>
<br/>        听胡婷提起凌辰，宁雅就想，这么多年的问候，这么多年夜深人静的聊天，难道都是假的?
<br/>
<br/>        想到这里，她突然就卡了，胡婷在电话里面说什么她一个字也没有听见。
<br/>
<br/>        她打开原来的那个手机，翻开和凌辰的聊天记录；
<br/>
<br/>        雅的凌：“你是我的心，你是我的眼，你是我动力的源泉。”
<br/>
<br/>        雅的凌：“风景在美，不及你的美。”
<br/>
<br/>        雅的凌：“那说不出口的爱，像清冽的甘泉，滋润着心田，陪着你度过每一天。”
<br/>
<br/>        雅的凌：“曾经沧海，曾经桑田，一直你都在我心尖。”
<br/>
<br/>        雅的凌：“想牵着你的手路过山，淌过水，追着白云，看夕阳下的美。”
<br/>
<br/>        呵，果然，胡婷说的对，会说情话的男人都是骗子。这还没有牵手呢，和别人去看夕阳下的美去了。
<br/>
<br/>        宁雅突然很好奇，凌辰的女朋友会是什么样子的。
<br/>
<br/>        说不定以后还能见到呢，凌辰也会说哪样的情话给她吗？
<br/>
<br/>        他们也天南地北的聊天吗？
<br/>
<br/>        站在商场中心，手中提的东西不知不觉掉到了地上，直到有人为她捡起来：“你的东西掉了。”
<br/>
<br/>        宁雅才恍惚的接过手袋，苦涩的笑着道了声：“谢谢！”
<br/>
<br/>        原来认为时光漫长，现在才知道没有凌辰的陪伴才是真正的时光漫长。
<br/>
<br/>        不管怎么漫长日子还得继续。
<br/>
<br/>        ......
<br/>
<br/>        宁雅放不下她骄傲的自尊，不想去打扰凌辰，却怎么也没有想到，她的爱情还有刘诚之在为她守卫。
