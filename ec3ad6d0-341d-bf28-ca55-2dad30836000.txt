　　智天使的话顿时又是在收看视频的人中引起骚动，显然在他手里还掌握着更加劲爆的爆料。
<br/>　　指挥中心，龙隐部队众人也是在猜测着智天使手里还掌握着什么东西，李然略一思考，眼神为之一凝，沉声说道：“完了，K国要乱了！”
<br/>　　李然的话也是提醒了罗昊，罗昊诧异地转头看向李然，问道：“你指的不会是那个吧？”
<br/>　　“很有可能就是那个。”李然点头回答道。
<br/>　　“如果真的是这样，那K国可能真的要乱了，K国政府将会面临近几年来最大的信任危机。”罗昊神情严肃的说道。
<br/>　　“喂喂喂，你们两个聪明人别在那里打哑谜行不行。”慕容飞在一旁嚷嚷道：“有什么东西就不能说明白点，这么打哑谜很有意思吗？”
<br/>　　“你就安心往下看吧。”
<br/>　　云熙把一个纸团扔在慕容飞脑袋上，让慕容飞闭嘴。
<br/>　　又是一段偷拍的视频，视频拍摄地点是一个机场，机场上停着一架商务机，商务机前有着几个人在交谈着什么。这段视频很明显是在很远距离，用手机偷拍下来的，视频算不上清晰，但还是可以看清楚视频中的人物，不过能听到的声音却只是周围风吹过的“呼呼”声。
<br/>　　出现在视频中的关键人物有三个，K国国防部长，在法庭上被宣判5年监禁的现场监控小组组长，还有被宣判10年监禁的狙击观察手！
<br/>　　“现在大家都明白了吧？”隐藏在阴影中的智天使又一次出现在视频中，说道：“原本应该被关押在监狱中的罪犯，却出现在机场上，这说明什么难道还不清楚吗？如果你们不清楚，我可以告诉你们，你们被骗了，被人耍了！”
<br/>　　“这些该死的混蛋！”
<br/>　　马路上，一个年轻男子手中握着手机大喊道：“我们不需要这种用谎言欺骗我们的政府！”
<br/>　　年轻男子的话立刻就得到了周围人的响应，马路上的行人纷纷爆发出要求K国政府对事情给出合理解释的呐喊声。而马路上还有着一些人在默默擦泪，他们在地铁爆炸案中失去了亲人，原本K国政府直播的庭审至少让他们得到了一点点的安危，而现在视频中曝光出来的东西，确实让他们觉得自己就像一只猴子一样被人耍了，被耍的晕头转向。
<br/>　　智天使录制的视频播放到此处就结束了，但是事情却并没有结束，原本录制视频播放突然变成了直播，还是跟之前视频中一样的昏暗房间，还是跟前面一样经过技术处理的低沉声音，还是一样隐藏在黑暗中让人看不清他的真实面目。
<br/>　　直播视频中，智天使拿起放在桌子上的手机，点开一个视频文件，放到摄像头前，说道：“就在几分钟前，我收到了这样一份视频文件。既然K国政府刻意包庇，让原本应该被关在监狱中的人坐着飞机离开K国，那就让我来替你们这两个逃跑的罪犯！”
<br/>　　视频中，监视小组组长和狙击观察员被反绑在椅子上，他们身上伤痕累累，很显然已经是遭受到了折磨。
<br/>　　一个穿着迷彩服，用一块阿拉伯方巾蒙着口鼻的雇佣兵握着手枪走到监视小组组长和狙击观察手面前，抬起他们的下巴问道：“名字！”
<br/>　　两个人没有回答。
<br/>　　雇佣兵用手枪对着两个人脑袋都是来了一下，鲜血很快就涌了出来，染红了他们的脸颊。雇佣兵再次问道：“告诉我，你们的名字！”
<br/>　　狙击观察手和监视小组组长分别说了自己的名字，雇佣兵点了点头退后两步，拉开手中手枪套筒把子弹上膛，对准了监视小组组长，扣动扳机。
<br/>　　嘡！
<br/>　　监视小组组长眉心中弹，混着骨头碎屑的血浆从他脑后喷出，溅落在他身后的墙壁上，子弹的侵彻力令得监视小组组长的脑袋猛地向后仰去，但很快又耷拉下来，血浆从他眉心弹孔中“滴答滴答”的滴落下来，滴落在他身上。
<br/>　　被绑在监视小组组长旁边椅子上的狙击观察手看到发生在自己身边这一幕，奋力的挣扎起来，尽管他们被抓到之后就饱受折磨，可是面对死亡，他还是激发出了他的潜能。
<br/>　　只不过他的潜能并没有能起到任何作用，枪声再次响起，狙击观察手饮弹而亡。
<br/>　　这样的一幕充满了血腥和暴力元素，常人看到会发出惊叫声很正常，但是马路上，看完这段视频的行人对此并没有什么反感，相反，他们还认为这样做才是正确的，这样做才是“伸张正义”！
<br/>　　马路上，声讨K国政府欺骗行为的声音再次响起，声势比之前还要壮大，从外面的那些声音中，罗昊能听出K国民众内心的愤怒。
<br/>　　“K国政府麻烦了！”
<br/>　　李然从椅子中站起身来，走到桌子前说道：“如果说之前的新闻发布会和庭审直播让他们站在了正义面，那么现在智天使已经让他们的优势丧失殆尽，他们不再是正以的。在K国民众心中，他们就是骗子，他们就是混蛋，无耻之徒！相反，地铁爆炸案的制造者智天使却成了正以的化身，因为他帮助那些受欺骗的K国民众惩处了罪犯。”
<br/>　　“斯德哥尔摩症候群？”罗昊看着李然问道。
<br/>　　“虽然还达不到斯德哥尔摩症候群这么严重，但至少已经对K国民众产生了实质的影响。”李然两根手指夹着铅笔转动着，说道：“如果现在跑去街上随便找看过刚才视频的人来询问，他们中大部分人对K国政府的憎恨要超过对智天使的憎恨。”
<br/>　　“智天使这么做是为了什么？”李毅峰问道。
<br/>　　“混乱。”李然停下手中转动铅笔的动作，说道：“为了混乱。K国政府召开的新闻发布会对智天使他们造成了不利的影响，所以他们要把水搅浑，只要K国越混乱，越方便他们撤离出K国，现在那些受到视频蛊惑的K国民众成了智天使的‘帮手’，这样，20个小时后，不管子午教官跟SHADOW对决的结果会是怎么样，对于智天使来说都是有利局面！”
