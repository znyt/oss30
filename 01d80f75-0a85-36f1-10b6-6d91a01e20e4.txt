　　第34章
<br/>　　过了正午，天气依旧炎热，日头落在身上像是火烧。
<br/>　　杂草丛生的山道，只留出一条窄窄的小径，天气潮湿、闷热，行在这样的路上，简直就是一种折磨，汗水如同拧毛巾一样，一层一层的往外冒。
<br/>　　可偏偏有两个人行在这样的路上，都是书生打扮，白袍束发。走在前面那人剑眉星目，身材修长。长袍挽在腰上，头发黏在脸上，皮肤被晒得有些枯黄。
<br/>　　“哥哥，你等下我，别走那么快。”身后跟着那人却单薄很多，像一个小孩穿着不属于他的衣服：“好端端不待在幽州城，偏偏往这里来，害的我和你一起受罪。”
<br/>　　“不行千里路，怎知天下事，只待在幽州城，哪里见得到这大好河山。”哥哥扭过头来，咧嘴一笑，笑容干净漂亮。
<br/>　　“这穷山恶水有什么好看的，我听说这里有山贼，你别把山贼招来。”弟弟小跑着前进，口中埋怨道。
<br/>　　“山贼有什么好怕的，若是敢来，凭我一人一剑，自可斩开一条道路。”哥哥一拍腰间佩剑，叮当作响。
<br/>　　“呔！”一声大吼，程大雷从林子里蹦出来，大喊道：“打劫啦，打劫啦，男的站左边，女的站右边，你……说你呢，站中间。”
<br/>　　弟弟脸上一红，忙站在哥哥身后。哥哥扫视程大雷一眼，锵啷拔出佩剑。
<br/>　　“有我在，不用怕。”
<br/>　　剑眉星目，丰神俊朗，他扫了扫程大雷手中的斧头，再看了看自己的佩剑，忽然一举手，做出投降姿态。
<br/>　　“我们投降！”
<br/>　　“咦！”
<br/>　　“咦！”
<br/>　　程大雷和那人的弟弟同时睁大眼睛。哥哥忽然一扯弟弟头上的帽子，道：“这是我妹妹，生得花容月貌，实在不行，你留下她，放过我。”
<br/>　　长发垂落，衬出一张吹弹可破的精致脸蛋，男扮女装更显几分特殊的妩媚。
<br/>　　“哥！”妹妹满脸不快，在他身上踹了一脚：“你怎么可以出卖我。”
<br/>　　“妹妹，我还有远大前程，你就牺牲一下，我会怀念你的。”
<br/>　　“谁让你怀念，你就是个混蛋。”
<br/>　　“停！”
<br/>　　程大雷及时制止这对兄妹，道：“叫什么名字？”
<br/>　　“我叫李婉儿，他叫李行哉。”这次是妹妹出卖哥哥。
<br/>　　“干嘛的？”程大雷问。
<br/>　　“诗人！”李行哉昂起头道。
<br/>　　“诗人？”程大雷诧异问。
<br/>　　“正是，我的梦想就是踏遍神州，写下名垂千古的诗篇。”李行哉一本正经。
<br/>　　“那么你一定去过很多地方？”程大雷想了想，道：“很好，恭喜你们，你们被绑架了。”
<br/>　　二人被程大雷赶着上山，李婉儿小声道：“哥哥，咱们真的要跟他上山么？”
<br/>　　“快走，啰嗦什么！”程大雷催促道。
<br/>　　最后程大雷押着二人来到蛤蟆寨，在经过校场，看到蛤蟆寨的大旗时，李婉儿怔了怔。
<br/>　　“这里，难道就是传说中的蛤蟆寨？”
<br/>　　“咦，传说这两个字从何说起？”程大雷颇有些莫名其妙。
<br/>　　李行哉望向程大雷：“难道你就是那个以五人之力，硬抗一千精兵，杀得黑石城丢盔弃甲的蛤蟆大王。”
<br/>　　“咦，难道我现在名头这么大么？”程大雷挠挠头。
<br/>　　“传说中还说你每顿饭要吃人肉，喝人血……你们抓我们上山，该不会想吃了我们吧？”李婉儿恐惧道。
<br/>　　“你看我像吃人肉的吗？”程大雷无奈道。
<br/>　　“像。”李婉儿猛地点点头。
<br/>　　程大雷扑街。
<br/>　　“好啦，我给你们交待一下蛤蟆寨的规矩。”程大雷收拾一下受伤的情绪，道：“你们就留在这里，我也不会打你们，也不会关你们，想走的话随意，不过这里方圆五十里都是山贼，如果你们自信可以平安走出青牛山，自可离开。”
<br/>　　“那你留我们在这里做什么？难道真的想……”李婉儿下意识抱紧胸脯。
<br/>　　“咳。”程大雷轻咳一声，恋恋不舍的收回在李婉儿胸脯的视线，看着李行哉道：“你不是诗人嘛，把你的见闻都写下来，什么时候写完了，我派人护送你们离开青牛山。”
<br/>　　“徐军师！”程大雷喊了一嗓子：“给他们准备笔墨，安排房间。”
<br/>　　李行哉和李婉儿被送进一件屋子，等屋子只剩下两人的时候，李婉儿不满道：“哥哥，你为什么要被故意留在这里，你的剑术连老师都称赞……”
<br/>　　“哈。”李行哉倒在土炕上，对于这里简陋的环境，他倒是很适应：“你不觉得这个山寨很有意思么？”
<br/>　　“哪里有意思了，我怎么没看出来？”
<br/>　　“打劫见闻的山贼我可是第一次见到啊，还有这座山寨总觉得有点别的东西。”李行哉摊开面前的宣纸：“来，磨墨，既然是诗人，就让我先赋诗一首。”
<br/>　　……
<br/>　　姓名：程大雷（小有名气的普通山贼）
<br/>　　年龄：18
<br/>　　技能：三板斧
<br/>　　隐藏属性：一个好人
<br/>　　李行哉兄妹刚走，程大雷就开始检查自己的信息。一直忙着其他事，倒很久没有看过自己的信息，却原来不知不觉间，自己的声望已经升级了。这应该是防御黑石城进攻后，消息渐渐传开，如此收割的名望。
<br/>　　以前是初出茅庐，如今是小有名气，也不知道声望如果继续升级，会不会有其他用途，至少目前看来，没个鸟用。
<br/>　　“大当家。”黄三元见到程大雷，打了个招呼。
<br/>　　程大雷想起一件事，问道：“你们把辣椒送到落叶城，反响怎么样？”
<br/>　　“辣椒我都交给苏家小姐，反响如何恐怕要过段日子才知道。”黄三元。
<br/>　　“那么你觉得最近用辣椒炒的菜味道怎么样？”
<br/>　　“一开始觉得并不适应，但隔几天不吃，还蛮想吃的。”黄三元。
<br/>　　看到黄三元这幅表情，程大雷心中就有了底气。看来辣椒还得好好种，山寨日后发展可全靠它了。
<br/>　　“其他山寨忙什么呢。”程大雷一边向山寨的农田走，一边问道：“都在打那批寿礼的主意，忙着打劫？”
<br/>　　“现在整个青牛山都在盯着这件事，只要那批寿礼从青牛山过，必然有人动手。”黄三元问：“大当家，咱们真的不动手？”
<br/>
