<br/>				当府里的人看到灰狼顶着一张红肿的嘴回来时，一个个都是一脸的诧异，好奇的打量着他，罗宇问：“灰狼，你这是怎么了？”<br/>　　<br/>　　“没什么，多喝了些汤，就成这样了。”灰狼说着，连忙闪身回了房。<br/>　　<br/>　　见此，众人笑了笑，倒也没再多问，却不知，他在接下来的几天，天天往茅房里跑……<br/>　　<br/>　　而在后院中，凤九去了她娘亲那里，将今天买的那些东西都跟她说了一下安排，最后跟家人闲坐了一会后，便回房中去了。<br/>　　<br/>　　沐浴过后倒在床上，听着脚步声进来，她看了一眼，见是轩辕墨泽，便道：“我爷爷肯放你回来了？还以为你会跟他下一整夜的棋呢！”<br/>　　<br/>　　轩辕墨泽唇角微勾，一边脱去外衣，来到桌边倒了杯水喝着，道：“连下三盘三盘全输，你觉得老爷子会留我下一整夜吗？”<br/>　　<br/>　　“今晚我娘跟我说，我哥去下聘的日子挑好了，说是下聘后三天就上门迎亲，这速度估计也是没谁了。”说到这，她笑了起来：“今晚看我哥那乐呵样，看来也是挺开心的。”<br/>　　<br/>　　“成亲是大喜之事，自然开心。”轩辕墨泽喝了杯水后，便来到床边坐下，看着躺在床上的她，道：“等他成亲后，我们也该离开了，到时，我们顺便把凤夜他们送回去吧！”<br/>　　<br/>　　“好。”她点了点头应了一声，往里面躺了一些，给他腾出位置来。<br/>　　<br/>　　“我还没沐浴，你先睡吧！”他站了起来，并没有躺下，而是让下人准备了沐浴的水，先去泡了个澡。<br/>　　<br/>　　见此，凤九打了个哈欠，翻了个身抱着被子便沉沉睡去。<br/>　　<br/>　　接下来的日子，凤府里着手全力安排关习凛下聘成亲一事，一个个都忙碌了起来，城中的一些世家也都知道了关习凛要成亲了，于是，也都纷纷准备了贺礼，准备到时贺喜。<br/>　　<br/>　　因两地相隔太远的原因，叶家的人也是在凤萧陪同关习凛去下聘后，来到了桃花坞这里。这是叶菁和关习凛两人商量好的，打算在桃花坞这里出嫁。<br/>　　<br/>　　成亲这一天，与凤府比邻的关家，门庭张灯红彩，一片喜气洋洋。而关家的祖宅那边，却是在听到消息后一个个坐在府中唉声叹气。<br/>　　<br/>　　一步错步步错，早知今日，当初就不应该那样对待关习凛，要不然，那样出色的一个儿郎，可就是他们关家的骄傲，可如今，纵是他再出色不凡，就算他依旧姓关，却也与他们关家无关。<br/>　　<br/>　　关习凛和叶菁成亲的这一天，请来的都只是一些相熟的亲人朋友，倒也没多少外人，哪怕有不少人千里而来道贺，却也无法进入关家大门。<br/>　　<br/>　　毕竟，他们也不是寻常人，他们也要预防有人趁机作乱，因此每个进关家的人都必须是知根知底的。<br/>　　<br/>　　这一天，凤卫的人也在关府里外把守着，就连罗宇等人也在喝了杯喜酒后，在府中府外注意着，也正是因此，这一天并没有发生什么事，而是欢欢喜喜的完成了关习凛和叶菁的婚事……<br/>　　<br/>　　洞房花烛夜，关习凛微晃着脚步被扶进了房。<br/>　　<br/>　　：。：<br/>
