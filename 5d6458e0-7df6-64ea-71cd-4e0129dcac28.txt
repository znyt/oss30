　　第60章
<br/>　　第二天辰时，也就是上午七点，蛤蟆寨的大门缓缓打开。
<br/>　　门口倒着一大片人，这些都是受饥饿与伤病折磨后，内心动摇，选择加入蛤蟆寨的人。他们在山寨门口求了一夜，守了一夜，早已有气无力，此时见大门终于打开，都投过去期待和哀求夹杂的目光。
<br/>　　程大雷面对这一幕，没有太多表示。一张桌子摆在门前，程大雷、徐神机、黄三元坐在桌子后。
<br/>　　“我们要加入蛤蟆寨。”
<br/>　　“求求你收留我们吧。”
<br/>　　“肃静，肃静。”徐神机维持着秩序：“想要加入蛤蟆寨，必须经过面试，达到条件才可以。”
<br/>　　“我想要加入蛤蟆寨。”那个小腿烂了一块的汉子走到桌前。
<br/>　　“姓名？”黄三元提着笔问。
<br/>　　“孙九。”
<br/>　　“以前是做什么的？”
<br/>　　“是给老爷做饭的。”
<br/>　　“腿上有伤？”
<br/>　　“被车轴砸住，不碍事，俺有膀子力气，什么活都能干。”孙九努力站直身子。
<br/>　　“你这个情况，很难办啊。”黄三元叹了一声，以目光询问程大雷。
<br/>　　孙九的心紧张得跳动起来，他哀求道：“求求您，收留我吧，我能吃苦，肯干活，一条腿瘸了，不算什么的。”
<br/>　　程大雷查看孙九的信息，惊喜的发现孙九竟然是一个等级为优秀的厨师，这实在出乎程大雷的预料。
<br/>　　“暂且留下，等级定位丙等。”
<br/>　　“谢谢大当家，谢谢黄总管。”孙九喜出望外，似乎完全忘了自己昨天是主动离开的。同时他又有些奇怪：“什么是丙等？”
<br/>　　“这个你以后就知道了，先进去让人给你看看伤，记住，你现在是实习期，表现不好是会随时被赶出去的。”
<br/>　　实习期这个词，孙九又听不懂，不过他明白自己是被留下了。同时心情又有些忐忑，生怕随时会被赶出去。
<br/>　　“下一个。”黄三元挥挥手。
<br/>　　有人第一个吃螃蟹，接下来的事情就顺利许多。程大雷定下甲乙丙丁四等，想要加入蛤蟆寨必须经过苛刻，甚至刁难的面试，最后方才会被录用，而一些人则是直接淘汰。
<br/>　　面对这样的情景，人心惶惶，生怕自己会被淘汰，落得无家可归。有些人不仅去想，昨天坚持要离开，如今又哭着喊着要加入，究竟图得是什么。有些人被淘汰后，则是哭天喊地的哀求，可程大雷展现了他从所未有的铁石心肠。
<br/>　　程大雷就是要通过刁难，剥夺他们的尊严。令他们明白，蛤蟆寨不是求着他们加入，而是反过来，加入蛤蟆寨就是一份荣誉，一份认可。人们对于自己轻易得到的东西向来不会珍惜，对费尽心血获得的东西才能视作珍宝，哪怕这珍宝只是一根稻草。
<br/>　　而加入蛤蟆寨的人，也获得了他们开来极好的待遇，有伤治伤，有病看病，山寨目前还没有医生，只有略懂些草药学的秦蛮暂时充当这个职位。不过山寨拥有『治愈』技能，这些人的伤病则都算不得太大的问题。
<br/>　　至少他们每人都能吃到一碗热腾腾的饱饭，山寨内飘出来的饭香就令山下饥肠辘辘的人望眼欲穿。
<br/>　　而在忙碌这些事情时，山下还发生了另外一件事。
<br/>　　陆哼带着一小队人，灰头土脸的从落叶城方向过来。当看到这里热闹的情景时，他下意识吓了一跳，这些人都在干什么。
<br/>　　“陆将，陆将！”
<br/>　　忽然有人高声喊他，陆哼抬起头，看到程大雷正兴奋地向他招手。
<br/>　　陆哼不是很想看见程大雷，进攻蛤蟆岭失败，让自己沦为笑柄。人们都说程大雷只用一招，就将自己斩落马下。
<br/>　　而事实上，是自己先在秦蛮手里受了伤，否则凭程大雷有什么资格和自己动手。
<br/>　　当自己耐着性子向他们解释：程大雷其实不行的……
<br/>　　然后他们就会说：呦呦，你让人家一招击败，难道人家不行，你行？你行你让人家一招击败。
<br/>　　说着话是一定要配上夸张的语气，挤眉弄眼的笑容。
<br/>　　从落叶城到黑石城，蛤蟆岭是必经之路，陆哼不想碰见程大雷，却又不得不碰见他。
<br/>　　“陆将，你们不是去给幽州王送寿礼么，怎么才回来？”程大雷小跑过来，语气竟然很亲切。
<br/>　　“嗯呐，办完事就回来了。”陆哼随意应付一句，他不想和程大雷动手，动手也打不过，秦蛮还在山上杵着呢。
<br/>　　“看情况，不是很顺利？”
<br/>　　“怎么不顺利，顺利的很好不好！”陆哼像被踩住脚一样大叫起来。而事实上，他这次去幽州城何止是不顺利，简直糟糕透顶。
<br/>　　幽州王杨龙停是这片土地实际的统治者，所以薛半川在篡位后急切的需要得到杨龙停支持，可有件事薛半川并不知道。
<br/>　　三十年前戎族入侵，韩虎踞死守孤城，为后方援军争取时间，如此方才击退戎族。韩虎踞靠此战扬名，成为黑石城城主。
<br/>　　这些事大家都知道，但大家不知道的是，当初率领那支援军的人正是如今的幽州王杨龙停。没有韩虎踞的死守，杨龙停不可能主持起力量击退戎族，他也不可能成为后来的幽州王。
<br/>　　二人同时从军，在一个小队中摸爬滚打三年，以兄弟相称。对于这些事，薛半川跟在韩虎踞身边这么多年，竟然毫不知情。
<br/>　　本来杨龙停还不知道韩虎踞死了，幽州城距离黑石城很远，消息传播不便。也就是陆哼这次送寿礼，杨龙停才知道消息。
<br/>　　面对韩虎踞的死，杨龙停只说了一个字：死。
<br/>　　他扣下寿礼，杀了几个陆哼的随从洗刀，并放陆哼几个回去，通知薛半川最近这些日子好好活着，他会发兵攻打黑石城，亲自在亡兄的坟前浇奠，用薛半川的血。
<br/>　　这些事，陆哼没有告诉程大雷，免得他幸灾乐祸。他试着转移话题，问：“这么多人在做什么？”
<br/>　　“喔，我们蛤蟆寨正在招人，待遇优厚，五险一金。”程大雷兴奋的挥挥手：“陆将有没有兴趣加入？”
<br/>　　陆哼激灵一下，以见鬼的目光望着程大雷，心想你有没有搞错，我们是仇人，我无时无刻不想杀死你，你竟然要我随你当山贼。
<br/>　　面对陆哼的表情，程大雷无奈的叹口气，冲陆哼伸出手：“那好吧，拿来。”
<br/>　　“拿来什么？”陆哼不解的问。
<br/>　　“过路费，两成。”
<br/>　　
<br/>(本章完)
