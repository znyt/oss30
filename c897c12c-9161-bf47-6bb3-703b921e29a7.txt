        办公桌后是整整的满墙书柜，书柜有暗梯，梯板上的按摩图案还是昼景给她的创意。
<br/>
<br/>        三年的时光，让他傅夜沉在那满墙书柜上整整齐齐地摆满了与商业、金融等有关各种书籍。
<br/>
<br/>        另一边，巨大的单面透明落地窗，干净明亮，充分地让阳光洒满整间办公室。
<br/>
<br/>        童筱颖看向宽大的办公桌后，此刻正专注办公的傅夜沉。
<br/>
<br/>        只见他正一边接听座机电话，一边飞快地签署着一叠文件，压根就没发现已经走进来的她。
<br/>
<br/>        童筱颖自顾自地在一旁的沙发上坐下，不动声色，悄悄地打量着正伏案认真办公的傅夜沉。
<br/>
<br/>        他脸上的线条轮廓分明，五官像中西混血的那种深邃与立体，眉宇间清朗俊秀，棱角分明的衬衫扣至最上头那粒，领带也打得工整，整个人干净清爽，透着十足的商务型男人的气息。
<br/>
<br/>        傅夜沉确实是一个很帅、很英俊、很有型的男人，只不过，她都认识他三年了，他居然一直没有入过她的心。
<br/>
<br/>        如果不是他强娶了她，或许，她这一辈子，她都不会再跟他有什么交集吧！
<br/>
<br/>        童筱颖的心思百转千回，静静地坐在沙发上，打量着自己的这个二婚老公。
<br/>
<br/>        直到过了整整一个钟头，傅夜沉才结束掉自己手头上的工作，且在无意间地抬头，才看到她。
<br/>
<br/>        “你什么时候到的？”他那张极俊的帅气面孔上，原本工作时淡然锐利的目光，转瞬间略透着一丝温柔与宠溺，唇瓣勾起一抹浅浅的弧度。
<br/>
<br/>        “刚到。”童筱颖悠闲自得地从舒适的皮沙发上站起身，顿了顿后，又补充了一句，“我是逛了一会儿街，才过来的。”
<br/>
<br/>        其实，她一直坐在这里等他，他竟然不知道！
<br/>
<br/>        想必，他工作时，是有多认真了。
<br/>
<br/>        “那我们走吧！”傅夜沉起身拿起一旁挂在落地衣架上的西装穿上，随手扣上笔挺西装的最下端一粒扣子，动作潇洒又不失优雅。
<br/>
<br/>        “好啊！”童筱颖微微一笑。
<br/>
<br/>        他步伐轻盈地走到她跟前，向她优雅地伸出了手。
<br/>
<br/>        童筱颖怔愣了一下，不明所以然，还以为他是向她要车钥匙，于是举起手提包开始埋头找车钥匙。
<br/>
<br/>        傅夜沉忍俊不禁地挑眉，大手直接包住她那只正在手提包里翻找东西的小手，嘴角微扬地牵着她往门外走去。
<br/>
<br/>        他们所经之处，员工纷纷驻足颔首，“总裁、总裁夫人，好！”的问候声一声接着一声响起。
<br/>
<br/>        她在一群女员工们的艳羡中，被他独一无二地牵着。
<br/>
<br/>        童筱颖一边走着，一边偷偷地瞄着身边这个帅得掉渣的男人。
<br/>
<br/>        十分怀疑他的动机！
<br/>
<br/>        他牵着她“招摇过市”，是想干什么？
<br/>
<br/>        “我们俩这样，像不像在谈恋爱？”傅夜沉紧紧地牵着童筱颖的手，一刻都不曾松懈，黑眸意味深长地睐着她。
<br/>
<br/>        童筱颖怔愣：“谈恋爱？！”
<br/>
<br/>        “追求、谈恋爱、结婚。”傅夜沉剑眉微挑，一本正经地说。
<br/>
<br/>        童筱颖很没情趣地泼冷水：“你是反着来的。”
<br/>
<br/>        “没关系，我会把该有的，都补给你，一项也不漏。”傅夜沉会心一笑。
<br/>
<br/>        自从她重新回到他身边后，他有感觉到她对他的态度转好了许多。
<br/>
<br/>        至少，不再抗拒他吻她，不再抗拒他接近她，甚至还会主动为她做一些事情。
<br/>
<br/>        比如说，为他下厨，在他工作的时候不打扰他，安安静静地耐心等他，还有现在主动约他出去吃饭。
<br/>
<br/>        他没谈过恋爱，但是为了她，他会认真地去学。
<br/>
<br/>        她不仅是他的初恋，还是他现在、将来唯一的爱人。
<br/>
<br/>        他剑眉星眸、挺鼻薄唇……
<br/>
<br/>        这男人的脸，俊美无暇，但笑起来的样子，嗯，怎么就那么坏？
<br/>
<br/>        童筱颖瞬间看出了神。
<br/>
<br/>        他的“坏”中，还带着一股迷人的魅力，诱惑着她身上每一处敏感神经。
<br/>
<br/>        傅夜沉不同于傅昼景，如果傅昼景是“斯文”，那么他傅夜沉一定是“败类”。
<br/>
<br/>        离开傅氏集团大厦大门后，他没有带她去停车场取车，而是牵着她，沿着街边，慢悠悠地散步。
<br/>
<br/>        她随着他亦步亦趋，他牵着她的那只手，手腕上露出了那条他戴了三年之久的青丝手链。
<br/>
<br/>        这条青丝手链意味着什么？
<br/>
<br/>        她不清楚。
<br/>
<br/>        但他心里十分清楚。
<br/>
<br/>        这是她的青丝，也是他的情思，更是他斩不断的情丝。
<br/>
<br/>        “筱颖，我……”
<br/>
<br/>        “大哥哥，买朵花送给你身边这位漂亮的小姐姐吧！”忽然，耳畔传来一个稚嫩的女童声，打断了傅夜沉欲要说的话。
<br/>
<br/>        童筱颖和傅夜沉不约而同地循声望去。
