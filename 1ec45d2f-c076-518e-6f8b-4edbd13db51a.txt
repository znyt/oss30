        连星儿下意识地转过身去四下看了看，长长的校道上，稀稀疏疏地走着几个人影，两边是参天大树，风吹过时，树叶沙沙作响。
<br/>
<br/>        她并没有发现什么可疑人物，难道是她的心理作用？
<br/>
<br/>        爹地派来保护她的那些保镖，只要她进了学校后，便不会再在暗中保护。
<br/>
<br/>        而这种被人盯着的感觉，她也是最近才感觉到的。
<br/>
<br/>        连星儿深吸了口气，回过身去加快了脚下的步伐。
<br/>
<br/>        当她赶回寝室去拿书的时候，寝室里一个人也没有，顾千凝她们已经去了自习室。
<br/>
<br/>        连星儿抱着书和笔记本，找了一间又一间的自习室后，终于在一间有空调的自习室看到了一个空位置。
<br/>
<br/>        结果，当她欣喜地抱着课本准备走过去的时候，岂料一个女生先她一步坐了进去。
<br/>
<br/>        但是，那女生刚坐下去一秒，又立马站了起来，聋拉着脑袋快速跑开了。
<br/>
<br/>        连星儿一脸懵，这是她跑过的第五间自习室了，她不能一直这样在找自习室的路上奔波，于是疾步上前，管他三七二十一地直接往那个空位置上坐了下去。
<br/>
<br/>        自习室里的课桌是张类似于餐桌的大长桌，方便大家围在一起自习时还可以彼此学习交流。
<br/>
<br/>        而此时，与连星儿面对面坐成一排的女生们纷纷抬头，用一种极其艳羡又极其公愤的眼神瞪着她。
<br/>
<br/>        一秒、两秒、三秒！
<br/>
<br/>        连星儿与她们对视三秒后，收回目光，淡定自若地坐下。
<br/>
<br/>        教室里鸦雀无声，连星儿坐下后，翻书都是蹑手蹑脚，生怕吵到周遭一起自习的同学。
<br/>
<br/>        就在她准备开始做英语测试试卷的时候，一张纸条从右手边递了过来。
<br/>
<br/>        连星儿看到纸条上的英文句后，诧异地偏过头去。
<br/>
<br/>        只见明司寒一手抵着脸颊，一手握着笔，对她咧嘴一笑。
<br/>
<br/>        连星儿瞥了明司寒一眼后，拿起笔在纸条上写了一句“懒得理你”的英文，又将纸条扔还给了明司寒。
<br/>
<br/>        明司寒从本子上捡起纸条看了看，只笑不语，但也没再打扰她连星儿的自习，而是一手托腮，意味深长地看着她。
<br/>
<br/>        连星儿一直专注于自己的测试卷，一双点漆的漂亮大眼，偶尔会快速地左右转动。
<br/>
<br/>        扫完题目，她便会开始动笔。
<br/>
<br/>        这丫头的侧颜很秀气，说她貌美肤白一点都不为过。
<br/>
<br/>        特别是她认真做题的样子，还透着一丝高中生的稚气，可爱得不得了。
<br/>
<br/>        明司寒在不知不觉中看出了神。
<br/>
<br/>        时间，好像只在这丫头身上流逝。
<br/>
<br/>        一套测试试卷做完后，连星儿在自己的本子里左翻右翻，才发现自己竟然没带测试试卷的答案与解析。
<br/>
<br/>        而这个时候，明司寒伸手便夺走了她手中的测试卷，从笔袋里拿出一支红笔，直接在上面飞快地批阅起来。
<br/>
<br/>        连星儿见状，索性趴在桌子上盯着明司寒那只握着笔在替她改卷子的手。
<br/>
<br/>        他的手指骨分明，修长白皙，指甲修剪得干净光滑，在自习室的日光灯下还泛着一丁点的微光。
<br/>
<br/>        只不过十分钟的时间里，他便将她做错了的题更正了过来，还把解析写在了正确答案的后面。
<br/>
<br/>        连星儿目瞪口呆了一下，从明司寒手中接过自己的英语测试卷，鼓起腮帮子，认认真真地看起了错题。
<br/>
<br/>        她倒是虚心受教，把自己的错题整理到错题本上后，在纸条上写了句英文的“谢谢”，递给了明司寒。
<br/>
<br/>        自习课的下课铃声响起，自习室里的同学陆陆续续地离开。
<br/>
<br/>        连星儿终于按捺不住地问：“你今天怎么会来自习？”
<br/>
<br/>        认识他的这段时间里，她还是第一次在自习室里遇到他。
<br/>
<br/>        明司寒勾唇一笑，反问：“你们寝室里的三个女生都提前来了自习室，就你没来。你老实交代，你做什么去了？”
<br/>
<br/>        “你监视我！”连星儿顿时没好气地对明司寒翻了个白眼。
<br/>
<br/>        明司寒笑道：“我只有两只眼睛，哪有功夫去监视你。只是碰巧遇到了，她们说不知道你去哪里了。但是能肯定你会回来上自习课。”
<br/>
<br/>        “所以，你就特意来给我占了位置？”
<br/>
<br/>        “不然呢？”明司寒扬眉，话锋一转，又回到了原来的那个问题上，“说说看，你去哪儿了？”
<br/>
<br/>        “我去见我师父了！”连星儿毫不忌讳地回答道。
<br/>
<br/>        明司寒顿时微眯起眸子，凑到这丫头的脸前，淡淡地说：“你这撒谎都不打草稿的吗？”
<br/>
<br/>        “我师父不是你战队里的成员吗？那你打电话问他，不就知道我说的是不是真的了。”连星儿莞尔一笑。
