夏夕绾虽然昨天才进入A大，但是她气质纤尘绝丽早已经折服了所有人，A大有了自己的校花，他们都当成宝贝一样护着的。
<br/>
<br/>        夏夕绾还是很感动的，这些上课就睡觉的同学们身上是有很多闪光点的，至于论坛上这些恶评她倒没有放在心上，很明显是有人买了水军来黑她和A大的。
<br/>
<br/>        夕绾，不好了，还有一件大事要告诉你，那个孙进已经发动了大半个上流圈子的人准备下午再来A大校门口堵你，他要扯下你脸上的面纱，让大家看你的笑话！夏夕绾纤长的羽捷眨巴了一下，她突然想起了厉嫣然那带着几分幸灾乐祸的眼神，难道她就开心的这件事？孙进带着很多人要来揭她的面纱？恩…其实，揭下也是可以的。
<br/>
<br/>        范恬和其他同学面色一变，当即如临大敌，很明显他们都被网上那些恶评给洗脑了，以为夏夕绾的脸上真的有什么缺陷。
<br/>
<br/>        不好了绾绾，要不然你今天先请假吧，避开孙进那伙人。
<br/>
<br/>        是啊夕绾，你先避避，你小时候是不是被烧伤，脸上留了一个大疤或者是先天性的胎记，没有关系，我认识一个很好的整容医生，可以帮你修复。
<br/>
<br/>        你不要乱说，夕绾心里为自己的脸一定很难受了，别哪壶不开提哪壶。
<br/>
<br/>        夏夕绾都要笑了，实际上她也勾起了红唇，澄亮的翦瞳看向众人，嗓音清丽从容道，“同学们，谢谢你们的诸多好意，我是不会躲起来的，孙进他们想来就来吧，都是小事，放轻松，回到座位上加油学习啦，很快就要联考了。”
<br/>
<br/>        夏夕绾顾盼流转的澄眸闪烁着智慧从容的光芒，迅速让躁动的大家都安静了下来，她的眼神无比坚定，让人心安。
<br/>
<br/>        大家都回到了座位上，不过不是学习，而是盯着她纤柔的身影发呆这么好看的一个小仙女为什么脸上有缺陷呢，好可惜啊。
<br/>
<br/>        ……
<br/>
<br/>        转眼就到了放学时分，厉嫣然知道精彩的时刻来临了，她拿出手机给陆寒霆打了一个电话。
<br/>
<br/>        那端悠扬的手机铃声响了一遍，然后被接起了，陆寒霆低沉磁性的嗓音传递了过来，“喂。”
<br/>
<br/>        厉嫣然迅速露出了甜美的微笑，“寒霆哥哥，你现在在哪里，你可以来接我放学吗？”
<br/>
<br/>        “我今天有事，去不了。”陆寒霆直接拒绝了。
<br/>
<br/>        厉嫣然一滞，寒霆哥哥来不了啊？
<br/>
<br/>        “寒霆哥哥，我刚看到隔壁A大那里好像有很大的动静，我表哥孙进带着半个圈子的人都堵在了A大的校门口，好像要堵那个夏夕绾，想揭下她脸上的面纱，我表哥他们都说那个夏夕绾虽然气质出尘，但是揭下面纱一定是个丑八怪，要不然她戴面纱干什么，寒霆哥哥，你真的不过来吗？”
<br/>
<br/>        陆寒霆那端默了几秒，然后道，“不过去。”
<br/>
<br/>        他直接将电话给挂断了。
